### 1. Clip.Pad — 눈으로 보는 클립보드 작업대

**한 줄 요약**: 복사한 것들이 시각적 카드로 쌓이는 작업대. 클릭 한 번에 다시 붙여넣기.

**왜 필요한가**:
클립보드는 컴퓨터에서 가장 많이 쓰는 기능이지만, 항상 "직전 1개"만 기억한다.
Windows 클립보드 기록(Win+V)이 있지만 — 텍스트 목록이라 시각적으로 구분이 어렵고,
이미지/색상 미리보기가 빈약하며, 자주 쓰는 항목을 고정하기 불편하다.

**핵심 컨셉 — "눈으로 보는" 것이 차별점**:
- 복사할 때마다 카드가 생성됨 (텍스트는 내용 미리보기, 이미지는 썸네일, URL은 파비콘+제목)
- 카드 색상으로 종류 자동 구분 (텍스트=파랑, 이미지=초록, URL=주황, 코드=보라)
- 자주 쓰는 항목은 ⭐ 고정 → 상단 고정 영역에 항상 표시
- 검색: 타이핑하면 즉시 필터링

**사용 흐름**:
1. 평소처럼 Ctrl+C로 복사 (Clip.Pad가 백그라운드에서 자동 수집)
2. 붙여넣기하고 싶을 때 단축키(예: Ctrl+Shift+V) → 카드 패널 팝업
3. 원하는 카드 클릭 → 즉시 붙여넣기 후 패널 닫힘
4. 끝. 그게 전부.

**누가 쓸까**:
- 아빠(개발자): 코드 조각, URL, 커맨드 반복 붙여넣기
- 딸: 숙제할 때 여러 문장 복사해두고 골라서 붙이기
- 아내: 현장 사진 여러 장 복사 → 보고서에 하나씩 붙여넣기

**기술 구현**:
- 트레이 앱 + 글로벌 핫키 + 클립보드 감시 (WPF `Clipboard` + Win32 `AddClipboardFormatListener`)
- 최근 50개 유지, 고정 항목은 무제한
- SQLite로 영속 저장 (앱 재시작해도 유지)

**기존 도구와 차별점**:
| 기존 도구 | 문제점 | Clip.Pad |
|-----------|--------|----------|
| Win+V | 텍스트만 리스트, 시각 구분 어려움 | 카드 형태, 색상+썸네일로 즉시 구분 |
| Ditto | 기능 많지만 UI가 2000년대 스타일 | 모던 다크 테마, 미니멀 |
| ClipboardFusion | 유료, 설정 복잡 | 무료, 설치 즉시 사용 |

### 2. Sound.Jar — 소리 항아리

**한 줄 요약**: 항아리에 소리 구슬을 넣으면 그 소리가 흘러나오는, 직관적인 앰비언트 사운드 믹서.

**왜 필요한가**:
집중할 때, 잠들기 전, 카페 분위기를 내고 싶을 때 — 배경 소리가 필요하다.
유튜브 "카페 소음 10시간"은 광고가 뜨고, 웹 앱은 브라우저 탭을 차지하고,
전용 앱은 대부분 슬라이더 10개가 늘어선 복잡한 오디오 믹서 UI다.

**핵심 컨셉 — "항아리에 구슬 넣기" 메타포**:
- 화면에 유리 항아리가 하나 있음
- 옆에 소리 구슬이 놓여 있음: 🌧️빗소리, 🔥모닥불, ☕카페, 🌊파도, 🐦새소리, ⚡천둥, 🍃바람, 🎹피아노
- 구슬을 항아리에 드래그하면 그 소리가 시작됨
- 같은 구슬을 더 넣으면 볼륨 UP, 빼면 볼륨 DOWN
- 항아리 조합을 "장면"으로 저장: 집중모드 / 수면모드 / 카페모드

**사용 흐름**:
1. 트레이 아이콘 클릭 → 작은 항아리 창
2. 빗소리 구슬 드래그 → 🌧️ 비 내리는 소리 시작
3. 모닥불 구슬 추가 → 🔥 따뜻한 벽난로 소리 겹침
4. "집중" 버튼으로 저장
5. 다음에는 "집중" 한 번 클릭이면 끝

**누가 쓸까**:
- 아빠: 코딩할 때 카페 분위기 (카페+빗소리+재즈피아노)
- 딸: 잠들기 전 파도소리+바람소리 (수면 타이머 30분 설정)
- 아내: 퇴근 후 릴렉스 (모닥불+새소리)
- 아이 키우는 여동생: 쌍둥이 재울 때 화이트노이즈 대용

**기술 구현**:
- NAudio 라이브러리로 다중 오디오 스트림 동시 재생
- 소리 파일은 내장 리소스 (루프 가능한 30초~1분 wav/ogg)
- 볼륨 개별 제어 + 마스터 볼륨
- 수면 타이머 (점진적 페이드아웃)

**기존 도구와 차별점**:
| 기존 도구 | 문제점 | Sound.Jar |
|-----------|--------|-----------|
| Noisli (웹) | 브라우저 탭 차지, 무료 제한 | 네이티브 앱, 완전 무료 |
| myNoise (웹) | 슬라이더 UI 복잡 | 구슬 드래그, 5살도 이해 |
| Rainy Mood | 비 소리 하나만 | 8종+ 자유 조합 |

### 3. Type.Wand — 타이핑 마법봉

**한 줄 요약**: 텍스트를 선택하고 단축키 한 번 → 즉석에서 대소문자 변환, 정리, 포맷팅.

**왜 필요한가**:
개발자는 하루에도 수십 번 텍스트를 변환한다 — camelCase↔snake_case, JSON 정리, 공백 제거.
일반인도 마찬가지다 — 이메일에서 복사한 텍스트의 서식 제거, 줄바꿈 정리, 한영 변환 실수 교정.
이 모든 게 흩어진 웹사이트와 도구에서 이루어지고 있다.

**핵심 기능 (단축키 한 번 → 팝업 메뉴)**:
```
📝 텍스트 정리
  ├─ 서식 제거 (순수 텍스트만)
  ├─ 줄바꿈 → 한 줄로
  ├─ 중복 줄 제거
  └─ 앞뒤 공백 제거

🔤 대소문자
  ├─ UPPER CASE
  ├─ lower case
  ├─ Title Case
  ├─ camelCase
  └─ snake_case

🔧 개발자 도구
  ├─ JSON 정리 (Pretty Print)
  ├─ JSON 압축 (Minify)
  ├─ URL 인코딩/디코딩
  ├─ Base64 인코딩/디코딩
  └─ 이스케이프/언이스케이프

📊 정보
  ├─ 글자 수 / 단어 수 / 줄 수
  ├─ MD5 / SHA256 해시
  └─ 바이트 크기
```

**사용 흐름**:
1. 아무 앱에서 텍스트 선택
2. 단축키 (예: Ctrl+Shift+T) → 커서 옆에 변환 메뉴 팝업
3. 원하는 변환 클릭
4. 선택된 텍스트가 즉시 변환된 텍스트로 교체됨 (또는 클립보드에 복사)

**누가 쓸까**:
- 아빠: JSON 정리, 케이스 변환, Base64 (매일 수십 회)
- 딸: 영어 숙제에서 대문자 변환
- 누구나: 웹에서 복사한 텍스트 서식 제거

**기술 구현**:
- 글로벌 핫키 + `SendKeys`로 선택 텍스트 가져오기 (Ctrl+C 시뮬레이션)
- 변환 후 `SendKeys`로 붙여넣기 (Ctrl+V 시뮬레이션)
- WPF Popup 윈도우 (커서 위치에 표시)

### 4. Rush.Clock — 출발 카운트다운

**한 줄 요약**: "몇 시까지 도착해야 해" → 역산해서 "몇 분 후 출발" 카운트다운을 보여주는 시계.

**왜 필요한가**:
"9시 회의니까... 지하철 40분... 걸어서 10분... 준비 15분... 7시 55분에 나가야 하네."
이 역산을 매일 아침 머릿속으로 한다. 그리고 종종 틀린다.

**핵심 컨셉 — 시계가 아니라 "카운트다운"**:
- 프리셋 등록: "출근" → 도착 09:00, 이동 50분, 준비 15분
- 트레이 아이콘이 남은 시간 표시: `47m` → `12m` → `3m` → `NOW!`
- 아이콘 색상이 변화: 초록(여유) → 노랑(슬슬) → 빨강(지금 나가!)
- 출발 10분 전, 5분 전, 즉시 — 3단계 알림

**프리셋 예시**:
```
🏢 출근      | 도착 09:00 | 이동 50분 | 준비 15분 | 월~금
🏫 딸 등교    | 도착 08:30 | 이동 15분 | 준비 20분 | 월~금
🏥 병원      | 도착 14:00 | 이동 30분 | 준비 10분 | 수시
👨‍👩‍👧 부모님 댁  | 도착 12:00 | 이동 20분 | 준비 10분 | 주말
```

**누가 쓸까**:
- 아빠: 출근/회의/약속 시간 역산 자동화
- 아내: 현장 출발 시간 관리
- 딸: "학교 갈 시간!" 카운트다운 (스스로 보고 준비)
- 부모님 방문, 여동생 집 방문 등 가족 약속

**기술 구현**:
- 트레이 앱 + 타이머 (1분 간격 체크)
- 프리셋은 JSON 파일 저장
- 요일별 자동 활성화 (평일만, 주말만, 매일)
- 윈도우 토스트 알림 (Microsoft.Toolkit.Uwp.Notifications)

### 5. Split.Ring — 동시 다중 타이머

**한 줄 요약**: 여러 개의 타이머를 동시에 돌리는 화면 위 타이머 대시보드.

**왜 필요한가**:
"라면 3분" 타이머를 설정하면서 동시에 "빨래 1시간" 타이머도 필요하다.
폰 타이머는 잠금화면 뒤에 숨고, Windows 시계 앱은 하나씩만 눈에 보인다.
코딩하다가 "이 빌드 5분 기다려야지" 하면서 까먹는 일이 다반사다.

**핵심 컨셉 — 화면 가장자리에 붙는 링(고리) 타이머**:
- 화면 우측 하단에 작은 원형 타이머들이 줄지어 표시
- 각 타이머는 원형 프로그레스 링 + 이름 + 남은 시간
- 새 타이머 추가: 우클릭 → "라면" 입력 → "3분" 입력 → 시작
- 자주 쓰는 타이머 프리셋: 라면(3분), 휴식(10분), 빨래(60분), 집중(25분)

**빠른 입력 문법** (어린이도 가능):
```
3분          → 이름 없는 3분 타이머
라면 3분     → "라면" 이름의 3분 타이머
1시간 30분   → 90분 타이머
오후 3시     → 오후 3시까지 카운트다운
```

**누가 쓸까**:
- 아빠: 빌드 대기, 미팅 시작까지, 점심시간 끝
- 딸: 숙제 시간 (30분 타이머), 게임 시간 제한 (1시간), 라면 끓이기 (3분)
- 온 가족: 요리 타이머 여러 개 동시에

**기술 구현**:
- WPF Always-on-top 윈도우 (투명 배경, 클릭 관통 가능)
- 원형 프로그레스: `Arc` + `DispatcherTimer`
- 완료 시 소리 알림 + 토스트 알림
- 드래그로 위치 이동 가능

### 6. Calc.Pop — 떠오르는 계산기

**한 줄 요약**: 단축키 한 번으로 커서 위치에 나타나는 미니 계산기. 자연어 수식 지원.

**왜 필요한가**:
Windows 계산기는 시작 메뉴에서 찾아 열어야 하고, 전체 윈도우를 차지한다.
개발자는 "0xFF를 10진수로", "15%의 세금", "KRW↔USD" 같은 계산을 자주 하지만
매번 구글에 검색하거나 계산기를 열기 귀찮다.

**핵심 기능**:
```
기본: 1024 * 768          → 786,432
비율: 32000의 15%         → 4,800
진법: 0xFF                → 255
시간: 2시간30분 + 45분    → 3시간 15분
단위: 5km → mi            → 3.11 mi
온도: 72°F → °C           → 22.2°C
```

**사용 흐름**:
1. 단축키 → 커서 옆에 한 줄 입력창 팝업
2. 수식 입력 → 실시간 결과 표시
3. Enter → 결과가 클립보드에 복사 + 창 닫힘
4. 또는 Esc로 그냥 닫기

**누가 쓸까**:
- 아빠: 진법 변환, 퍼센트 계산, 단위 변환
- 딸: 산수 숙제 검산 (직관적 수식 입력)
- 아내: 자재 단가 계산, 면적 환산

### 7. Stamp.It — 사진 도장 찍기

**한 줄 요약**: 사진을 드래그하면 날짜/메모/워터마크를 찍어주는 도구.

**왜 필요한가**:
건설 현장 사진에 날짜와 메모를 넣어야 할 때, 포토샵은 과하고 그림판은 부족하다.
가족 사진에 날짜를 넣고 싶을 때, 폰 앱을 쓰자니 PC에서 작업 중이다.
SNS에 올릴 사진에 간단한 워터마크를 넣고 싶을 때.

**사용 흐름**:
1. 사진 파일을 Stamp.It 윈도우에 드래그 앤 드롭 (또는 클립보드에서 붙여넣기)
2. 미리보기 표시
3. 도장 선택:
   - 📅 날짜 (자동: 2026-02-20)
   - ✏️ 메모 (직접 입력)
   - 💧 워터마크 (반투명 텍스트)
   - 📐 리사이즈 (가로 1920px로 축소 등)
4. 위치 선택 (좌상/우하/중앙 등)
5. "저장" 클릭 → 완료

**누가 쓸까**:
- 아내(건설감리): 현장 사진 + 날짜 + 위치메모 → 보고서용 사진 일괄 처리
- 딸: 그림에 이름표 넣기
- 아빠: 스크린샷에 주석 달기

**기술 구현**:
- WPF 드래그앤드롭 + `System.Drawing` 또는 `SkiaSharp`
- 일괄 처리 지원 (여러 장 드래그)
- EXIF 날짜 자동 읽기

### 8. Whisper.Ping — 가족 한마디

**한 줄 요약**: 같은 집 Wi-Fi에 있는 가족 PC에 "밥 먹어~" 한 줄 메시지를 보내는 앱.

**왜 필요한가**:
"밥 다 됐어!" 하고 소리치는 대신, PC로 한 줄 보내면 안 될까?
카톡은 폰을 찾아야 하고, 이메일은 과하고, 사내 메신저는 가족용이 아니다.
같은 집 네트워크에서, 앱 설치만 하면 되는, 초간단 가족 메시지 보드.

**핵심 컨셉**:
- 같은 LAN의 Whisper.Ping 앱끼리 자동 발견 (UDP 브로드캐스트)
- 메시지 입력 → Enter → 상대방 PC에 토스트 알림 팝업
- 메시지 이력 최근 20개 유지 (채팅이 아님, "한마디 전달"에 집중)
- 프리셋 버튼: "밥 먹어", "나 왔어", "전화해"

**누가 쓸까**:
- 가족 간: "밥 다 됐어!", "숙제 했어?", "아빠 퇴근했어"
- 부모님 PC에 설치: "엄마 저녁에 갈게요" (카톡 대신)

**기술 구현**:
- UDP 브로드캐스트로 자동 디스커버리 (제로 설정)
- TCP 소켓으로 메시지 전송 (LAN 전용, 인터넷 불필요)
- 토스트 알림으로 수신 표시
- 트레이 앱 (항상 대기)

### 9. Color.Drop — 화면 색상 추출기

**한 줄 요약**: 단축키 → 화면 아무 곳 클릭 → 그 픽셀의 색상을 HEX/RGB로 복사.

**핵심 기능**:
- 단축키로 활성화 → 커서가 돋보기로 변환 (확대 미리보기)
- 클릭 → 색상 정보 팝업: `#1E90FF` / `rgb(30, 144, 255)` / `hsl(210, 100%, 56%)`
- 클릭 한 번으로 원하는 형식 복사
- 최근 색상 팔레트 히스토리 (최근 20색)
- 조화 색상 추천 (보색, 유사색)

**누가 쓸까**:
- 아빠: UI 개발 시 색상 추출 (매일)
- 딸: "이 색깔 예쁘다" → 코드로 저장

### 10. QR.Snap — QR 코드 즉석 생성/인식

**한 줄 요약**: 클립보드의 텍스트/URL → QR 코드 생성 | 화면의 QR 코드 → 텍스트 추출.

**핵심 기능**:
- **생성**: 클립보드에 URL이 있으면 자동으로 QR 코드 생성 → 이미지 복사/저장
- **인식**: 화면 영역 선택 → QR 코드 인식 → 텍스트를 클립보드에
- 생성 이력 저장 (최근 10개)

**누가 쓸까**:
- 아빠: 개발 중 QR 테스트, URL 공유
- 아내: 현장 안내문에 QR 코드 넣기
- 부모님: "이 QR 코드 뭐야?" → PC 화면에서 바로 인식

### 11. Pixel.Tape — 화면 위 줄자

**한 줄 요약**: 화면 위에 투명한 줄자를 올려놓고 픽셀/cm 단위로 거리를 재는 도구.

**핵심 기능**:
- 단축키 → 화면 위에 드래그로 측정선 생성
- 수평/수직/대각선 거리 표시 (px 단위)
- DPI 기반 실제 단위 변환 (cm, mm, inch)
- 여러 측정선 동시 유지 가능
- 가이드선 기능 (수평/수직 점선)

**누가 쓸까**:
- 아빠: UI 요소 간격 측정, 디자인 검수
- 아내: 도면 화면에서 대략적 치수 확인
