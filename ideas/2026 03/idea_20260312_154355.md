### 1. `Flash.Card` — 로컬 간격반복 플래시카드 앱

**카테고리**: Productivity / Learning / Offline
**핵심 기능**: SM-2 알고리즘 기반 간격반복(Spaced Repetition) 플래시카드 관리 앱. 완전 오프라인 동작, Anki 덱(.apkg) 가져오기 지원, 복습 세션 통계 시각화로 장기 기억력 향상을 돕는다.

**상세 기능**:
- SM-2(SuperMemo 2) 간격반복 알고리즘: 품질 평가(Again/Hard/Good/Easy) → 다음 복습일 자동 산출
- 카드 정면/이면 자유 형식 (텍스트, 이미지, 마크다운 렌더링, 코드 구문 강조)
- 덱 관리: 덱 중첩, 서브덱, 태그 필터, 드래그 정렬
- Anki `.apkg` 가져오기 / 표준 CSV·Markdown 내보내기
- 일일 복습 목표 설정 + 연속 학습 스트릭 트래커
- 학습 통계 대시보드: 보류·신규·재학습·성숙 카드 비율, 성공률 차트, 예상 복습 부담 캘린더
- 전역 단축키 팝업으로 빠른 카드 추가 (클립보드에서 자동 채우기)
- 텍스트 음성 합성(SAPI) 연동 — 언어 학습 카드 발음 재생

**기술 스택**:
- WPF / MVVM, SQLite (EF Core), SM-2 알고리즘 C# 구현
- Markdig (마크다운 렌더), SkiaSharp (카드 이미지 렌더)
- Anki `.apkg` SQLite 파싱 (미디어 압축 해제 포함)
- SAPI5 (System.Speech.Synthesis)

**수익 모델**: 완전 무료 오프라인
**차별화**: avoid.md에 유사 앱 없음. Type.Rocket은 타이핑 WPM 연습, Dict.Cast는 사전 조회 — Flash.Card는 임의 지식 영역(언어·의학·코딩)의 장기 기억 내재화에 특화된 복습 엔진.

### 2. `Epub.Cast` — 로컬 전자책 리더

**카테고리**: Productivity / Reading / Media
**핵심 기능**: EPUB 2/3·MOBI·EPUB3 고정 레이아웃 전자책 오프라인 뷰어. 로컬 라이브러리 관리, 독서 진행률 자동 저장, 하이라이트·메모 기능, TTS 연동으로 들으며 읽기를 지원한다.

**상세 기능**:
- EPUB 2/3, MOBI, AZW3 파싱 및 WebView2 렌더링 (CSS3 완전 지원)
- 로컬 라이브러리: 폴더 감시 자동 추가, 표지 썸네일, 메타데이터(제목·저자·장르) 편집
- 독서 진행률 자동 저장 (재실행 시 마지막 위치 복원)
- 형광펜 4색 하이라이트 + 인라인 메모, 하이라이트 모음 Markdown 내보내기
- 폰트 종류·크기·배경색·줄 간격·여백 개인화 (라이트/다크/세피아 테마)
- 내장 SAPI/Edge Neural TTS 연동 — 페이지 자동 재생, 속도 조절
- 정의 팝업: 선택 단어 → 내장 Dict.Cast 연동 (설치 시)
- 전체 텍스트 검색 (FTS5, 챕터 단위 결과)

**기술 스택**:
- WPF + WebView2 (EPUB CSS 렌더), EpubSharp / MobiReader
- SQLite (EF Core) 라이브러리·하이라이트 저장
- System.Speech.Synthesis (SAPI) / SharpZipLib (EPUB 압축)

**수익 모델**: 완전 무료 오프라인
**차별화**: PDF.Forge는 PDF 편집, Mark.View는 Markdown 뷰어, Ink.Cast는 노트 앱. EPUB 규격 전자책 리더는 avoid.md에 없음.

### 3. `SVG.Forge` — 오프라인 SVG 벡터 에디터

**카테고리**: Creative / Dev / Design
**핵심 기능**: 완전 오프라인 기본 SVG 벡터 그래픽 에디터. 도형·패스·텍스트 레이어 편집, Boolean 연산, SVG/PNG/PDF 내보내기를 지원하여 아이콘 디자인·다이어그램 작성에 활용한다.

**상세 기능**:
- 캔버스: 무한 줌·패닝, 그리드·스냅·가이드라인, 다중 레이어
- 기본 도형 (사각형·원·다각형·별·선·폴리라인) + 자유 Bezier 패스 편집
- Boolean 연산: Union·Subtract·Intersect·XOR (Clipper2 라이브러리)
- 텍스트 도구: 폰트 선택, 경로 따라 텍스트(TextOnPath)
- 채우기: 단색·선형/방사형 그래디언트·패턴, 스트로크(대시·끝점)
- CSS 트랜스폼 애니메이션 편집기 (간단한 rotate/scale 키프레임)
- SVG 1.1 / SVG/SMIL 소스 편집 탭 (실시간 프리뷰 연동)
- 내보내기: SVG(인라인/외부), PNG(DPI 설정), PDF, ICO 연동

**기술 스택**:
- WPF + SkiaSharp (캔버스 렌더) / 또는 WPF DrawingContext 직접
- Clipper2 (Boolean), VectSharp (SVG 파싱/내보내기)
- Svg.NET 라이브러리 (SVG 파싱)

**수익 모델**: 완전 무료 오프라인
**차별화**: Pixel.Forge는 래스터 픽셀아트, Draw.Board는 자유형 화이트보드(벡터 출력 없음), Icon.Forge는 ICO 형식 변환기. SVG 벡터 에디터(패스·Boolean) 분야는 avoid.md에 없음.

### 5. `Net.Trace` — 시각적 트레이스라우트 + 세계지도

**카테고리**: Network / Tools
**핵심 기능**: 목적지 IP/도메인까지의 traceroute 경로를 각 홉의 지리적 위치로 세계지도 위에 핀으로 표시하고 홉별 RTT·패킷 손실을 차트로 실시간 보여주는 네트워크 진단 도구.

**상세 기능**:
- ICMP / UDP traceroute 실행 (TTL 증가법), IPv4·IPv6 선택
- 각 홉 IP → MaxMind GeoLite2 오프라인 DB로 국가·도시·좌표 조회
- 세계지도 (WPF Custom Map or MapControl) 위에 홉 핀 + 선 경로 표시
- 홉별 테이블: 홉#·IP·호스트명·위치·RTT(평균/최소/최대)·손실%
- 실시간 지속 모니터링 모드: N초마다 재실행, RTT 히스토리 라인 차트
- 복수 목적지 병렬 트레이스 (탭 UI)
- 결과 스냅샷 PNG 저장 / CSV 내보내기

**기술 스택**:
- C# `IcmpClient` (raw socket) / `System.Net.NetworkInformation.Ping` TTL 조작
- MaxMind GeoLite2 (오프라인 `.mmdb`) + MaxMind.Db NuGet
- WPF (커스텀 지도 캔버스 + SkiaSharp 핀 렌더)
- LiveCharts2 (RTT 라인 차트)

**수익 모델**: 완전 무료 오프라인
**차별화**: Net.Scan은 LAN ARP 기기 탐지(내부망), Uptime.Eye는 HTTP 폴링 가동 여부. Net.Trace는 인터넷 경로 추적·지리적 시각화 특화.

### 6. `Audio.Cut` — 오프라인 오디오 파형 트리머·편집기

**카테고리**: Audio / Tools
**핵심 기능**: MP3·WAV·FLAC·OGG 오디오 파일을 파형으로 시각화하고 구간 선택 트림·페이드 인/아웃·볼륨 정규화·무음 제거를 비파괴 방식으로 편집한 뒤 원하는 포맷으로 내보내는 경량 오디오 에디터.

**상세 기능**:
- 파형 시각화 (줌·패닝, 스크롤), 재생 헤드 + 루프 재생
- 구간 선택·삭제·트림, 다중 구간 마킹 (챕터 분할 내보내기)
- 페이드 인/페이드 아웃 (선형·지수·S커브)
- 볼륨 정규화 (Peak / RMS / EBU R128 LUFS)
- 무음 구간 자동 감지 → 하이라이트 / 일괄 제거
- 속도 변경 (피치 유지 Time-stretch, NAudio 또는 SoundTouch)
- 내보내기: MP3(CBR·VBR 비트레이트), WAV, FLAC, OGG
- 무제한 Undo/Redo, 작업 히스토리 사이드바

**기술 스택**:
- WPF + SkiaSharp (파형 렌더), NAudio (오디오 디코딩·인코딩·재생)
- SoundTouch.NET (Time-stretch·Pitch shift)
- LAME (MP3 인코딩 via NAudio wrapper)

**수익 모델**: 완전 무료 오프라인
**차별화**: Music.Player는 재생기, Video.Trim은 영상 트리머(오디오 전용 아님), Pitch.Find는 BPM·키 분석만. 오디오 파형 편집·비파괴 트리밍 기능은 avoid.md에 없음.

### 7. `OCR.Forge` — 오프라인 문서 일괄 OCR 변환기

**카테고리**: Files / Productivity / Tools
**핵심 기능**: 스캔 PDF·이미지 파일(JPG/PNG/TIFF)을 완전 오프라인 Tesseract OCR로 일괄 처리하여 검색 가능한 PDF 또는 DOCX·TXT로 변환하는 문서 디지털화 도구. 폴더 감시 자동 처리를 지원한다.

**상세 기능**:
- 입력 지원: JPEG·PNG·TIFF·BMP·PDF (페이지별 이미지 추출)
- Tesseract 5 OCR: 100개+ 언어팩, 한국어·영어·일본어 기본 내장
- 전처리: 기울기 보정(Deskew)·노이즈 제거·이진화 자동 적용
- 출력 형식: 검색 가능 PDF (OCR 레이어 내장)·DOCX·TXT·hOCR
- 일괄 처리: 폴더 드래그&드롭, 병렬 처리(코어 수 설정), 진행률 표시
- 결과 미리보기: 원본 이미지와 인식 텍스트 나란히 표시, 오류 구간 빨간 밑줄
- 폴더 감시 모드: 지정 폴더에 파일 추가 시 자동 OCR
- 신뢰도(Confidence) 리포트 CSV 내보내기

**기술 스택**:
- WPF / MVVM, Tesseract5 NuGet (tessdata 내장)
- PdfSharpCore (검색 가능 PDF 레이어), Docx (DOCX 생성)
- OpenCvSharp4 (이미지 전처리), Parallel.ForEach (병렬 OCR)

**수익 모델**: 완전 무료 오프라인
**차별화**: Screen.Pickup/Screen.OCR은 화면 영역 실시간 OCR, Snap.Cast는 스크린샷 → 마크다운 레이아웃 분석. OCR.Forge는 문서 파일 일괄 디지털화(Tesseract 배치 처리) 특화.

### 8. `IP.Forge` — IP 서브넷 계산기 / CIDR 시각화 도구

**카테고리**: Network / Dev / Tools
**핵심 기능**: IPv4·IPv6 주소와 CIDR 표기법을 입력하면 서브넷 마스크·네트워크 주소·브로드캐스트·호스트 범위·와일드카드를 즉시 계산하고 비트 다이어그램으로 시각화하는 네트워크 엔지니어 도구.

**상세 기능**:
- IPv4: IP+마스크 또는 CIDR(예: `192.168.1.100/24`) 입력 → 즉시 계산
  - 네트워크·브로드캐스트·첫/마지막 호스트, 호스트 수, 클래스(A/B/C)
  - 와일드카드 마스크, 바이너리 비트 시각화(4×8 블록, 네트워크/호스트 색상 구분)
- IPv6: CIDR 계산, 압축/전체 표기 변환, 인접 서브넷 목록
- 수퍼넷(Supernetting) 계산: 여러 CIDR 블록 → 최소 커버 수퍼넷
- VLSM(가변 서브넷): 필요 호스트 수 목록 입력 → 효율적 서브넷 분할 자동 계획
- 서브넷 목록 표: /24를 /26으로 분할 등 분할 시각화
- IP 범위 → CIDR 목록 변환 (시작IP–끝IP 입력)
- 결과 클립보드 복사 / TXT 내보내기

**기술 스택**:
- WPF / MVVM, 순수 C# 비트 연산 (외부 의존성 최소화)
- System.Net.IPAddress (파싱), SkiaSharp (비트 블록 다이어그램)

**수익 모델**: 완전 무료 오프라인
**차별화**: Quick.Calc는 16진수·IEEE 754 비트 연산 계산기. IP.Forge는 IP 서브넷·CIDR·VLSM 네트워크 설계 계산에 특화. avoid.md에 없음.

### 9. `Score.Cast` — 악보 뷰어 + MIDI 재생기

**카테고리**: Audio / Music / Creative
**핵심 기능**: MusicXML·ABC Notation 악보 파일을 오프라인으로 렌더링하여 음악 악보를 화면에 표시하고, 내장 MIDI 합성기로 즉시 재생, 조성 조옮김, 파트 선택, PNG/PDF 인쇄를 지원하는 악보 뷰어.

**상세 기능**:
- MusicXML(.xml/.mxl) / ABC Notation(.abc) / MIDI(.mid) 파일 불러오기
- 표준 오선보 렌더링 (음표·쉼표·박자·조표·나타냄말, SharpMeasures 또는 자체 렌더)

  1. **실루엣** — 국가 외곽선만 보고 국가명 맞히기 (4지선다 / 직접 입력)
  2. **국기** — 국기 이미지 → 국가명 맞히기
  3. **수도** — 국가명 → 수도 맞히기 / 수도 → 국가명
  4. **지도 클릭** — 세계지도에서 제시된 국가 위치 클릭
  5. **혼합** — 4종 랜덤 출제
- 196개 UN 회원국 + 주요 영토 전체 포함
- 오답 항목 약한 SM-2 간격반복으로 재출제
- 지역 필터: 아시아·유럽·아프리카·아메리카·오세아니아 선택
- 연속 정답 스트릭 + 빠른 정답 보너스 배수
- 세계지도 채워가기: 맞힌 국가 색상으로 점진적 채움
- 일일 챌린지: 20문제 고정 시드, 정확도·시간 기록

**기술**: GeoJSON (Natural Earth) 오프라인 내장, SkiaSharp 벡터 지도 렌더, SQLite 오답 이력
**매력**: 공부가 되면서도 "맞혔을 때" 쾌감. 지도 채워가기 시각적 진행감이 강한 중독 요소.
**차별화**: Color.Blitz는 스트룹 인지 속도 게임, Word.Bomb은 단어 폭탄. 지리 퀴즈 장르는 avoid.md에 없음.

### 8. `Pipe.Lay` — 타이머 파이프 회전 퍼즐

**장르**: Puzzle / Casual / Strategy
**핵심 메커닉**: N×M 격자의 파이프 타일을 클릭해 90° 회전시켜 수원지에서 출구까지 완전히 연결된 경로를 만드는 시간 제한 퍼즐. 물이 자동으로 흘러오기 전에 모든 파이프를 돌려 연결해야 한다.

**게임플레이**:
- 수원지 → 출구, 물이 N초 후 자동 흐르기 시작 (카운트다운)
- 파이프 타일 종류: 직선·꺾임(L)·T자·크로스·막힌 끝·단방향 밸브·잠긴 파이프·순간이동 포탈
- 물 흐름 후 연결 안 된 부분은 누수 → 잔여 칸 수 기준 별 평가
- 레벨 패키지: 5×5(입문)→8×8(중급)→12×10(고급)→무한 랜덤 모드
- 특수 조건: 고정 파이프(회전 불가), 색상별 독립 경로(동시에 여러 색 연결)
- 시간 여유 보너스: 남은 카운트다운 시간 비례 추가 점수
- 에디터 모드: 직접 레벨 제작 → 공유 코드 (Base64)

**기술**: WPF 타일 그리드, BFS 흐름 시뮬레이션, 절차적 레벨 생성 (연결 경로 먼저 생성→타일 배치)
**매력**: 연결 완성 순간의 시각적 만족감. 타이머 압박이 속도전으로 만들어 두근두근한 재미.
**차별화**: Flow.Pipe는 같은 색 점끼리 파이프 연결(시간 없음, 채우기 퍼즐). Pipe.Lay는 타일 회전·물 흐름 타이머·단방향 밸브가 핵심 차이. avoid.md의 Flow.Pipe와 명확히 다름.

### 9. `Trail.Blaze` — 파쿠르 모멘텀 콤보 러너

**장르**: Arcade / Platformer / Runner
**핵심 메커닉**: 절차 생성 도시 환경을 가로질러 달리며 벽달리기·슬라이딩·레지 그랩·공중 구르기를 연결해 콤보 멀티플라이어를 쌓는 사이드스크롤 파쿠르 러너. 속도와 콤보 체인이 상호강화되어 경지에 오를수록 더 빠르고 유려해진다.

**게임플레이**:
- WASD/방향키 이동 + 공간 타이밍 점프 (Coyote Time 0.1초)
- 파쿠르 액션: 벽달리기(5 타일), 레지 그랩(매달리기→도약), 슬라이드(저자세 통과+가속), 공중 구르기(충격 흡수)
- 모멘텀 시스템: 콤보 유지 시 캐릭터 속도 비례 가속, 착지 실수 시 리셋
- 콤보 멀티플라이어: 동작 다양성·끊기지 않는 연속성에 따라 x1~x8
- 절차 생성 도시 배경: 건물 높이·간격·장애물 패턴 무한 조합, 낮·황혼·야간 사이클
- 장애물: 사인파 이동 크레인, 수평 도미노 벽, 낙하 지붕, 역방향 강풍
- 데일리 시드 도전: 동일 트랙으로 최고 점수 경쟁
- 스킨 해금: 일정 콤보·거리 달성 시 캐릭터 색상·이펙트 해금

**기술**: MonoGame 또는 SkiaSharp + 커스텀 물리(중력·마찰·Coyote Time), 절차 생성 세그먼트 빌더
**매력**: "완벽한 연결" 달성 시 유체처럼 흐르는 이동감. 콤보가 쌓일수록 시각·청각 이펙트 증폭으로 몰입도 상승.
**차별화**: Neon.Run은 단순 장애물 점프 타이밍, Dash.City는 일반 러너, Crumble.Run은 Voronoi 파괴 지형 러너. Trail.Blaze는 파쿠르 액션 다양성·콤보 체인 모멘텀 시스템이 핵심 — avoid.md에 없음.

### 10. `Swarm.Ant` — 개미 군집 ACO 시뮬레이션 + 퍼즐

**장르**: Simulation / Strategy / Puzzle / Sandbox
**핵심 메커닉**: Ant Colony Optimization(ACO) 페로몬 기반 경로 탐색·강화·소멸을 실시간으로 시각화하는 개미 군집 시뮬레이션. 사용자가 장벽·먹이·물을 배치해 군집 행동을 유도하는 샌드박스 모드와, 목표 달성 조건이 있는 퍼즐 챌린지 모드를 제공한다.

**게임플레이**:
- 페로몬 시각화: 먹이 탐색(파란)/귀환(초록) 페로몬 밀도 색상 오버레이
- 개미 행동 3단계: 탐색(랜덤 워크) → 먹이 발견(페로몬 분비) → 경로 강화 (다른 개미 합류)
- 샌드박스 도구: 벽 그리기, 먹이 배치, 물 장벽, 여왕개미 巢 위치, 페로몬 제거제
- 멀티 군집: 빨강·파랑·초록 3개 군집 경쟁 (영역별 자원 쟁탈)
- 사용자 인터랙션: 클릭으로 개미 통로 막기, 페로몬 강화·약화, 먹이 순간이동
- 퍼즐 모드: "A에서 B까지 최단 경로 형성" "30초 내 특정 구역 점령" "장벽 최소로 두 군집 분리" 등 15+ 챌린지
- 슬로우/패스트 포워드 (0.1×~10×)
- 시뮬레이션 스냅샷 PNG 저장 / GIF 타임랩스 생성

**기술**: C# ACO 파라미터 (α 페로몬 비중, β 휴리스틱 비중, ρ 증발률), SkiaSharp 픽셀 수준 페로몬 히트맵 렌더, 공간 해싱(먹이 탐색 최적화)
**매력**: 단순 규칙에서 창발하는 복잡한 패턴이 경이로움을 준다. 직접 장벽 배치로 군집을 유도하는 능동적 인터랙션.
**차별화**: Colony.Sim은 범용 에이전트 군집 AI (다종 경쟁·영양소 확산), Spore.Net은 균류 네트워크 최적화. Swarm.Ant는 ACO 알고리즘·페로몬 경로 시각화·개미 특화 행동이 핵심 — avoid.md에 없음.
