# 2개 프로젝트 구현 — 25차 브레인스토밍 선택 아이디어

## 작업 개요

25차 브레인스토밍(idea_20260314_184028.md)에서 선택된 2개 프로젝트를 구현.

## 구현 내용

### 1. Icon.Maker (Applications/Tools.Utility/)
SVG/PNG → ICO 멀티사이즈 변환 + 픽셀 그리드 에디터

**주요 기능**:
- SVG (Svg.Skia 래스터라이징) / PNG (SkiaSharp 디코딩) / ICO 로드
- 16/32/48/64/128/256px 6개 사이즈 자동 리사이즈
- WriteableBitmap 줌 픽셀 에디터 (연필/지우개/채우기 도구)
- 우클릭 스포이드 색상 추출
- ICO 내보내기 (IcoEncoder: ICONDIR+ICONDIRENTRY+PNG 바이너리 직접 인코딩)
- PNG 개별 추출
- 다크/라이트 배경 미리보기 전환

### 2. JSON.Tree (Applications/Development/Inspector/)
JSON / YAML / TOML 구조 탐색기

**주요 기능**:
- 3개 포맷 자동 감지 파싱 (System.Text.Json / YamlDotNet / Tomlyn)
- WPF TreeView 가상화 트리 탐색
- 실시간 키·값 검색 + 매칭 수 표시
- Side-by-Side Diff 모드: 추가(녹)/삭제(적)/변경(황) 배지
- 재귀 트리 Diff (오브젝트: 키 매칭, 배열: 인덱스 매칭)
- 노드 선택 → 경로 표시 (예: $.users[0].email)
- JSON → YAML → TOML 변환 내보내기
- 클립보드 붙여넣기 / 드래그&드롭 지원
