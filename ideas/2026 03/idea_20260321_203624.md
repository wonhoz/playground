### 1. `Sub.Forge` — SRT/VTT/ASS 자막 올인원 편집기

**카테고리**: Media / Files
**핵심 기능**: 동영상 파일 없이도 자막 파일(SRT/VTT/ASS/SSA)을 정밀 편집하는 오프라인 도구. 타임라인 타임라인 기반 UI에서 각 자막 블록의 시작/종료 타이밍을 ms 단위로 조정하고, 다중 언어 트랙을 나란히 비교·편집한다.

**상세 기능**:
- SRT / WebVTT / ASS / SSA 파싱 + 저장 (포맷 간 변환 포함)
- 타임라인 시각화 — 자막 블록 드래그로 시간 이동, 길이 조절
- 비디오 썸네일 추출(FFmpeg 연동) → 타임라인 위에 프레임 미리보기
- 다중 언어 Side-by-Side — 원문·번역 라인 1:1 병렬 편집
- 일괄 시간 이동 (싱크 오프셋 ±ms 전체 적용)
- CPS(초당 글자 수) / CPL(줄 글자 수) 초과 구간 빨간 경고
- 자막 자동 병합·분리 (기준 길이/공백 설정)
- 맞춤법 검사 통합 (한국어/영어)
- 내보내기: SRT / VTT / ASS / TXT (순수 텍스트)

**기술 스택**:
- .NET 10 / WPF / MVVM
- FFmpeg.AutoGen (비디오 썸네일 추출)
- SubtitleEdit 오픈소스 파서 참고
- AvalonEdit (자막 텍스트 편집, 구문 강조)

**수익 모델**: 무료 오픈소스
**차별화**: Video.Trim(무손실 컷 편집)은 비디오 자체를 다루지만 Sub.Forge는 자막 타이밍·텍스트 전문 편집기. SubtitleEdit와 달리 WPF 다크 테마 + 다중 언어 비교 UI에 집중.

### 2. `Read.Flow` — RSVP 속독 훈련기

**카테고리**: Productivity / Education
**핵심 기능**: RSVP(Rapid Serial Visual Presentation) 방식으로 텍스트를 화면 중앙에 한 단어씩 고속 표시하여 안구 이동을 없애고 독서 속도를 높이는 훈련 도구. WPM 목표 설정과 진도 통계로 꾸준한 훈련을 지원한다.

**상세 기능**:
- 텍스트 붙여넣기 / TXT·EPUB·PDF 파일 직접 불러오기
- 표시 모드: 단어 단위 / 2~3단어 청크 / 문장 단위 전환
- WPM(분당 단어) 50~1500 단계 조절, 점진 자동 가속
- ORP(Optimal Recognition Point) 글자 빨간 강조 (Spritz 방식)
- 쉼표·마침표 감지 → 자동 일시 정지 (설정 가능)
- 진도 기록: 날짜별 WPM·이해도 자가평가 히스토리 그래프
- 야간 다크 테마 / 폰트 크기·색상 완전 커스터마이즈
- 전역 단축키 일시정지/재개/속도 조절

**기술 스택**:
- .NET 10 / WPF
- PdfPig (PDF 텍스트 추출)
- VersOne.Epub (EPUB 파서)
- System.Speech (선택적 TTS 병행)

**수익 모델**: 무료
**차별화**: Type.Rocket(타이핑 속도 훈련)은 키보드 입력이 주제. Read.Flow는 안구 움직임 제거를 통한 **읽기 속도** 훈련으로 완전히 다른 스킬셋.

**핵심 기능**: 기존 GIF 파일의 파일 크기를 대폭 줄이는 오프라인 최적화 도구. 팔레트 색수 양자화, 중복 프레임 제거, 프레임 속도 조절, WebP/APNG 대안 포맷 변환을 제공한다.

**상세 기능**:
- 드래그&드롭 GIF 불러오기 + 프레임별 미리보기 스트립
- 팔레트 색수 최적화 (256→128→64→32색 슬라이더)
- 중복·거의 동일 프레임 자동 감지·제거 (Perceptual hash)
- 손실 GIF 압축 (Gifsicle lossy 알고리즘)
- 디더링 방식 선택 (Floyd-Steinberg / Ordered / None)
- WebP 애니메이션 / APNG 변환 내보내기 (더 작은 크기)
- 원본 vs 결과 용량·품질 SSIM 실시간 비교 패널
- 일괄 처리: 폴더 내 전체 GIF 최적화
- 설정 프리셋 저장 (소셜미디어 용 / 최소용량 / 고품질)

**기술 스택**:
- .NET 10 / WPF
- Magick.NET (ImageMagick 래퍼)
- FFmpeg (WebP/APNG 변환)
- AnimatedGif 라이브러리

**수익 모델**: 무료
**차별화**: GIF.Cast(화면 GIF 녹화)는 신규 GIF 생성이고, GIF.Opt는 **기존 GIF 최적화**. Screen.Recorder는 파일 저장형 녹화로 완전히 다름.

**기술 스택**:

- NAudio (사인파 CW 신호 생성)

**상세 기능**:
- 드래그&드롭 원본(v1) / 대상(v2) 파일 로드
- XDelta3 / BSDiff / Bsdiff4 알고리즘 선택
- 패치 생성: 델타 크기 예상치 실시간 계산, 압축 레벨 조절
- 패치 적용: 원본 + 패치 파일 → 결과물 SHA-256 검증
- HEX 비교 패널: 두 파일 차이 구간 색상 강조
- 일괄 처리: 다수 파일 쌍 패치 생성 (게임 에셋 배포)
- 패치 메타데이터: 적용 전/후 파일 해시 임베딩
- 역방향 패치 자동 생성 (다운그레이드 지원)

**기술 스택**:
- .NET 10 / WPF
- SharpBsDiff (BSDiff 포트)
- VCDiff / xdelta3 CLI 래핑
- AvalonHex (HEX 뷰어 컨트롤)

**수익 모델**: 무료
**차별화**: Hex.Peek(헥스 에디터)은 파일 내용 탐색·편집. Bin.View는 PE 리소스 브라우저. Patch.Forge는 두 바이너리의 **델타 패치 파일 생성·배포**에 집중.

- 분석 결과 PDF/HTML 리포트 내보내기

**기술 스택**:
- .NET 10 / WPF
- Readability 오픈소스 알고리즘 포팅

### 8. `Wall.Plan` — 벽 인테리어 레이아웃 플래너

- 드래그&드롭 배치 + 그리드 스냅 (기본 10cm 간격)
- 간격 자동 균등 배분 (가로·세로 중앙 정렬 보조선)
- 못 위치 계산: 아이템별 행거 타입(중앙 1개·양쪽 2개·D-링) 선택 → 못 좌표 출력
- 레이어 갤러리 월: 살롱 스타일 혼합 배치 템플릿
- PDF 내보내기: A4 실측 도면 + 못 위치 체크리스트
- SVG 내보내기 (인테리어 디자인 앱 붙여넣기용)

**기술 스택**:
- .NET 10 / WPF
- SkiaSharp (캔버스 렌더링)
- PdfSharpCore (PDF 출력)

**차별화**: Draw.Board(무한 캔버스 화이트보드)는 범용 드로잉. Wall.Plan은 실제 벽 치수 기반의 **인테리어 액자 배치 전용** 도구로 못 위치까지 계산.

### 9. `Ratio.Forge` — 디자이너 수치 계산기

**카테고리**: Dev / Design / Tools
**핵심 기능**: UI/UX·그래픽 디자이너가 매일 필요로 하는 수치 계산을 모아둔 전문 계산기. 황금비·√2 비율·8pt 그리드·픽셀↔rem·타이포그래피 모듈라 스케일 계산을 오프라인에서 즉시 수행.

**상세 기능**:
- 황금 비율 계산: 한 변 입력 → 나머지 변 자동 (Φ = 1.618)
- √2 비율 계산 (A 시리즈 용지 규격, ISO 216)
- 타이포그래피 모듈라 스케일: 베이스·비율 설정 → Major Third / Perfect Fourth / Golden 등 전체 스케일 출력
- 픽셀 ↔ rem / pt / em 변환 (기준 font-size 설정)
- 8pt Grid 계산기: 원하는 값 → 가장 가까운 8 배수 제안
- 반응형 브레이크포인트 계산기 (xs/sm/md/lg/xl 구간 커스텀)
- 색상 대비비 빠른 계산 (WCAG 2.1 AA/AAA)
- 계산 히스토리 + 즐겨찾기 저장

**기술 스택**:
- .NET 10 / WPF
- 수학 계산 순수 C# (외부 라이브러리 최소화)
- SkiaSharp (비율 시각화 다이어그램)

**수익 모델**: 무료
**차별화**: Unit.Forge(12카테고리 단위 변환)는 물리 단위 변환. Quick.Calc(HEX/Bin/IEEE754)는 개발자 계산기. Ratio.Forge는 **디자인 시스템 수치 (비율·타이포·그리드)** 전문.

- 타블렛(WinTab API) 연동 시 실제 펜압 감지

**기술**: Dynamic Time Warping(DTW) 경로 유사도, Catmull-Rom 스플라인 스무딩, 속도 기반 굵기 매핑
**매력**: 단순한 선 하나에서 오는 집중과 완성의 쾌감. 점수보다 "완벽한 획"을 위한 반복 수련 욕구.
**차별화**: 잉크 계열 게임(Ink.Spread·Ink.Trail·Fluid.Paint)은 전파·전략·자유 페인팅. Ink.Stroke는 **획의 운동 곡선 정밀 재현**이라는 완전히 다른 스킬.

### 9. `Farm.Flow` — 자동화 농장 파이프라인 퍼즐

**장르**: Simulation / Puzzle / Automation
**핵심 메커닉**: 밭·수확기·컨베이어·관개시설을 격자에 배치해 작물 재배→수확→가공→출하 파이프라인을 최적화하는 자동화 퍼즐. 타이밍 동기화와 병목 제거가 핵심.

**게임플레이**:
- 격자 맵 (16×16 ~ 32×32): 흙·모래·돌·물 타일 배치
- 시설 배치: 씨앗 투입기·관개 파이프·성장 가속기·수확 로봇·컨베이어·창고
- 작물 성장 사이클: 씨앗→새싹→성장→수확 타이머 (관개 ON = 속도 2×)
- 컨베이어 방향 설정: 분기·합류·분류 게이트 배치
- 병목 감지: 적재 초과 시 컨베이어 정체→작물 손실 경고
- 목표: N턴 이내 X kg 수확·출하 (별점 3개)
- 계절 이벤트: 비(물 자동 공급)/가뭄(관개 비용 2×)/서리(성장 중단) 랜덤
- 레벨 에디터 + 친구에게 맵 공유

**기술**: 이산 이벤트 시뮬레이션, 그래프 흐름 분석, 병목 감지 큐 오버플로 모델
**매력**: 연결 완성 시 매끄럽게 흘러가는 파이프라인 시각화. 병목 발견→수정의 반복 최적화 루프 중독성.
**차별화**: Auto.Build(테트로미노 자동화 퍼즐)는 도형 배치. Colony.Sim(개미 군집)은 생물 AI. Farm.Flow는 **농업 생산 체인 시간 동기화 퍼즐**로 고유 도메인.

### 10. `Plat.Flip` — 밟으면 뒤집히는 발판 플랫포머

**장르**: Puzzle / Platformer / Action
**핵심 메커닉**: 모든 발판이 밟으면 뒤집혀 반대쪽 면으로 변한다. 앞면은 견고하고 뒷면은 투명(낙하). 동일 발판을 두 번 밟으면 원래대로 돌아오는 반전 메커닉을 이용해 경로를 설계.

**게임플레이**:
- 캐릭터: 달리기·점프·더블점프 기본 이동
- 발판 타입: 일반(밟으면 뒤집힘)·고정(변하지 않음)·연쇄(인접 발판 도미노 뒤집힘)·시한(N초 후 자동 리셋)
- 목표: 모든 코인 수집 + 출구 도달
- 도전 과제: "발판 최소 뒤집기 수로 클리어" → 역산 퍼즐 설계
- 경쟁 모드: 두 플레이어가 같은 맵 — 상대가 뒤집은 발판이 나에게 영향 (로컬 2P)
- 발판 기억: 현재 뒤집힌 상태를 색상으로 표시 (퍼즐 상태 추적)
- 되감기: 마지막 5개 액션 취소 (전략 재시도)
- 100+ 레벨 + 커뮤니티 레벨 에디터

**기술**: 2D 물리 엔진(AABB 충돌), 발판 상태 비트마스크, BFS 솔버 (최소 뒤집기 수 계산)
**매력**: "이 발판을 밟으면 저게 뒤집힌다"는 연쇄 예측이 만드는 독특한 공간 사고. 익숙한 플랫포머에 반전 퍼즐을 덧입힌 신선함.
**차별화**: Phase.Gate(차원 전환 발판)는 두 차원 동시 존재. Jelly.Jump(소프트바디)는 변형 피지컬. Plat.Flip은 **발판 상태 뒤집기 전략**이 핵심으로 다른 메커닉.
