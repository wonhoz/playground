### 1. `Dict.Cast` — 완전 오프라인 사전 팝업 (S티어)

**경로 예정**: `Applications/Tools/Productivity/Utility/Dict.Cast/`

#### 핵심 가치
Stay.Awake·Prompt.Forge처럼 **전역 단축키로 즉시 호출**, 화면 어디서나 단어 조회.
인터넷 없이, 브라우저 전환 없이 뜻·유의어·예문을 바로 확인.

#### 기능
- 영어 사전 (Wiktionary XML dump 로컬 파싱) + 한국어 사전 (공공데이터포털 KDic)
- 전역 단축키(Win+Shift+D) 플로팅 팝업 — 현재 선택 텍스트 자동 읽기 (SendMessage WM_COPY)
- 발음 기호(IPA) + SAPI 원클릭 발음 재생
- 유의어(Thesaurus) / 반의어 탭
- 예문 3~5개 표시
- 최근 조회 히스토리 (로컬 SQLite)
- 단어장 저장 + CSV/Anki 덱 내보내기 → **학습 도구로 활용, 수익 포인트**

#### 차별화
| 기존 | 차이 |
|------|------|
| `Spell.Cast` | 맞춤법 교정기, 사전 조회 없음 |
| `Echo.Text` | TTS 출력기, 뜻 조회 없음 |
| `Quick.Memo` | 메모장, 검색 없음 |

#### 수익 시나리오
- 무료: 기본 영어사전
- 프리미엄: 한국어 사전 + 유의어 + Anki 내보내기 ($4.99 일회성)
- 번들: 단어장 클라우드 동기화 ($1.99/월)

### 2. `Tag.Forge` — MP3/FLAC 태그 일괄 편집기 (S티어)

**경로 예정**: `Applications/Tools/Productivity/Media/Tag.Forge/`

#### 핵심 가치
대용량 음악 라이브러리 소유자의 **만성 고통** 해결.
Music.Player로 재생은 하지만 태그가 엉망이면 재생 목록·정렬이 전부 망가짐.

#### 기능
- MP3(ID3v2.4) / FLAC(Vorbis Comment) / OGG / M4A 지원 (TagLib-Sharp)
- 폴더 드래그&드롭 → 트랙 목록 일괄 편집 (그리드 인라인 편집)
- **파일명 ↔ 태그 양방향 규칙 동기화**
  예) `트랙번호 - 아티스트 - 제목.mp3` ↔ Tag 자동 매핑
- **MusicBrainz API 연동** — 선택 트랙 자동 메타데이터 조회 (온라인 선택)
- 앨범아트 드래그 임베드 + 일괄 적용
- 트랙 번호 자동 순번 재정렬
- 오프라인 우선 (MusicBrainz는 opt-in)

#### 차별화
| 기존 | 차이 |
|------|------|
| `Music.Player` | 재생만, 태그 편집 없음 |
| `Pitch.Find` | BPM/키 분석만, 태그 저장 없음 |
| `Batch.Rename` | 파일명만, ID3 내부 태그 수정 불가 |

#### 수익 시나리오
- 무료: 단일 파일 편집, 50곡 이하 배치
- 프리미엄: 무제한 배치 + MusicBrainz 자동 조회 ($6.99 일회성)

### 3. `Folder.Sync` — 로컬 폴더 동기화 GUI (A티어)

**경로 예정**: `Applications/Tools/Productivity/Files/Folder.Sync/`

#### 핵심 가치
robocopy/xcopy 명령어를 모르는 사람도 폴더 백업·동기화를 GUI로.
FreeFileSync 오픈소스가 있지만 UI가 복잡하고 무거움 → **경량·미니멀 대체**.

#### 기능
- 원본 ↔ 대상 폴더 쌍 등록 (다중 프로파일)
- 단방향(백업) / 양방향(미러) / 미러(삭제 포함) 모드 선택
- **실행 전 미리보기**: 추가/수정/삭제 예정 파일 트리 diff 표시
- 파일 필터 (확장자, 숨김, 크기 임계치)
- 실행 결과 로그 (성공/실패 파일, 소요 시간)
- 스케줄러 연동 (Windows Task Scheduler 자동 등록)
- 충돌 파일 처리 정책 선택 (최신 우선 / 원본 우선 / 수동 결정)

#### 차별화
| 기존 | 차이 |
|------|------|
| `Drop.Bridge` | LAN 기기 간 전송, 로컬 폴더 동기화 아님 |
| `File.Guard` | 변경 감사 로거, 동기화 아님 |
| `Batch.Flow` | 파일 처리 파이프라인, 양방향 동기화 아님 |

#### 수익 시나리오
- 무료: 2개 폴더 쌍, 스케줄러 없음
- 프리미엄: 무제한 프로파일 + 스케줄러 + 이메일 리포트 ($5.99)

### 4. `Layout.Forge` — 키보드 키 재배치 프로파일 에디터 (B티어)

**경로 예정**: `Applications/Tools/Dev/System/Layout.Forge/`

#### 핵심 가치
Caps Lock → Ctrl, 한영키 → 한/영, 방향키 재배치 등을
레지스트리(Scancode Map)나 AutoHotKey 없이 **GUI로 시각적 편집 + 프로파일 전환**.

#### 기능
- ANSI 104키 / ISO 105키 레이아웃 시각적 편집 (Key.Map 캔버스 재활용 가능)
- 키 클릭 → 대상 키 선택 드롭다운 (무효화, 다른 키, 미디어 키, 유니코드 문자)
- 프로파일 저장/불러오기 (게임용, 코딩용, 기본)
- Windows Scancode Map 레지스트리 자동 생성/적용/복원
- 재부팅 없이 적용 가능한 항목은 즉시 반영 (SendInput 기반)
- 특수: Fn 레이어 시뮬레이션 (키 홀드 시 다른 키 코드로 변환)

#### 차별화
| 기존 | 차이 |
|------|------|
| `Mouse.Flick` | 마우스 제스처 → 단축키 매핑, 키보드 재배치 아님 |
| `Key.Map` | 단축키 시각화 도구, 키 재배치 기능 없음 |
| `Key.Tape` | 매크로 녹화기, 물리 키 스캔코드 변경 아님 |

### 5. `Sched.Cast` — Task Scheduler GUI 대체 (A티어)

**경로 예정**: `Applications/Tools/Dev/System/Sched.Cast/`

#### 핵심 가치
Windows 작업 스케줄러 UI는 2000년대 수준으로 불편함.
cron 표현식으로 쉽게, **실행 결과 알림**까지 한 번에.

#### 기능
- 예약 작업 목록 (Windows Task Scheduler API 래퍼)
- cron 표현식 입력 + Cron.Cast 스타일 실시간 파싱 ("다음 실행: X분 후")
- 스크립트(.ps1/.cmd/.bat)/실행 파일 등록
- 실행 성공/실패 → Windows Toast 알림 + 이메일(선택)
- 실행 이력 뷰어 (타임라인, 성공률 차트)
- 시작 조건 (로그인 시, 네트워크 연결 시, 배터리 미연결 시)
- 실행 중 작업 강제 종료

#### 차별화
| 기존 | 차이 |
|------|------|
| `Cron.Cast` | cron 표현식 검증기, 실제 작업 예약 안 함 |
| `Batch.Flow` | 파일 처리 파이프라인, 시간 기반 스케줄러 아님 |
| `Watch.Dog` | 프로세스 크래시 재시작, 일정 기반 실행 아님 |

#### 수익 시나리오
- 무료: 작업 5개
- 프리미엄: 무제한 + 이메일 알림 + 이력 30일 저장 ($4.99)

### 7. `Proc.Map` — 프로세스-네트워크 연결 시각화 (B티어)

**경로 예정**: `Applications/Tools/Dev/Network/Proc.Map/`

#### 핵심 가치
"이 앱이 어디에 접속하고 있나?" 를 실시간 그래프로.
보안 의식 있는 파워 유저, 개발자 타겟.

#### 기능
- 실행 중 프로세스 → 활성 TCP/UDP 연결 목록 (IP Helper API)
- 외부 IP → 지오로케이션 DB(MaxMind GeoLite2 로컬) → 국가 플래그 + 도시
- 프로세스 계층 트리 (부모-자식 관계, ppid 기반)
- **방향 그래프 시각화**: 프로세스 노드 → 연결 엣지 → 외부 IP 클러스터
- 의심 IP 자동 플래그 (블랙리스트 대조, 토르 출구 노드, 알려진 악성 IP)
- 특정 프로세스 연결 차단 (Windows Firewall 규칙 자동 생성)
- 스냅샷 저장 + 비교 (이전 상태와 새로 생긴 연결 강조)

#### 차별화
| 기존 | 차이 |
|------|------|
| `Port.Watch` | 로컬 포트 점유 프로세스, 원격 IP 시각화 없음 |
| `Net.Scan` | LAN 기기 탐지, 실행 중 프로세스 연결 추적 아님 |
| `App.Cage` | 방화벽 GUI 래퍼, 현재 연결 그래프 시각화 없음 |

### 8. `Font.Sub` — 웹폰트 서브셋 + WOFF2 변환기 (B티어)

- 글리프 미리보기 그리드 (포함/제외 문자 확인)
- CSS `@font-face` 코드 자동 생성 복사

### 1. `Warp.Drift` — 중력 렌즈 슈터 (S티어)

**경로 예정**: `Games/Arcade/Warp.Drift/`

#### 한 줄 설명
블랙홀이 탄환 궤도를 휘게 한다. 이 중력 왜곡을 **계산하고 이용해서** 적을 격파.

#### 게임플레이
- 우주 배경, 플레이어 함선 중앙 고정 (회전 조준, 클릭 발사)
- 맵 곳곳의 **블랙홀/중성자별** → 근처를 지나는 탄환을 굴곡 궤도로 휨
- 직선으로 안 닿는 적 → 블랙홀 중력을 활용한 **커브 샷**으로만 격파 가능
- 탄환 궤도 실시간 예측선 표시 (발사 전 궤도 미리보기)
- 적도 중력 영향 받음 → 예측 불가 기동
- 블랙홀 강도·위치 변화하는 스테이지 진행 (30+ 레벨)
- 파워업: 중력 무시 탄 / 스플릿 탄 / 블랙홀 임시 생성

#### 왜 재미있나
- Orbit.Craft 팬이라면 중력 역학 계산하는 쾌감 즉시 이해
- "아 이 각도로 쏘면 저 블랙홀 돌아서 맞겠다!" 직관적 유레카 순간 반복
- 탄환 궤도 예측선이 실시간으로 휘어가는 시각 효과 = 네온 시각 만족감

#### 기술
- `RK4 수치 적분`으로 탄환 N-체 중력 경로 계산 (Orbit.Craft 기술 재활용)
- WPF Canvas + `CompositionTarget.Rendering` 게임 루프

### 2. `Rift.Jump` — 포털 2D 순간이동 플랫포머 (S티어)

**경로 예정**: `Games/Puzzle/Rift.Jump/`

#### 한 줄 설명
화면 어디든 **마우스 클릭 두 번으로 균열 2개** 생성, 들어가면 반대편으로 나온다.
모멘텀이 보존된다 — 빠르게 들어가면 빠르게 나온다.

#### 게임플레이
- 키보드 이동(WASD), 마우스 좌클릭 = 균열 A, 우클릭 = 균열 B
- 균열을 통과하면 **속도 벡터 방향 + 크기 보존** → 수직 하강으로 들어가 수평으로 나오기 가능
- 퍼즐 요소: 고속 통과 필요 구간, 두 균열을 연속 통과하는 루프 가속, 시간 내 목표 도달
- 환경 인터랙션: 물·용암·얼음 위에는 균열 생성 불가
- 50+ 큐레이션 레벨 + 레벨 에디터 + 커뮤니티 공유
- 보스: 균열을 막아오는 에너지 실드 적

#### 왜 재미있나
- "모멘텀 보존" 한 규칙이 만들어내는 무한한 퍼즐 가능성
- Neon.Slice처럼 한 번 익히면 손이 멈추지 않는 리듬감
- 레벨 에디터 → UGC가 게임 수명을 무한 연장

#### 기술
- 2D 물리: 기본 중력 + 충돌 AABB
- 균열 렌더링: `RenderTargetBitmap` 포털 뷰 스티치

### 3. `Prism.Break` — 백색광 RGB 분리 퍼즐 (A티어)

**경로 예정**: `Games/Puzzle/Prism.Break/`

#### 한 줄 설명
백색광이 프리즘을 통과하면 R·G·B 세 빛으로 갈라진다.
배치된 거울·렌즈·색필터로 정확한 색의 빛을 목표에 도달시켜라.

#### 게임플레이
- 광원(백색/단색) → 격자 위에 프리즘·거울·볼록렌즈·색필터·색합성기 배치
- **색 혼합 물리**: R+G=Yellow, G+B=Cyan, R+B=Magenta, R+G+B=White
- 목표: 지정 수용체에 **정확히 일치하는 색** 전달
- 레이저 빔이 실시간으로 반사·굴절·분리되는 네온 시각 효과
- 후반 레벨: 빛 세기(밝기) 개념 추가 → 두 빔을 합쳐야 충분한 밝기 달성
- 타임 어택 모드 (초 안에 경로 완성)

#### 왜 재미있나
- Laser.Net이 '경로 연결'이라면 Prism.Break는 '색 계산 + 경로'
- Hue.Flow 팬이 좋아할 색상 기반 퍼즐 + Orbit.Craft 팬이 좋아할 물리 예측
- "빨강+파랑을 섞으면 보라" 직관을 활용하는 순간의 만족감

#### 기술
- 레이 캐스팅 (격자 셀 단위 전파)
- 색 가산 혼합: RGB float 채널별 누적

### 4. `Atom.Craft` — 원소 결합 분자 합성 퍼즐 (B티어)

**경로 예정**: `Games/Puzzle/Atom.Craft/`

#### 한 줄 설명
원소 원자를 드래그해 결합시키면 분자가 된다.
H₂O, CO₂, NH₃... 목표 분자를 만들어라.

#### 게임플레이
- 격자 위에 원소 원자(H, O, C, N, Na, Cl 등) 배치
- 원자 간 결합: 단일/이중/삼중 결합 클릭으로 선택
- **원자가 규칙**: H=1, O=2, C=4, N=3 → 결합 수 초과 시 불안정(빨간 경고)
- 목표 분자 완성 시 3D 회전 모델로 클리어 연출
- 진행: H₂→H₂O→CH₄→NH₃→C₆H₁₂O₆(포도당) 순으로 난이도 상승
- 보너스: 실제 물질 이름 + 용도 설명 팝업 ("이것은 암모니아, 비료의 원료")
- 자유 모드: 원소 무제한 배치해서 실제 분자 만들기 샌드박스

#### 왜 재미있나
- 화학 지식 없어도 원자가 규칙 하나로 즐길 수 있음
- 학습 효과 덤 (학교·학원 교육 도구로 수익화 가능)
- Sand.Fall처럼 "규칙이 단순, 결과가 다양"한 구성

#### 수익 시나리오
- 무료: 50레벨
- 교육 라이선스: 학급 단위 구매 → **B2B 수익** (학원, 학교)

### 5. `Maze.Dread` — 절차 생성 미로 탈출 (B티어)

**경로 예정**: `Games/Arcade/Maze.Dread/`

#### 한 줄 설명
매번 새롭게 생성되는 미로. 빛은 작은 원형 시야뿐.
그리고 무언가가 쫓아오고 있다.

#### 게임플레이
- Wilson's 알고리즘 기반 미로 절차 생성 (무한 변형)
- 탑뷰, 주변 시야 제한 (원형 가시 반경, 미니맵 없음)
- 추격자 AI: Dijkstra 경로 탐색, 소리 반응 (달리면 소리, 걸으면 조용)
- 아이템: 플레어(일시 조명), 함정 설치, 시야 확대
- 탈출구 도달 = 클리어 / 추격자에 잡히면 게임 오버
- 매일 랜덤 시드 챌린지 맵 (전 세계 동일 미로, 최단 시간 기록)
- 난이도: 미로 크기·추격자 수·시야 반경 조절

#### 왜 재미있나
- Dodge.Blitz처럼 '생존' 긴장감 + Sand.Fall처럼 매번 다른 환경
- 시야 제한이 만드는 심리적 공포감 (소리 효과만으로도 충분)
- 데일리 챌린지가 재방문 유도

### 6. `Colony.Sim` — 군집 생명체 진화 시뮬레이션 (A티어)

**경로 예정**: `Games/Simulation/Colony.Sim/`

#### 한 줄 설명
세포·개미·균류... 군집 생명체들이 영양소를 놓고 경쟁한다.
조건만 설정하고 지켜보면 예상치 못한 패턴이 등장한다.

#### 게임플레이 / 샌드박스
- 격자 기반 세계: 영양소 농도 확산 (Fick's Law 근사)
- 3~5종 군집 동시 진행: 각 군집 고유 이동 전략(랜덤/그라디언트/페로몬)
- **페로몬 신호**: 먹이 발견 시 경로에 화학물질 마킹 → 동료가 따라옴
- 종 간 경쟁: 충돌 시 크기/수로 흡수 또는 회피
- 사용자 인터랙션: 영양소 붓기, 독소 장벽 그리기, 특정 군집 강화
- 통계 패널: 군집별 개체 수 시계열 그래프, 영역 점유율 히트맵
- 도전 모드: "이 조건에서 파란 군집을 30초 이내 우점종으로 만들어라"

#### 왜 재미있나
- Sand.Fall의 "무한히 관찰하게 되는" 창발 현상
- Life.Sim(Conway)보다 훨씬 다채롭고 능동 인터랙션 있음
- Orbit.Craft처럼 물리/수학 기반이지만 직관적으로 아름다움

### 7. `Hex.Storm` — 육각 격자 마이크로 RTS (B티어)

**경로 예정**: `Games/Strategy/Hex.Storm/`

#### 한 줄 설명
헥사곤 격자 위에서 펼쳐지는 가장 빠른 실시간 전략 게임.
유닛 생산-이동-전투 사이클을 **30~90초 안에** 끝낸다.

#### 게임플레이
- 7~19 헥사 격자 (매우 작음 = 결정이 빠름)
- 자원 헥사 점령 → 자동 생산 → 인접 헥사 이동·전투
- 유닛 3종: 기병(빠름, 약함) / 보병(보통) / 포(강함, 느림)
- 전투: 육각 어드밴티지(높은 칸 공격력 +20%), 포위 보너스
- AI 3단계 (Easy/Hard/Godlike) — Hard부터 봇이 착각을 일으키는 페인트 기동
- **스피드런 챌린지**: 최소 턴 수로 맵 완전 점령
- 멀티: 로컬 핫시트 2P

#### 왜 재미있나
- 판이 작아서 "5분짜리 Chess" 느낌 → 쉽게 한 판 더
- Cell.War처럼 영역 전략이지만 유닛 종류와 지형이 있어 전술 깊이 있음
- 스피드런 요소가 재방문 동기
