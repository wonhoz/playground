### 1. `Key.Stream` — 강의·발표용 키입력 실시간 화면 오버레이

**카테고리**: Dev/Tools / Productivity
**핵심 기능**: 스크린캐스트·강의 녹화 중 현재 누른 키 조합을 화면 하단에 실시간으로 표시해준다. Ctrl+C, Alt+Tab 같은 단축키가 영상에서 명확히 보여 시청자가 따라하기 쉬워진다.

**상세 기능**:
- 전역 키후킹으로 모든 키 조합 캡처 (WH_KEYBOARD_LL)
- 화면 하단/좌하/우하 위치 선택, 반투명 배경 오버레이
- 키 표시 지속 시간 설정 (0.5초~3초 페이드아웃)
- 마우스 클릭 이벤트도 표시 옵션 (LMB/RMB/Scroll 아이콘)
- 특수키 아이콘 렌더링 (⌘ Ctrl / ⇧ Shift / ↑ Up 등)
- 테마 선택: 다크/라이트/투명, 폰트 크기 조절
- 제외 키 목록 (비밀번호 입력 시 숨김)
- 트레이 상주, 단축키로 토글

**기술 스택**:
- WinForms/WPF 투명 오버레이 창 (TopMost + WS_EX_TRANSPARENT)
- `SetWindowsHookEx(WH_KEYBOARD_LL)` 전역 훅
- 키 아이콘 렌더링: WPF DrawingVisual or SkiaSharp
- 트레이: NotifyIcon

**수익 모델**: 완전 무료 오픈소스
**차별화**: Key.Test는 하드웨어 키 테스트 도구, Key.Heat는 히트맵 분석기. Key.Stream은 실시간 화면 표시 전용 — 스크린캐스터를 위한 프레젠테이션 보조 도구.

### 2. `OAuth.Peek` — JWT/OIDC/OAuth2 토큰 로컬 디코더 + 플로 시각화

**카테고리**: Dev/Tools / Security
**핵심 기능**: JWT 토큰을 붙여넣으면 헤더·페이로드·서명을 즉시 디코딩하고, OIDC Discovery 엔드포인트 연동으로 JWK 서명 검증, OAuth2 Authorization Code / Client Credentials 플로 시퀀스 다이어그램을 시각화한다.

**상세 기능**:
- JWT 즉시 디코딩 (HS256/RS256/ES256 지원)
- JWK 엔드포인트 연동 서명 검증 + 만료 시간 카운트다운
- OIDC Discovery (/.well-known/openid-configuration) 파싱 + 엔드포인트 목록 표시
- OAuth2 플로 다이어그램 시각화 (Authorization Code / PKCE / Client Credentials / Device Code)
- 클레임별 설명 툴팁 (sub, iss, aud, exp 등 RFC 7519 표준)
- Bearer/API Key/Basic Auth 헤더 빠른 생성 및 클립보드 복사
- 토큰 히스토리 (최근 10개) 로컬 저장
- Base64URL 인코딩/디코딩 보조 도구

**기술 스택**:
- `System.IdentityModel.Tokens.Jwt` (JWT 파싱)
- `Jose-JWT` 또는 `Microsoft.IdentityModel.JsonWebTokens`
- WPF FlowDocument 시퀀스 다이어그램 렌더링
- `HttpClient` OIDC Discovery 조회

**수익 모델**: 완전 무료
**차별화**: Text.Forge는 JWT 디코딩이 12가지 기능 중 하나. Cert.View는 X.509 인증서. OAuth.Peek는 OAuth2/OIDC 에코시스템 전체를 다루는 전문 도구.

- 드래그앤드롭 백로그→스프린트 계획·할당

- 로컬 팀원 프로필 (이름+아바타 색상)

**기술 스택**:
- WPF MVVM, LiveCharts2 (번다운·Velocity 차트)
- `System.Text.Json` 직렬화
- WPF DragDrop API

**수익 모델**: 완전 무료
**차별화**: Task.Cast는 순수 Kanban(기간·포인트 없음), Cal.Block은 시간 블로킹, Timeline.Craft는 타임라인 편집기. Plan.Board는 스프린트 계획+번다운이라는 애자일 특화 기능 집합.

### 4. `Prompt.Lab` — LLM 프롬프트 체인 노드 에디터

**카테고리**: AI / Dev/Tools
**핵심 기능**: 여러 LLM 호출을 노드-엣지 그래프로 연결해 프롬프트 파이프라인을 시각적으로 설계한다. 각 노드의 출력이 다음 노드의 입력 변수로 자동 주입되며, 전체 체인을 단일 실행 버튼으로 돌릴 수 있다.

**상세 기능**:
- 드래그 가능한 노드 캔버스 (노드 = 프롬프트 템플릿 + 모델 선택)
- `{{변수}}` 플레이스홀더 → 이전 노드 출력 또는 사용자 입력으로 자동 치환
- 노드 종류: LLM 호출 / 텍스트 가공 / 조건 분기 / JSON 파싱 / 출력 저장
- Claude API (Anthropic), OpenAI, Ollama 로컬 모델 연동
- 실행 시 각 노드 상태(대기/실행중/완료/오류) 색상 하이라이트
- 체인 템플릿 저장/불러오기 (JSON)
- 병렬 노드 실행 지원 (await Task.WhenAll)
- 토큰 비용 집계 표시

**기술 스택**:
- WPF Canvas 노드 에디터 (드래그·연결선 베지어 곡선)
- Anthropic SDK (claude-sonnet-4-6), `HttpClient` OpenAI API
- `System.Text.Json` 체인 직렬화

**수익 모델**: 무료 (API 키는 사용자 제공)
**차별화**: Diff.Prompt는 A/B 비교, Prompt.Forge는 라이브러리 저장. Prompt.Lab은 여러 호출을 그래프로 연결하는 시각적 체인 빌더.

### 5. `Env.Diff` — .env / config 파일 구조 비교기 + 누락 키 감지

**카테고리**: Dev/Tools
**핵심 기능**: `.env`, `appsettings.json`, `config.ini` 두 파일(또는 두 환경 — dev/prod)을 나란히 비교해 누락된 키, 값이 다른 키, 시크릿 마스킹 처리를 한 번에 보여준다. "이 환경에는 있는데 저 환경에는 없다"는 실수를 방지한다.

**상세 기능**:
- .env / JSON / INI / YAML / TOML 지원 (파일 자동 감지)
- 3열 비교: 왼쪽 파일 | Diff 상태 | 오른쪽 파일
- 누락 키(왼쪽만·오른쪽만·양쪽 모두) 색상 코딩
- 값 차이 인라인 하이라이트 (문자열 diff)
- 시크릿 마스킹 (`PASSWORD`, `SECRET`, `KEY`, `TOKEN` 포함 키 자동 마스킹)
- 누락 키 보고서 텍스트/JSON 내보내기
- .env 템플릿(키만 있는 .env.example) 자동 생성
- 드래그앤드롭 파일 열기

**기술 스택**:
- `DotNetEnv` (.env 파싱), `YamlDotNet`, `System.Text.Json`
- `DiffPlex` 값 차이 비교
- WPF 3열 DataGrid

**수익 모델**: 완전 무료
**차별화**: Env.Guard는 Windows 시스템 환경변수 GUI. Schema.Diff는 SQLite/OpenAPI 스키마 구조 비교. Env.Diff는 환경별 config 파일 키-값 비교 전용.

### 6. `Shell.Book` — PowerShell / Bash 명령어 히스토리 북마크 GUI

**카테고리**: Dev/Tools / Productivity
**핵심 기능**: PowerShell / bash 히스토리 파일을 자동 파싱해 명령어를 태그·카테고리로 북마크하고, 전역 단축키로 호출하는 검색 팝업을 띄운다. 자주 쓰지만 매번 찾는 긴 명령어를 즉시 클립보드에 복사한다.

**상세 기능**:
- `PSReadLine` 히스토리 파일 (`ConsoleHost_history.txt`) 자동 로드
- `~/.bash_history` / `~/.zsh_history` 지원
- 즐겨찾기 등록 + 사용자 정의 별칭 메모
- 태그 (git/docker/npm/python 등 자동 감지)
- 전역 팝업 검색 (Ctrl+Shift+H 기본) — 입력 즉시 퍼지 검색
- 클립보드 복사 또는 새 터미널 창 직접 실행 옵션
- 히스토리 자동 새로고침 (FSW 감시)
- 프리셋 컬렉션 JSON 내보내기·가져오기

**기술 스택**:
- WPF, `FileSystemWatcher`
- `SetWindowsHookEx` 전역 단축키
- 퍼지 검색: FuzzySharp 또는 직접 구현 (Levenshtein)

**수익 모델**: 완전 무료
**차별화**: Alias.Forge는 별칭 설정 전용. Shell.Book은 히스토리 전체를 팝업 검색+북마크 라이브러리로 관리.

### 7. `Dock.Cast` — 작업 표시줄 앱 그룹·프리셋 런처 관리자

**카테고리**: System / Productivity
**핵심 기능**: Windows 작업 표시줄 핀 구성을 여러 "프리셋"으로 저장해 상황(개발 모드/미팅 모드/집중 모드)별로 즉시 전환한다. 앱 그룹 아이콘, 빠른 실행 스크립트, 자동 전환 스케줄을 지원한다.

**상세 기능**:
- 작업 표시줄 핀 구성을 프리셋으로 스냅샷·복원 (Shell AppBar 또는 레지스트리)
- 프리셋별 앱 자동 시작·종료 (예: 미팅 모드 → Zoom 시작, Slack 활성화)
- 사용자 정의 아이콘 + 단축키 트리거
- 시간 기반 자동 프리셋 전환 스케줄러
- 트레이 아이콘 우클릭 → 즉시 프리셋 전환
- 현재 실행 앱 목록을 프리셋에 포함하는 "현재 상태 캡처" 버튼
- 다중 모니터 프리셋 지원

**기술 스택**:
- WinForms 트레이 앱
- `Shell32.dll` + `IStartMenuPinnedList` (핀 제어)
- `Microsoft.Win32.Registry` (작업 표시줄 설정 레지스트리)
- `Process.Start`, `Process.Kill`

**수익 모델**: 완전 무료
**차별화**: Run.Deck는 프로젝트 단위 런처. Tile.Cast는 창 타일링. Dock.Cast는 작업 표시줄 핀 구성 전체를 프리셋으로 관리·전환.

### 8. `Tab.Host` — 임의 앱 창을 탭 컨테이너에 임베드하는 멀티탭 그룹 관리자

**카테고리**: System / Productivity
**핵심 기능**: 현재 실행 중인 임의의 Windows 앱 창(브라우저·IDE·탐색기 등)을 드래그&드롭으로 탭 컨테이너 창 안에 임베드한다. Windows Terminal의 탭 UX를 모든 앱에 적용하는 셸 역할을 한다.

**상세 기능**:
- 실행 중인 창을 드래그 → Tab.Host에 탭으로 추가 (`SetParent` WinAPI)
- 탭 드래그로 순서 변경·그룹 분리·새 컨테이너 창으로 분리
- 키보드 단축키: Ctrl+Tab(다음), Ctrl+W(닫기), Ctrl+T(새 탭)
- 탭 레이아웃 저장·복원 (앱 경로 기준)
- 탭별 제목 커스텀 + 아이콘 오버라이드
- 핀/즐겨찾기 탭 상단 고정
- 투명 테두리 프레임 (선택한 탭 창만 포커스)
- 세션 자동 저장·복원

**기술 스택**:
- WPF, `SetParent(hwnd, containerHwnd)`, `MoveWindow`
- `EnumWindows` + `GetWindowLongPtr` (EXSTYLE)
- WPF TabControl 커스텀 ControlTemplate

**수익 모델**: 완전 무료
**차별화**: Win.Scope는 Z-order·투명도 조작기. Screen.Split는 화면 구역 스냅. Tab.Host는 여러 앱 창을 탭 UX 하나로 묶어 운영.

### 9. `Typo.Guard` — 코드베이스 기술 오타 감지기

**카테고리**: Dev/Tools
**핵심 기능**: `typos` / `codespell` 스타일의 기술 용어 오타 감지를 Windows GUI로 제공한다. 변수명·주석·문서 문자열에서 흔한 오타(teh→the, recieve→receive, seperate→separate)를 폴더 단위 일괄 탐지하고 원클릭 수정한다.

**상세 기능**:
- 폴더/파일 드래그앤드롭으로 전체 스캔
- 내장 오타 사전 (6,000+ 영어 기술 오타 패턴 — typos 데이터셋 기반)
- 사용자 정의 허용 목록 (도메인 용어, 브랜드명 제외)
- 파일 종류 필터 (.cs/.py/.md 등 선택)
- 인라인 diff 미리보기 후 선택적 수정
- .bak 백업 옵션
- CI/CD 연동용 CLI 모드 (exit code 반환)
- Git diff 모드: 변경된 파일만 스캔

**기술 스택**:
- WPF + 내장 오타 사전 JSON
- `DiffPlex` 미리보기
- 정규식 + 단어 경계 매칭
- `System.IO.FileSystemWatcher` 실시간 감시 모드

**수익 모델**: 완전 무료
**차별화**: Spell.Cast는 한국어·영어 일반 맞춤법 검사기. Str.Forge는 Find&Replace. Typo.Guard는 코드/기술 문서에 특화된 오타 감지 전용.

### 10. `Color.Walk` — Windows 11 시스템 색상 토큰 라이브 추출 + Mica/Acrylic 시뮬레이터

**카테고리**: Dev/Tools / Design
**핵심 기능**: Windows 11의 강조 색상·Mica·Acrylic·Fluent 색상 토큰을 실시간으로 추출해 CSS/XAML/JSON 코드로 복사하고, 임의 팔레트를 Mica 효과 미리보기 창에 적용해 앱 디자인을 시뮬레이션한다.

**상세 기능**:
- `UISettings.GetColorValue` 로 강조 색상·배경 토큰 실시간 읽기
- `DwmGetColorizationColor` Composition Color 추출
- Windows 11 색상 테마 팔레트 전체 (Light/Dark 11가지 토큰) 시각화
- 임의 강조 색상 입력 → Mica/Acrylic 미리보기 시뮬레이션 (WPF 렌더링)
- CSS 커스텀 프로퍼티 / Tailwind / WPF ResourceDictionary / WinUI 형식 내보내기
- 강조 색상 변경 이벤트 실시간 감지 (Windows 설정 연동)
- 색상 토큰별 WCAG 2.1 대비 비율 자동 계산
- 즐겨찾기 팔레트 저장

**기술 스택**:
- WPF, `Windows.UI.ViewManagement.UISettings`
- `DwmApi.DwmGetColorizationColor` P/Invoke
- WPF Backdrop API (Windows 11 Mica 효과 미리보기)

- 타임랩스 재생: 분열 전체 과정을 10× 고속 애니메이션

**기술**: 셀룰러 오토마타 + 규칙 DSL 인터프리터, BFS 분열 시뮬, WPF Canvas 세포 렌더링 (SkiaSharp)
**매력**: 단순한 규칙에서 복잡한 형태가 창발하는 경이감. 생물학 원리를 직접 발견.
**차별화**: Colony.Sim은 페로몬 개미 군집. Bloom.Craft는 세포 vs AI 격자 전략. Cell.Grow는 분열 규칙 자체를 편집하는 메타 시뮬레이션.

### 10. `Rune.Web` — 거미줄 회로망 절점 룬 배치 에너지 흐름 완성 퍼즐

**장르**: Puzzle / Logic / Fantasy
**핵심 메커닉**: 방사형 거미줄 격자의 각 절점에 룬 심볼(증폭·분기·소멸·변환)을 배치해 중심 에너지 원에서 출발한 흐름이 모든 외곽 수용체에 정확한 세기로 도달하게 한다.

**게임플레이**:
- 거미줄 구조: 동심원 N개 × 방사선 M개의 절점망
- 에너지 흐름은 절점 룬에 따라 분기·증폭·감쇄·방향 전환
- 룬 종류: 🔴 증폭(×2), 🔵 분기(흐름 3방향 분리), ⚫ 소멸(흐름 차단), 🟡 변환(세기 변환 공식)
- 목표: 각 수용체가 요구하는 에너지 세기(1~8)를 정확히 받아야 클리어
- 룬 배치 수 제한 (예: 5개 이하)
- 흐름 시뮬 버튼: 에너지 전파 애니메이션 (빛이 거미줄 타고 퍼지는 효과)
- 40개 스테이지 (단일 경로 → 복잡한 다중 분기)
- 에디터 모드: 커스텀 스테이지 제작 + 공유 코드

**기술**: 방사형 그래프 탐색 (BFS 에너지 전파), WPF Canvas 방사형 절점 렌더링, 애니메이션 타이머
**매력**: 거미줄이라는 시각적 독창성. 에너지가 퍼져나가는 애니메이션의 아름다움. 수학적 분기 계산의 퍼즐감.
**차별화**: Volt.Chain은 전류 스위치·AND/NOT 게이트. Net.Weave는 그래프 평면화 위상 퍼즐. Rune.Web은 방사형 에너지 흐름 분배라는 독자적 구조.
