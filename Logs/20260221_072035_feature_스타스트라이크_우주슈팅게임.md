# Star.Strike - 레트로 우주 슈팅 게임 개발

- **일시**: 2026-02-21 07:17 ~ 07:20 (KST)
- **태그**: `feature`
- **프로젝트**: `Games/Star.Strike`

## 작업 내용

키보드 방향키로 플레이하는 레트로 우주 슈팅 게임 개발.

### 구현 기능

| 기능 | 설명 |
|------|------|
| 게임 엔진 | `CompositionTarget.Rendering` 기반 고정 프레임 게임 루프 |
| 플레이어 | 방향키/WASD 이동, SPACE 발사, 피격 시 무적 시간 (2초 깜빡임) |
| 적 유닛 | Basic (빨강), Fast (주황), Tank (보라) 3종류 |
| 발사 체계 | 플레이어 탄 (청록), 적 탄 (빨강) 독립 발사 |
| 충돌 감지 | AABB 충돌: 탄-적, 탄-플레이어, 적-플레이어 |
| 웨이브 시스템 | 점진적 난이도 상승 (적 수 증가, 스폰 간격 감소, 상위 적 등장) |
| UI | 타이틀 화면, HUD (점수/라이프/웨이브), 일시정지(P), 게임오버, 하이스코어 |
| 이펙트 | 배경 별 패럴랙스 스크롤 (3레이어), 폭발 파티클 (적 색상별) |
| 다크 테마 | `#0D0D2B` 우주 배경, 네온 색상 UI, DwmSetWindowAttribute 다크 타이틀바 |

### 아키텍처

```
Engine/
  GameEngine.cs      - 게임 루프 (Start/Stop/Pause/Resume)
  InputManager.cs    - 키보드 입력 상태 추적
  GameObject.cs      - 엔티티 베이스 (위치, 충돌, 비주얼)
Entities/
  Player.cs          - 플레이어 우주선
  Enemy.cs           - 적 유닛 (Basic/Fast/Tank)
  Bullet.cs          - 총알 (플레이어/적 구분)
  Particle.cs        - 폭발 파티클
  Star.cs            - 배경 별
```

### 조작법

| 키 | 동작 |
|----|------|
| ↑↓←→ / WASD | 이동 |
| SPACE | 발사 |
| P | 일시정지/해제 |
| ENTER | 시작/재시작 |
| ESC | 타이틀로 돌아가기 |
