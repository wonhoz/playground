> **카테고리**: Tools / Dev / Data
> **수익 잠재력**: ★★★★★ (개발자 도구, API/SaaS 확장 용이)

### 4. `Cert.View` — X.509 인증서 체인 분석기
> **카테고리**: Tools / Dev / Network
> **수익 잠재력**: ★★★☆☆ (보안 엔지니어·DevOps 틈새 도구)

**핵심 기능**
- `.cer` / `.crt` / `.pem` / `.pfx` / `.p7b` 파일 드래그&드롭 로드
- 인증서 체인 시각화: Root CA → Intermediate CA → Leaf 계층 트리
- 각 인증서 상세: Subject / Issuer / SAN / 유효기간 / 서명 알고리즘 / 키 길이
- 유효기간 타임라인: 만료 D-day 색상 경고 (D-30 주황 / 만료 빨강)
- 도메인 직접 조회: `example.com:443` 입력 → TLS 핸드셰이크로 체인 수집
- OCSP / CRL 상태 확인 (온라인)
- PEM 텍스트 내보내기, 지문(SHA-256) 비교

**기술 스택**
- WPF, net8.0-windows, System.Security.Cryptography.X509Certificates, SslStream

**왜 수익이 나는가**
- 인증서 만료로 인한 서비스 장애는 DevOps 악몽 → 미리 시각화하는 도구에 실 수요
- OpenSSL CLI는 어려움 → GUI 대체
- 보안 감사·컴플라이언스 문서화용 인증서 체인 리포트 출력
