### 1. `Find.Fast` — 통합 파일·내용 검색기
**카테고리**: Files / Dev / Tools

**한 줄 설명**: Everything 스타일 즉시 파일명 검색 + ripgrep 스타일 내용 검색을 단일 UI에서.

**핵심 기능**:
- 파일명 검색: NTFS MFT 직접 읽기(Everything 방식) → 타이핑 즉시 결과
- 내용 검색: 정규식/리터럴 full-text grep (멀티스레드 병렬 처리)
- 탭 분리: "파일명" 탭 / "내용" 탭 / "최근 검색" 탭
- 결과 미리보기 패널: 텍스트 파일 내용, 이미지 썸네일, 바이너리 헥스
- 필터 패널: 확장자, 날짜 범위, 크기 범위, 경로 포함/제외 패턴
- 즐겨찾기 경로(루트 제한) 등록 — 특정 프로젝트 폴더 집중 검색
- 컨텍스트 메뉴: 탐색기 열기 / 경로 복사 / VSCode에서 열기 / 기본 앱으로 열기
- 전역 단축키(예: Alt+Space) 팝업 모드 지원

**수익/비전**: Everything은 파일명만, 에디터 내 검색은 IDE에 종속. 통합 버전은 개발자 일상 도구. Pro 버전(색인 저장, 네트워크 드라이브 지원) 과금 가능.

**기술**: C# / WPF, P/Invoke NTFS MFT, MemoryMappedFile grep, Virtual list panel

**수익/비전**: 개발자·엔지니어·요리사·여행자 모두 쓰는 만능 도구. 가볍고 오프라인 → Stay.Awake급 상시 의존 앱.

### 3. `Win.Event` — 경량 Windows 이벤트 로그 뷰어
**카테고리**: Dev / Tools / System

**한 줄 설명**: Windows EVT/EVTX 이벤트 로그 전용 경량 뷰어 — 기본 이벤트 뷰어의 느리고 복잡한 UI 대체.

**핵심 기능**:
- 로그 소스: 시스템 / 응용 프로그램 / 보안 / 사용자 정의 채널, 또는 EVTX 파일 직접 열기
- 실시간 스트리밍: 새 이벤트 즉시 추가 (EventLogWatcher)
- 즉각 필터: 레벨(정보/경고/오류/위험), EventID, 소스, 키워드, 시간 범위
- 정규식 메시지 검색 + 하이라이팅
- 색상 레벨 코딩: 경고=노랑, 오류=빨강, 위험=다홍 네온
- Toast 알림 규칙: 특정 EventID·레벨 발생 시 즉시 알림
- 결과 내보내기: CSV / JSON / EVTX 재저장
- 선택 이벤트 클립보드 복사 (버그 리포트용)

**차별점**: Log.Lens는 일반 텍스트 로그 파일 뷰어. Win.Event는 Windows ETW/EVTX 네이티브 로그 전용.

**수익/비전**: 서버 운영자·개발자·IT 관리자 필수 진단 도구. 빠른 속도 + 알림 규칙이 핵심 가치.

**기술**: C# / WPF, System.Diagnostics.Eventing.Reader, EventLogWatcher

**수익/비전**: 보고서 조립, 제안서 재활용, 계약서 섹션 맞춤형 구성 등 B2B 활용도 높음. 템플릿 라이브러리 Pro 과금 가능.

**기술**: C# / WPF, DocumentFormat.OpenXml, PdfSharpCore, WebView2

### 10. `Net.Ghost` — MAC 주소 스푸퍼 & 관리자
**카테고리**: Network / Privacy / Tray

**한 줄 설명**: 트레이 상주 NIC MAC 주소 랜덤화·프리셋 관리 — 공공 WiFi 프라이버시 보호.

**핵심 기능**:
- 인터페이스 목록 감지 (WiFi / 이더넷 / Bluetooth 어댑터)
- 원클릭 무작위 MAC 생성 (OUI 범위 선택: 로컬/유니캐스트 준수)
- 제조사 OUI 프리셋 선택 (Samsung, Apple, Intel 등으로 위장)
- 커스텀 MAC 프리셋 저장 (집 / 카페 / 사무실 프로파일)
- 재부팅 후 복원 지원 (레지스트리 `NetworkAddress` 값 관리)
- 원래 MAC으로 즉시 복원 원클릭
- 연결 네트워크 변경 시 자동 프로파일 전환 (SSID 기반)
- 현재 MAC·원본 MAC·OUI 제조사 정보 트레이 툴팁 표시

**차별점**: WiFi.Vault는 저장된 비밀번호 조회 도구. Net.Ghost는 MAC 레이어 신원 변경으로 완전히 다른 영역. 국내 미제안 도구.

**수익/비전**: 프라이버시 의식 증가 + 공공 WiFi 추적 이슈로 실수요 뚜렷. VPN 앱처럼 구독 과금 가능.

**기술**: C# / WinForms 트레이, Registry `HKLM\SYSTEM\CurrentControlSet\Control\Class`, NetSh / DeviceIoControl, WMI NetworkAdapter
