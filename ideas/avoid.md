# Avoid — 중복 방지 인덱스

> 브레인스토밍 전 아래 서브파일을 확인해 이미 제안·구현된 아이디어를 피한다.
> 마지막 갱신: 2026-03-19

---

## 서브파일 목록

| 파일 | 내용 | 항목 수 |
|------|------|---------|
| [`avoid/apps.md`](avoid/apps.md) | 구현 완료 앱 + 아카이브 | ~70개 |
| [`avoid/games.md`](avoid/games.md) | 구현 완료 게임 + 아카이브 | ~45개 |
| [`avoid/backlog.md`](avoid/backlog.md) | 미구현 아이디어 백로그 (앱·게임) | ~340개 |
| [`avoid/similar-groups.md`](avoid/similar-groups.md) | 유사 개념 그룹 (중복 주의) | ~130개 그룹 |

---

## 빠른 확인 가이드

1. **새 앱 아이디어** → `apps.md` + `backlog.md` (앱 섹션) + `similar-groups.md` 확인
2. **새 게임 아이디어** → `games.md` + `backlog.md` (게임 섹션) + `similar-groups.md` 확인
3. **비슷한 개념 있는지** → `similar-groups.md`에서 그룹명 키워드 검색
4. **구현 완료 여부** → `apps.md` 또는 `games.md` 확인

---

## 업데이트 규칙

- **채택 결정 즉시** 추가 — 구현 완료를 기다리지 않음
- 구현 완료 후 `backlog.md`에서 `apps.md` 또는 `games.md`로 이동
- 거절된 아이디어도 `backlog.md`에 추가 (재제안 방지)
- 유사 개념 그룹은 `similar-groups.md`에 추가
