### 1. BT.Cast — Bluetooth 기기 원클릭 전환 트레이

**카테고리**: System / Tray
**핵심 기능**: 헤드폰·스피커·게임패드 Bluetooth 기기를 Windows 설정 없이 트레이에서 즉시 연결·해제·전환

**상세 기능**:
- 트레이 클릭 → 현재 페어링된 모든 기기 목록 팝업 (연결 상태 표시)
- 기기 클릭 한 번으로 Connect/Disconnect 토글
- 기기 그룹 프리셋 (헤드폰 그룹 / 스피커 그룹 / 작업 모드 프리셋)
- 연결·해제 시 Toast 알림 (기기명 + 배터리 잔량)
- 자동 재연결: 마지막 연결 기기 기억, 시스템 재시작 후 자동 복원
- 기기별 볼륨 레벨 기억 (연결 시 자동 설정)

**기술 스택**:
- `Windows.Devices.Bluetooth` WinRT API
- `BluetoothDevice.FromBluetoothAddressAsync()` / `DeviceInformation.FindAllAsync()`
- `InTheHand.Net.Bluetooth` NuGet (32feet.NET) — 오디오 프로파일(A2DP) 연결
- WinForms NotifyIcon + DarkMenuRenderer

**수익 모델**: 무료 유틸리티 → GitHub Stars → 기부/Sponsor
**유사 앱과 차별화**: Windows 설정 > Bluetooth 화면에 진입할 필요 없음, 헤드폰 교체가 잦은 개발자·게이머에게 실질적 시간 절약

**카테고리**: Productivity / Tools
**핵심 기능**: 클립보드 전체 히스토리를 SQLite에 저장하고 빠른 전문 검색으로 재사용

**상세 기능**:

- SQLite FTS5 전체 텍스트 검색 (타이핑 중 실시간 필터)
- 핀 고정: 자주 쓰는 내용 영구 보관
- 태그 카테고리 (코드/URL/이메일/주소/기타)
- 자동 민감정보 감지: 신용카드 번호·비밀번호 패턴 → 마스킹 표시

- 전역 단축키 팝업 (Win+Shift+V 권장)

**기술 스택**:
- `AddClipboardFormatListener()` Win32 API

- WPF + 전역 `RegisterHotKey`

**차별화**: `Clipboard.Stacker`는 최근 N개 스택 순환 방식, `Clip.Find`는 무제한 히스토리 + 전문 검색 + 민감정보 보호

### 1. Word.Weave — 인접 글자 체인 단어 찾기

**장르**: Word / Puzzle / Casual
**핵심 메커닉**: N×N 격자에 배치된 글자 타일에서 인접(8방향) 타일을 체인으로 연결해 단어를 완성하는 Boggle 변형

**게임플레이**:
- 5×5 격자에 무작위 배치된 26개 영어 알파벳 타일
- 마우스 드래그로 인접 타일 체인 → 단어 완성 시 점수 획득
- 단어 길이에 따른 점수 배율 (4글자=x2, 5글자=x3, 희귀 단어=x5)
- 90초 타임어택 + 무제한 편안 모드
- 매일 새로운 일일 챌린지 시드
- 사용한 타일은 제거 → 위에서 새 타일 낙하 (연쇄 단어 기회)
- 특수 타일: 와일드카드(★), 배수기(2x), 폭탄(주변 타일 제거)

**기술**: 내장 영어 사전 (70,000단어 BinarySearch), 격자 DFS 경로 탐색
**매력**: Wordle 류 일상 두뇌 자극 + 어휘력 향상, 한국어 버전도 가능

### 2. Code.Idle — 개발자 스타트업 방치형 게임

**장르**: Idle / Incremental / Casual
**핵심 메커닉**: 코드 라인 자동 생산 → 기능 완성 → 앱 출시 → 사용자·수익 확보 → 팀 고용 → AI 자동화 루프

**게임플레이**:
- 시작: 나 혼자 코드 1라인/초 생산 (클릭 보너스)
- 단계별 성장:
  1. 개인 개발자 → 첫 앱 출시
  2. 팀 고용: 주니어 개발자 고용 (라인/초 증가)
  3. 기술 부채 관리: 쌓이면 버그 발생 → 수익 감소
  4. AI 코드 생성 도입: 지수 증가
  5. 스타트업 → 기업 상장
- 업그레이드 트리: 커피머신(+10% 속도) / 모니터 추가 / 깃허브 스폰서십
- 시장 이벤트: "Product Hunt 1위!" → 트래픽 폭발 → 서버 비용 급증
- 메타 유머: "기술 면접 통과" 이벤트, "NPM 의존성 취약점" 재난 이벤트
- 프레스티지: 회사 매각 → 스타 파워 누적 → 재시작 시 보너스

**기술**: 단순 숫자 누적형 (JavaScript-like timer loop in C#), WPF 애니메이션
**매력**: 개발자 자조 유머 + 방치형 특유의 숫자 성장 쾌감, 커뮤니티 공유 밈

### 3. Magnet.Jump — 자기 극성 전환 플랫포머

**장르**: Puzzle / Platformer
**핵심 메커닉**: 캐릭터가 N/S 극성을 보유하며 환경의 자기 발판에 흡착·반발해 이동하는 물리 플랫포머

**게임플레이**:
- 스페이스: 극성 전환 (N↔S)
- 파란 발판(N극): 같은 N극 반발(도약력), 반대 S극 흡착
- 빨간 발판(S극): 반대로 동작
- 혼합 퍼즐: 연속 극성 전환으로 미로 통과
- 특수 요소:
  - 회전 자기장 구역: 극성이 자동 반전
  - 자기 체인: 적을 극성으로 흡착해 발판으로 사용
  - 전자기 차폐 구역: 극성 비활성, 순수 점프만 가능
- 50개 레벨 + 레벨 에디터

**기술**: 벡터 자기력 거리²에 반비례 계산, Box2D 라이트 물리
**매력**: Magnet.Maze는 볼 유도 퍼즐, Magnet.Jump는 실시간 액션+퍼즐 조합의 플랫포머

### 4. Root.Spread — 지하 뿌리 성장 전략

**장르**: Strategy / Simulation / Casual
**핵심 메커닉**: 지하 타일맵에서 뿌리를 퍼뜨려 영양소를 수집하고 암석·경쟁 식물을 우회하는 전략 시뮬레이션

**게임플레이**:
- 타일맵 격자 (지하 단면뷰)
- 클릭으로 뿌리 성장 방향 지정 (영양소 비용 소모)
- 영양소 종류: 질소(성장 속도), 인(분기 능력), 칼륨(깊이 탐색)
- 장애물: 암석(우회 필요), 산성 토양(피해), 건조 구역(영양소 없음)
- 경쟁 요소: 다른 식물 뿌리 침범 → 자원 쟁탈
- 지상 연동: 뿌리 성장 → 지상 식물 키 성장 → 목표 높이 달성
- 절차 생성 맵 + 도전 레벨 (최소 뿌리 길이로 목표 달성)

**기술**: 가중치 BFS 경로 탐색, 영양소 확산 그리드 시뮬레이션
**매력**: Leaf.Grow는 L-시스템 지상 성장, Root.Spread는 지하 자원 관리 전략 — 완전히 다른 장르

### 5. Zipline.Rush — 집라인 네트워크 물리 퍼즐

**장르**: Puzzle / Physics / Engineering
**핵심 메커닉**: 앵커 포인트 사이에 집라인 케이블을 설치해 화물을 목표 지점까지 중력+마찰 물리로 정확히 배송

**게임플레이**:
- 2D 레벨: 출발점·목표점·앵커 가능 지점 배치
- 클릭으로 두 앵커 연결 → 집라인 설치
- 화물이 집라인 따라 이동 (각도·마찰·속도 물리 계산)
- 퍼즐 목표: 화물이 목표 도착 + 제한 시간 이내 + 최소 앵커 수
- 특수 요소:
  - 탄성 앵커: 충격 흡수 (속도 감소)
  - 도르래: 방향 전환
  - 무게추: 반대편 화물 당기기
  - 바람: 집라인 흔들림
- 레벨 에디터 + 커뮤니티 공유

**기술**: Verlet 케이블 시뮬레이션 (세그먼트 제약), 중력+마찰 적분
**매력**: 완전히 새로운 물리 퍼즐 메커닉, Bridge.Craft와 달리 동적 물체 이동이 핵심

### 6. Spore.Net — 균류 네트워크 성장 전략

**장르**: Simulation / Strategy / Puzzle
**핵심 메커닉**: 슬라임 몰드(Physarum polycephalum) 알고리즘 기반 균류 네트워크 성장으로 영양원을 연결하고 영역을 확장

**게임플레이**:
- 맵에 흩어진 영양원(먹이) 발견·연결
- 균류 실(Hypha): 영양원 향해 탐색 촉수 성장, 연결 시 네트워크 강화
- Physarum 알고리즘: 많이 통행한 경로는 두꺼워지고 덜 통행한 경로는 퇴화
- 클릭 개입: 성장 방향 힌트 제공, 장벽 설치
- 경쟁 모드: 두 균류 종이 같은 맵에서 영양원 경쟁
- 도전 모드: 최소 실 총 길이로 모든 영양원 연결 (최적 스패닝 트리 도전)
- 샌드박스: 자유 맵 편집 + 슬라임 성장 관찰

**기술**: 에이전트 기반 Physarum 근사 시뮬레이션 (Chemoattractant + 페로몬 맵)
**매력**: 실제 생물 알고리즘 기반 고유 메커닉, Colony.Sim(곤충 군집)과 전혀 다른 심미적 체험

### 7. Echo.Drum — 드럼 패턴 기억·재현 리듬 게임

**장르**: Rhythm / Music / Memory
**핵심 메커닉**: 드럼 패턴을 듣고 정확히 재현하는 청각 기억력 리듬 게임

**게임플레이**:
- 8박자 드럼 패턴 재생 (킥/스네어/하이햇/심벌 4채널)
- 플레이어가 동일 패턴을 타이밍 맞춰 입력 (Q/W/E/R 키)
- 난이도 증가: 8박자 → 16박자 → 폴리리듬 (7/8박자 등)
- 템포 증가: BPM 80 → 200
- 특수 도전:
  - 블라인드 재현: 패턴 한 번만 듣고 입력
  - 노이즈 혼입: 틀린 소리가 섞인 패턴에서 원본 추출
  - 레이어 분리: 복합 패턴에서 특정 채널만 재현
- 자유 창작 모드: 드럼 시퀀서로 비트 작곡 + 공유 코드
- 리더보드: 최고 연속 재현 기록

**기술**: NAudio (PCM 드럼 샘플 믹싱), 타이밍 정확도 ms 판정
**매력**: Sound.Grid는 멜로디 격자 창작, Echo.Drum은 청각 기억+리듬 재현 — 다른 재미

### 8. Block.Shift — 색상+기호 조합 슬라이딩 타일 퍼즐

**장르**: Puzzle / Casual
**핵심 메커닉**: 각 타일이 색상(3종)+기호(4종) 조합을 가지며, 행·열 전체 슬라이드로 목표 패턴을 완성

**게임플레이**:
- 4×4 격자, 타일마다 색(Red/Blue/Green) × 기호(Circle/Triangle/Star/Moon) 조합 (12가지)
- 조작: 행 전체 왼쪽/오른쪽 슬라이드, 열 전체 위/아래 슬라이드 (루비큐브 2D 버전)
- 목표: 지정 패턴 배치 달성 (색 기준, 기호 기준, 또는 둘 다)
- 이동 횟수 제한으로 별점 3개 도전
- 특수 타일:
  - 고정 타일: 슬라이드 불가
  - 와일드 타일: 어떤 색/기호도 될 수 있음
  - 회전 타일: 슬라이드 시 90° 회전
- 절차 생성: 목표에서 역으로 N번 랜덤 슬라이드하여 퍼즐 생성 (유일해 보장)
- 레벨 에디터 + 일일 챌린지

**기술**: 순열 상태 BFS 최단 해법 검증, WPF 슬라이드 애니메이션
**매력**: Ice.Slide(단방향 미끄럼)와 달리 행/열 전체 루프 슬라이드 — 큐브 퍼즐 느낌

### 9. Light.Cast — 볼록·오목 렌즈 빛 집중 퍼즐

**장르**: Puzzle / Optics
**핵심 메커닉**: 볼록렌즈·오목렌즈·반사경을 배치해 광원의 빛을 목표물에 집중시키는 빛 세기 기반 퍼즐

**게임플레이**:
- 광원: 고정 위치에서 평행광 방출
- 목표: 특정 타겟에 필요 광도 이상 집중 (온도계처럼 게이지 표시)
- 렌즈 배치:
  - 볼록렌즈: 빛 수렴 (초점 거리 내 최대 강도)
  - 오목렌즈: 빛 발산 (면적 확장, 세기 감소)
  - 반사경: 방향 전환 (각도 조절 가능)
  - 반투명 필터: 일부 통과·일부 산란
- 다중 목표: A=강하게, B=약하게 동시 달성
- 제약: 렌즈 개수 제한, 배치 불가 구역
- 방해 요소: 열에 약한 보호 대상 물체 → 지나치게 집중시 파괴 패널티
- 차별화: Laser.Net은 레이저 반사 경로, Prism.Break은 색 분리, Light.Cast는 빛 세기/열 집중

**기술**: 광선 레이캐스팅 + 굴절 스넬의 법칙 2D 근사, 세기 누적 계산
**매력**: 물리 교육적 요소 + 퍼즐 창의성

### 10. Sky.Drift — 상승기류 타기 글라이더 아케이드

**장르**: Arcade / Physics / Casual
**핵심 메커닉**: 글라이더 날개 물리와 열상승기류(Thermal)를 이용해 최대 거리·고도를 달성하는 수직 무한 스크롤러

**게임플레이**:
- 화면: 아래에서 위로 스크롤 (높이 = 점수)
- 조작: 좌우 키로 기울기 제어, 상승기류에 진입하면 자동 상승
- 물리:
  - 날개 양력: 속도×받음각에 비례
  - 중력: 지속 하강
  - 열상승기류(Thermal): 따뜻한 지면 위 상승 기류 (시각적 아지랑이 표시)
  - 하강기류: 피해야 하는 구름·장벽 아래
- 장애물: 새떼(방향 변경 필요), 폭풍구름(난류), 강풍 구역
- 진행: 높이 올라갈수록 기류 더 복잡, 새로운 지형(산맥/폭풍/야간)
- 최고 고도·비행 거리 일일 리더보드
- 글라이더 스킨 커스터마이징

**기술**: Verlet 날개 물리 + 기류 벡터 필드, 절차 생성 스크롤 환경
**매력**: Neon.Run은 수평 장애물 러너, Sky.Drift는 수직 상승+유체 물리 — 명상적이고 독창적인 리듬
