# Auto.Build — 자동화 공장 퍼즐 게임 구현

**날짜**: 2026-03-10
**태그**: feature
**상태**: 완료

---

## 개요

`Games/Puzzle/Auto.Build` — 컨베이어 벨트와 기계 장치를 배치해 자동화 라인을 구성하는 WPF 퍼즐 게임.

---

## 구현 내용

### 프로젝트 구조
- `Auto.Build.csproj` — net10.0-windows, WPF, SelfContained, PublishSingleFile
- `App.xaml` / `App.xaml.cs` — DwmSetWindowAttribute 다크 타이틀바 전역 적용
- `MainWindow.xaml` — 다크 그린 테마 (#0F1A0F), 3열 레이아웃 (도구팔레트/게임캔버스/정보패널)
- `MainWindow.xaml.cs` — 전체 게임 로직 (~400줄)
- `Resources/app.ico` — 멀티사이즈 (256/48/32/16px), 컨베이어+기어 디자인

### 게임 기능
- **기계 타입** 7종: Spawner, Belt(4방향), Processor, Sorter, Merger, Collector
- **아이템 색상** 5종: Red(원료), Green(원료), Blue(원료), Yellow(가공품), Purple(합성품)
- **시뮬레이션**: DispatcherTimer 틱 기반, 0.5×/1×/2×/4× 속도 조절
- **5개 스테이지**: 기초 이송 → 가공 체인 → 색상 분류 → 합성 라인 → 종합 공장
- **샌드박스 모드**: 자유 배치 및 무한 실행
- **그리드**: 12×8 (CELL=58px), Canvas 직접 렌더링

### 업데이트 파일
- `Playground.slnx` — Games/Puzzle 폴더에 Auto.Build 항목 추가
- `+publish-all.cmd` — Games/Puzzle 섹션에 Auto.Build 추가
- `+publish.cmd` — 메뉴 #4 Auto.Build 삽입, 4→5부터 68→69 리넘버링 (총 69개)
