### 1. `MIDI.Forge` — 오프라인 MIDI 시퀀서 & 피아노롤 에디터

**카테고리**: Audio / Creative
**핵심 기능**: 외부 DAW 없이 Windows에서 완전 오프라인으로 MIDI 음악을 작성·편집할 수 있는 경량 피아노롤 에디터. Score.Cast(뷰어 전용)와 달리 직접 음표를 그리고 트랙을 편집한다.

**상세 기능**:
- 피아노롤 그리드에서 마우스 드래그로 음표 추가·이동·길이 조절
- 다중 트랙 레이어 (멜로디/화음/베이스/드럼 각 레인)
- SF2/SF3 사운드폰트 소프트 신스 내장 재생 (NAudio 또는 DryWetMidi)
- Velocity·Pitch Bend·CC 자동화 레인 편집
- 템포·박자 변경, 구간 반복·루프 재생
- MIDI (.mid) / MusicXML 가져오기·내보내기
- 피아노 건반 실시간 MIDI 키보드 입력 지원

**기술 스택**:
- DryWetMidi (음표 편집, MIDI 파싱/내보내기)
- NAudio + SF2 소프트 신스 (MidiSoundPlayer)
- WPF Custom Canvas (피아노롤 직접 렌더링)
- NFluidssynth 또는 내장 SF2 렌더러

**수익 모델**: 무료 오픈소스 (Playground 프로젝트)
**차별화**: Score.Cast는 MusicXML 뷰어+재생 전용; MIDI.Forge는 **직접 작성·편집**이 핵심. Tag.Forge(ID3 태그)와 전혀 다른 작곡 도구.

- 기기 유형 자동 분류 아이콘 (라우터·스위치·PC·모바일·IoT)
- 노드 클릭 → 기기 상세 (MAC·IP·포트·제조사·응답시간)

**기술 스택**:

- LLDP 패킷 파싱

**수익 모델**: 무료
**차별화**: Net.Scan은 기기 목록+포트 특화; Topo.Cast는 관계 그래프 시각화+계층 레이아웃 특화.

### 10. `Ping.Map` — 전세계 CDN/클라우드 엔드포인트 지연시간 히트맵

**카테고리**: Network / Tools
**핵심 기능**: Net.Trace(traceroute 경로 추적)와 달리 **전세계 주요 CDN·클라우드 엔드포인트를 한 번에 핑**해 세계지도 위 지연시간 히트맵으로 시각화. 서버 리전 선택이나 CDN 성능 검증에 활용.

**상세 기능**:
- AWS·Azure·GCP·Cloudflare·Fastly 등 100+ 엔드포인트 내장
- 동시 멀티스레드 핑 측정 → 세계지도 색상 히트맵 (SkiaSharp)
- 지연시간별 색상 그라디언트 (초록<20ms → 빨강>200ms)
- 시간별 측정 히스토리 그래프 (최근 24시간)
- 최적 리전 자동 추천 (상위 3개 엔드포인트)
- 커스텀 엔드포인트 추가 (자체 서버 포함)
- 결과 CSV/JSON/PNG 내보내기

**기술 스택**:
- System.Net.NetworkInformation.Ping (비동기 멀티핑)
- SkiaSharp (세계지도 히트맵 렌더링)
- Equirectangular 투영 (Net.Trace 지도 코드 재활용)

**수익 모델**: 무료
**차별화**: Net.Trace는 단일 목적지 홉별 경로; Ping.Map은 전세계 다수 엔드포인트 동시 측정 히트맵.

**매력**: 한국어를 모국어로 쓰는 사람에게 딱 맞는 두뇌 게임. 자모 피드백으로 영어 Wordle보다 논리 추론이 정교해 중독성 강화
