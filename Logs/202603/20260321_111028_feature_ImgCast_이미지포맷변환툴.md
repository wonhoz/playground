## 작업 내용

지침서(`ideas/이미지 포맷 변환 툴 개발 지침서.md`)를 기반으로 `Img.Cast` 신규 개발.

### 프로젝트 위치
`Applications/Photo.Picture/Img.Cast/`

### 구현 기능

| 기능 | 내용 |
|------|------|
| 입력 포맷 | SVG, PNG, JPG/JPEG, BMP |
| 출력 포맷 | ICO (멀티사이즈), PNG, JPG, BMP |
| ICO 사이즈 | 16·24·32·48·64·128·256px (7종) |
| Drag & Drop | 다중 파일 + 폴더 드롭 지원 |
| 배치 처리 | 비동기(`async/await`) — UI 프리즈 없음 |
| 품질 유지 | SkiaSharp `SKFilterQuality.High` + Aspect Ratio 유지 + 투명 패딩 |
| JPG 품질 | Slider (10~100, 기본 95) |
| 덮어쓰기 | 체크박스 미체크 시 `_converted` 접미사 |
| 진행 표시 | ProgressBar + 현재 파일명 / 카운트 |
| 결과 다이얼로그 | 성공/실패 수 + 실패 파일 목록 및 에러 메시지 |
| 예외 처리 | 개별 실패 허용 (전체 중단 없음) |
| 다크 테마 | ComboBox / CheckBox / ProgressBar / Slider / ScrollBar ControlTemplate 완전 재정의 |
| DWM 타이틀바 | `DwmSetWindowAttribute` 다크 모드 적용 |

### 주요 파일

```
Img.Cast/
├── Img.Cast.csproj       (Svg.Skia, SkiaSharp.Views.WPF)
├── App.xaml / App.xaml.cs
├── MainWindow.xaml        (다크 테마 UI, ControlTemplate 전체 포함)
├── MainWindow.xaml.cs     (Drop 처리, 비동기 변환, 결과 다이얼로그)
└── Services/
    ├── IcoEncoder.cs      (ICONDIR/ICONDIRENTRY PNG-embedded ICO 생성)
    └── ImageConverter.cs  (SVG→Bitmap, 리사이즈, 포맷 인코딩, 파일 수집)
```
