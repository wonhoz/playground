## 개요

OAuth2/OIDC 토큰을 로컬에서 즉시 분석하는 WPF 개발 도구. 외부 서비스 없이 JWT 디코딩, 서명 검증, OIDC Discovery, OAuth2 플로 시각화 등을 한 곳에서 처리.

## 주요 기능

### 탭 1 — 🔐 JWT 디코더
- `JsonWebTokenHandler.ReadJsonWebToken()` 으로 JWT 파싱
- 헤더: Base64URL 직접 파싱 → JSON → DataGrid
- 페이로드: 클레임 목록 + RFC 7519 설명 34개 사전
- `exp`, `iat`, `nbf` Unix 타임스탬프 → 날짜 포맷 변환
- 만료 카운트다운 (`DispatcherTimer`, 1초 갱신)
- JWK URI 서명 검증 (`TokenValidationParameters` + `JsonWebKeySet`)
- `iss` 클레임에서 JWK URI 자동 추론

### 탭 2 — 🌐 OIDC Discovery
- `HttpClient.GetStringAsync()` → `JsonDocument.Parse()`
- `/.well-known/openid-configuration` 자동 경로 추가
- `jwks_uri` 값 JWK 입력란 자동 채우기
- Key-Value DataGrid 바인딩

### 탭 3 — 🔄 OAuth2 Flow 다이어그램
- WPF Canvas 직접 렌더링 (Rectangle + Line + Polygon + TextBlock)
- 6가지 플로: Authorization Code, PKCE, Client Credentials, Device Code, Implicit, ROPC
- 각 플로 단계별 시퀀스 화살표 + 레이블
- Implicit/ROPC는 보안 경고 메시지 포함

### 탭 4 — 📋 Auth Header 생성기
- Bearer Token: `Authorization: Bearer {token}`
- Basic Auth: `Authorization: Basic {Base64(user:pass)}`
- API Key: `{headerName}: {key}` (헤더 이름 커스텀)
- 생성 결과 즉시 클립보드 복사

### 탭 5 — 🔤 Base64URL 변환
- Base64URL ↔ UTF-8 텍스트 양방향 변환
- 표준 Base64 폴백 자동 처리

### 공통
- 토큰 히스토리 최근 10개 (`%APPDATA%/OAuthPeek/history.json`)
- 자동 타입 감지 (JWT / URL / 텍스트)
- `DwmSetWindowAttribute(hwnd, 20, ...)` 다크 타이틀바

## 기술 선택

- **Microsoft.IdentityModel.JsonWebTokens 8.4.0**: JWT 파싱/검증 표준 라이브러리
- **Microsoft.IdentityModel.Protocols.OpenIdConnect 8.4.0**: OIDC Discovery 지원
- **WPF Canvas 직접 렌더링**: 외부 차트 라이브러리 없이 플로 다이어그램 구현
