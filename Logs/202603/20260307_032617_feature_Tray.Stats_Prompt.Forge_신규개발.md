## Tray.Stats

**위치**: `Applications/Tools/System/Tray.Stats/`
**플랫폼**: .NET 8.0 WinForms (NotifyIcon + 커스텀 팝업)

### 구현 내용
- `TrayApp.cs` — ApplicationContext, NotifyIcon, 1초 타이머, DarkMenuRenderer
- `Services/StatsCollector.cs` — PerformanceCounter (CPU/Disk), Win32 GlobalMemoryStatusEx (RAM), NetworkInterface 바이트 델타 (Net), GPU Engine PerformanceCounter
- `UI/PopupForm.cs` — 300×410px 보더리스 팝업, DoubleBuffered 커스텀 드로잉, 60초 CircularBuffer 그래프
- `Program.cs` — 단일 인스턴스 Mutex, SetPreferredAppMode(2) ForceDark
- 임계치 초과 시 BalloonTip 경고 (CPU 90%, RAM 90%, Disk 95%)
- 트레이 아이콘 동적 드로잉 — CPU% 기반 컬러 호 (녹/황/적)

### 색상
- 배경 `#13131F`, CPU 녹→황→적, RAM `#60A5FA`, Disk `#FB923C`, Net `#34D399`, GPU `#C084FC`

## Prompt.Forge

**위치**: `Applications/Tools/Productivity/Utility/Prompt.Forge/`
**플랫폼**: .NET 8.0 WPF MVVM

### 구현 내용
- `Models/PromptItem.cs` — 프롬프트 데이터 모델 (제목/내용/태그/서비스/즐겨찾기/버전/메모)
- `Services/Database.cs` — SQLite + FTS5 가상 테이블, CRUD, 트리거 자동 인덱싱, 버전 히스토리
- `ViewModels/MainViewModel.cs` — INPC, 검색/필터/즐겨찾기, 버전 저장
- `MainWindow.xaml` — 2열 레이아웃 (목록 패널 300px | 에디터 *), 다크 테마 완전 커스텀
- `Views/FillVarsDialog.xaml` — `{{변수}}` 파서 + 입력 다이얼로그, 채운 결과 자동 클립보드 복사
- 전역 단축키 Win+Shift+P (RegisterHotKey, HwndSource WM_HOTKEY 훅)
- 퍼플 액센트 `#7C3AED`

### 기술 선택
- `Microsoft.Data.Sqlite 8.0.0` — NuGet 패키지
- FTS5 `content=prompts` — 주 테이블과 동기화, INSERT/UPDATE/DELETE 트리거 자동 관리
- Regex `\{\{(\w+)\}\}` — 변수 추출
