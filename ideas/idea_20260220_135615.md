# Playground 신규 아이디어 브레인스토밍

> 작성일: 2026-02-20
> 기준: 기존 8개 앱 분석 + 사용자 라이프스타일 고려
> 원칙: **작지만 매일 쓰는 것**, **없으면 불편한 것**, **돈 주고도 못 사는 것**

---

## 핵심 철학

StayAwake가 사랑받는 이유를 분석하면:
- ✅ 설치 후 잊어도 되는 "셋 앤 포겟"
- ✅ 매일 자동으로 나를 도와줌
- ✅ 기존 솔루션(마우스 지글러)보다 훨씬 스마트
- ✅ 없으면 진짜 불편함

이 기준으로 아이디어를 선별했습니다.

### 1. Clipboard.Stacker (Tools)

**한 줄 요약**: 여러 개를 복사하고, 순서대로 붙여넣기

| 항목 | 내용 |
|------|------|
| **문제** | Ctrl+C → Ctrl+V 반복 지옥. 엑셀에서 이름, 전화번호, 주소를 각각 복사해서 다른 곳에 붙여넣을 때 3번 왔다갔다 |
| **솔루션** | Ctrl+C를 여러 번 하면 스택에 쌓임. Ctrl+V 할 때마다 FIFO로 하나씩 꺼냄. Win+V 히스토리도 포함 |
| **킬러 기능** | ① 스택 모드: 5개 복사 → 5개 순서대로 붙여넣기 ② 즐겨찾기 핀: 자주 쓰는 텍스트 (계좌번호, 주소, 이메일 서명) 단축키 한 방 ③ 텍스트 변환: 붙여넣기 전에 UPPER/lower/trim/정규식 치환 |
| **형태** | 트레이 앱 + 전역 단축키 팝업 (Quick.Launcher처럼) |
| **대상** | 모든 사무직 종사자 |
| **수익 가능성** | ★★★★☆ — Ditto, CopyQ 같은 앱들이 있지만 "스택 붙여넣기"에 집중한 건 드묾 |
| **난이도** | ★★☆☆☆ |
| **그룹** | Tools |
