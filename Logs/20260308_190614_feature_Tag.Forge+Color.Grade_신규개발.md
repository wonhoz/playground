## 작업 내용

### Tag.Forge — MP3/FLAC 음악 태그 일괄 편집기

- **경로**: `Applications/Tools/Productivity/Media/Tag.Forge`
- **기능**
  - MP3(ID3v2.4) / FLAC(Vorbis) / OGG / M4A / AAC / WMA / Opus 지원
  - 폴더 드래그&드롭 → 트랙 목록 DataGrid 일괄 편집
  - 인라인 셀 편집 + 수정 행 좌측 황금색 스트라이프 시각화
  - 파일명↔태그 패턴 양방향 동기화 (`{track}/{artist}/{title}/{album}/{year}/{genre}`)
  - MusicBrainz API 조회 (rate-limit 1.1req/s 준수)
  - 앨범아트 미리보기 + 변경 + 삭제 + 다중 트랙 일괄 적용
  - 트랙 번호 자동 순번 (선택 or 전체)
- **기술 스택**: WPF, net8.0-windows, TagLibSharp 2.3.0

### Color.Grade — 이미지 LUT 색보정 도구

- **경로**: `Applications/Tools/Productivity/Creative/Color.Grade`
- **기능**
  - JPEG/PNG/BMP/TIFF 드래그&드롭 로드
  - 내장 알고리즘 프리셋 9종:
    - 원본, 웜, 쿨, 빈티지, B&W, B&W 하이컨트, 틸&오렌지, 팝, 시네마틱
  - 외부 `.cube` LUT 파일 로드 + 삼선형 보간(trilinear interpolation) 적용
  - 6종 보정 슬라이더: 노출/대비/채도/색온도/하이라이트/쉐도우
  - Before/After 분할 뷰 실시간 미리보기 (60ms 디바운스, 800px 리사이즈)
  - 단일 파일 내보내기 (JPEG/PNG/BMP/TIFF) + 배치 폴더 처리
  - WriteableBitmap 기반 순수 CPU 픽셀 처리 (NuGet 불필요)
- **기술 스택**: WPF, net8.0-windows, 외부 NuGet 없음
