### 1. `Date.Calc` — 날짜·시간 계산기

**카테고리**: Tools.Utility / Desktop
**핵심 기능**: 두 날짜 사이 간격 계산, N일 후/전 날짜 찾기, 요일 확인, 나이·디데이 계산을 한 화면에서 처리하는 오프라인 날짜 계산기. 개발·일정 관리에서 매일 필요한 계산을 즉시 해결한다.

**상세 기능**:
- 두 날짜 사이 일수·주·월·년 차이 즉시 계산
- 날짜 + N일 더하기/빼기 (영업일 제외 옵션)
- 특정 날짜의 요일 표시 및 해당 월 달력 미니 뷰
- 나이 계산 (생년월일 입력 → 만 나이 + 개월 + 일수)
- D-Day 카운트다운 (마감일·기념일 저장 목록)
- KST/UTC/타임존 변환 탭 포함
- 결과 클립보드 복사 원클릭

**기술 스택**:
- .NET 10 WPF / NodaTime (날짜 연산 정확도)
- DateTimeOffset + TimeZoneInfo (타임존)
- 달력 미니뷰: Custom WPF DrawingVisual

**수익 모델**: 무료 오픈소스
**차별화**: Unit.Forge(단위변환), Text.Forge(인코딩)는 숫자·텍스트 변환에 집중. Date.Calc는 날짜 특화 — 영업일 계산·D-Day 목록·달력 시각화가 핵심

### 2. `Case.Forge` — 개발자 텍스트 케이스 변환기

**카테고리**: Dev/Tools / Tray
**핵심 기능**: 클립보드 텍스트를 camelCase, snake_case, PascalCase, kebab-case, SCREAMING_SNAKE_CASE, Title Case 등 10가지 케이스로 즉시 변환하는 전역 팝업 도구. 코드 리팩토링·API 명세 작성 시 반복 타이핑을 제거한다.

**상세 기능**:
- 10가지 케이스 동시 미리보기 (한 화면에 모두 표시)
- Win+Shift+C 전역 단축키 팝업 호출
- 입력 텍스트 자동 감지 (현재 케이스 강조)
- 목록에서 클릭 시 해당 케이스 즉시 클립보드 복사
- 다국어 인식: 공백·언더스코어·하이픈·점(.)·슬래시 구분자 자동 파싱
- 즐겨찾기 케이스 상단 고정
- 트레이 아이콘 우클릭 → 직접 변환 서브메뉴

**기술 스택**:
- .NET 10 WinForms Tray / Humanizer.Core (케이스 변환)
- 전역 단축키: RegisterHotKey Win32 API

**수익 모델**: 무료 오픈소스
**차별화**: Text.Forge(해시·인코딩·Base64), Str.Forge(멀티파일 Find&Replace) 와 완전히 다른 도메인 — identifier 케이스 변환은 개발자가 하루에도 수십 번 필요로 하는 독립적 작업

### 3. `Dice.Cast` — 주사위 & 무작위 추첨기

**카테고리**: Tools.Utility / Fun
**핵심 기능**: D4/D6/D8/D10/D12/D20/D100 주사위를 굴리고, 팀 무작위 편성·라플 추첨·범위 랜덤 숫자를 한 앱에서 처리. 보드게임·회의 게임·스탠드업 무작위 발표 순서 결정에 즉시 활용.

**상세 기능**:
- 주사위 공식 입력 (예: 2d6+3, 4d8) + 결과 히스토리

- 팀 편성 (N명 → K팀 무작위 배분)
- 범위 지정 랜덤 숫자 (최솟값, 최댓값, 제외 번호)

**기술 스택**:
- .NET 10 WPF / System.Security.Cryptography.RandomNumberGenerator (암호학적 난수)
- 굴림 애니메이션: WPF Storyboard

**수익 모델**: 무료 오픈소스
**차별화**: QR.Forge(QR 생성), Badge.Forge(배지 생성) 등 기존 유틸과 카테고리 완전 상이. 주사위+추첨+팀편성 3기능 통합이 핵심

### 4. `Img.Crop` — 이미지 퀵 크롭 + 비율 프리셋

**수익 모델**: 무료 오픈소스
**차별화**: Img.Cast(포맷 변환 전문), Img.Forge(배치 리사이즈·필터·워터마크), Web.Shot(웹페이지 캡처) — 크롭 + 비율 프리셋 + 황금비 가이드 결합은 현재 없음

### 5. `Pwd.Gen` — 보안 비밀번호·API 키 생성기 트레이

**카테고리**: Security / Tray
**핵심 기능**: 전역 단축키 팝업으로 즉시 강력한 비밀번호·API 키·UUID·시크릿 토큰을 생성·복사하는 트레이 앱. Pass.Vault(저장 금고)와 달리 생성 전문 — 가볍고 빠른 단일 목적 도구.

**상세 기능**:
- 비밀번호 생성 (길이·대소문자·숫자·특수문자 조합 슬라이더)
- API 키 형식: hex-32, base64url-32, uuid-v4, nanoid
- 비밀번호 강도 실시간 표시 (엔트로피 비트 계산)
- 기억하기 쉬운 패스프레이즈 생성 (diceware 단어 목록)
- 생성 즉시 클립보드 자동 복사 + 토스트 확인
- 최근 생성 이력 5개 (암호화 저장, 창 닫으면 삭제)
- Win+Shift+P 전역 단축키

**기술 스택**:
- .NET 10 WinForms Tray / System.Security.Cryptography.RandomNumberGenerator
- 강도 계산: zxcvbn-cs

**수익 모델**: 무료 오픈소스
**차별화**: Pass.Vault(AES-256 비밀번호 저장소 ≠ 생성기), Key.Stash(API 키 보관 ≠ 생성). Pwd.Gen은 저장 없이 즉시 생성·복사만 하는 초경량 트레이

### 10. `Wake.Timer` — 로컬 PC 절전 해제 알람 타이머

**카테고리**: System / Tray
**핵심 기능**: 지정한 시각에 PC를 자동으로 절전/최대 절전 해제하고 알람을 울리는 트레이 앱. Wake.Cast(WOL 원격 기동)와 완전히 다른 로컬 Wake-on-Timer — 아침 기상·렌더링 완료 알림·예약 작업 실행에 활용.

**상세 기능**:
- 특정 시각 또는 N분 후 Wake Timer 예약 (Windows Task Scheduler + BIOS Wake Timer)
- Wake 후 알람음 재생 (내장 사운드 또는 MP3 선택)
- Wake 후 자동 실행 프로그램/스크립트 설정
- 다음 Wake 예약 시각 트레이 아이콘 툴팁 표시
- 반복 일정 (평일 08:00 매일 등)
- 현재 절전 상태 표시, 즉시 절전 진입 버튼
- Wake 이력 로그 (언제 깨어났는지)

**기술 스택**:
- .NET 10 WinForms Tray / Windows Task Scheduler API (Microsoft.Win32.TaskScheduler)
- ACPI Wake Timer: SetWaitableTimer / Task Scheduler wake condition
- 알람 재생: NAudio

**수익 모델**: 무료 오픈소스
**차별화**: Wake.Cast ✅(WOL 원격 PC 기동 ≠ 로컬 타이머), Stay.Awake ✅(절전 방지 ≠ 절전 해제 예약), Sched.Cast ✅(Task Scheduler GUI ≠ Wake 특화 알람). 로컬 Wake Timer + 알람 + 자동 실행 통합은 없음

### 1. `Num.Rush` — 슐테 테이블 집중력 순서 클릭 게임

**장르**: Casual / Brain Training
**핵심 메커닉**: 5×5 그리드에 1~25 숫자가 무작위 배치, 가능한 빠르게 1→2→3→...→25 순서로 클릭. 시선 이동 범위를 최소화하는 집중력과 주변시 훈련 게임 — 단순하지만 중독적이고 매 판 기록이 달라진다.

**게임플레이**:
- 5×5, 7×7, 9×9 그리드 크기 선택 (난이도 조절)
- 순서 클릭 완료 시간 측정 (ms 단위 정밀도)
- 틀린 셀 클릭 시 오답 효과 + 시간 패널티
- 세션 최고 기록·평균·개선 그래프 표시
- 컬러 테마 모드 (숫자 대신 색상+숫자 혼합 고난도)
- 역순 (25→1) 모드
- 하루 Top 5 자동 저장

**기술**: WPF Canvas 그리드, Fisher-Yates 셔플, Stopwatch 정밀 타이머
**매력**: 10초 안에 끝나는 한 판, 매 판 최고 기록 갱신 욕구, 집중력 향상 효과 실감
**차별화**: Geo.Quiz(지리 교육 퀴즈), Crossword.Cast(낱말 퍼즐) 등 기존 교육 게임과 달리 시각 집중력·반응속도 훈련에 특화. Reflex.Tap(반응속도 솔로, similar groups)과 달리 공간 탐색+순서 기억이 핵심

### 2. `Connect.Cast` — 커넥트4 클래식 대전

**장르**: Strategy / Classic / VS
**핵심 메커닉**: 7×6 격자에 번갈아 칩을 투하해 가로·세로·대각선 4개를 먼저 연결하는 클래식 전략 게임. 단 몇 초면 규칙을 이해하지만, AI를 이기려면 2~3수 앞을 내다봐야 하는 깊이가 있다.

**게임플레이**:
- 로컬 2P (같은 키보드) 또는 AI 상대 모드
- AI 난이도 5단계 (Minimax + Alpha-Beta Pruning 깊이 조절)
- 클릭/키보드 (1~7) 열 선택, 칩 낙하 애니메이션
- 승리 시 연결된 4개 칩 하이라이트
- 게임 기록: 현재 세션 승/패/무 집계
- 무승부 감지 (격자 꽉 참)
- 색상 테마: 고전 노랑/빨강 / 네온 / 다크 모노크롬

**기술**: WPF Canvas, Minimax Alpha-Beta Pruning (AI), 낙하 물리 애니메이션
**매력**: 아이부터 어른까지 아는 룰, AI 최고 난이도 도전, 2P 대전의 실시간 심리전
**차별화**: Spot.Diff (시각 퍼즐 ≠ 전략 대전), Dodge.Blitz ✅ (슈터 ≠ 전략). 열 투하 방식 4-연결 전략 게임은 목록·백로그 어디에도 없음

### 3. `Mine.Sweep` — 지뢰찾기 클래식 + 모던 UX

**장르**: Puzzle / Casual
**핵심 메커닉**: 숫자 단서로 지뢰 위치를 추론해 안전한 칸을 모두 열어야 하는 고전 퍼즐. Windows 내장 게임이 제거된 이후 독립 앱으로 재구현 — 깔끔한 다크 UI와 현대적 편의 기능 추가.

**게임플레이**:
- 난이도: 쉬움(9×9·10개), 보통(16×16·40개), 어려움(30×16·99개), 커스텀
- 우클릭 = 깃발, 중간 클릭 = 주변 자동 열기 (코드)
- 첫 클릭 보장 (지뢰 없음), BFS 자동 확장
- 시간 측정 + 최고 기록 테이블
- 세이프 줌: 불확실 상황 3번 힌트 (지뢰 1개 위치 공개)
- 승리 시 배경 폭죽 애니메이션
- 다크 테마 + 레트로 픽셀 폰트 테마 선택

**기술**: WPF Canvas, Flood-fill BFS, System.Random 지뢰 배치
**매력**: 누구나 아는 게임, 10분 안에 끝나는 집중 퍼즐, "마지막 칸 클릭" 긴장감
**차별화**: 기존 게임 목록에 지뢰찾기 없음. Puzzle/Casual 장르이나 물리·색상·교육이 아닌 확률 추론 기반으로 완전히 다른 경험

### 4. `Checkers.Cast` — 체커스(드래프츠) 클래식 대전

**장르**: Strategy / Board / Classic
**핵심 메커닉**: 8×8 보드에서 대각선으로 이동해 점프로 상대 말을 제거하는 클래식 체커스(드래프츠). 단순한 규칙 속에 선형 계획과 수순 계산의 깊이가 있다. Connect.Cast(넓이 우선)와 달리 긴 중반전 포지셔닝 게임.

**게임플레이**:
- 로컬 2P 또는 AI 상대 (5단계 난이도)
- 규칙 선택: 영국식(8×8), 국제식(10×10)
- 강제 점프 규칙 + 복수 점프 연쇄 처리
- 킹 승격 시 왕관 애니메이션, 양방향 이동 허용
- 이동 취소 (Ctrl+Z), 힌트 모드 (최선 수 강조)
- AI: Alpha-Beta Pruning + 위치 평가 함수
- 승부 기록 + 베스트 게임 리플레이 저장

**기술**: WPF Canvas, Alpha-Beta Pruning AI, 강제 점프 탐색 알고리즘
**매력**: 짧지만 집중도 높은 대결, AI 최고 난이도에 도전하는 성취감, 복수 점프 연쇄의 짜릿함
**차별화**: Connect.Cast(커넥트4 열 투하 ≠ 체커스 이동), Chess.Forge(변형 체스 ≠ 드래프츠), Flip.Card(오델로 변형 ≠ 체커스). 체커스는 목록·백로그 어디에도 없음

### 5. `Spot.Diff` — 틀린 그림 찾기 퍼즐 게임

**장르**: Puzzle / Casual / Visual
**핵심 메커닉**: 두 이미지 중 다른 점 N개를 찾아 클릭. 픽셀 아트 스타일 이미지를 절차적으로 생성해 매 판 새로운 퍼즐 제공. Img.Compare(개발자 SSIM 비교 도구)와 완전히 다른 게임 경험.

**게임플레이**:
- 원본 이미지에서 N개(기본 5개) 차이를 자동 생성 (색상 교체, 오브젝트 추가/제거)
- 시간 제한 (난이도별 30초~3분)
- 정확 클릭 시 원 마커 표시 + 점수, 오클릭 시 시간 감소 패널티
- 힌트: 다른 부분 영역 강조 (3회 제한)
- 이미지 카테고리: 도시 풍경, 자연, 사무실, 게임 픽셀 아트 등
- 완성 후 두 이미지 슬라이더 비교 뷰
- 사용자 이미지 불러오기 (수동 차이점 표시 에디터)

**기술**: WPF Canvas, SkiaSharp 이미지 조작, 절차적 차이 생성 알고리즘 (색 팔레트 교체·오브젝트 레이어 on/off)
**매력**: 집중해서 찾았을 때의 "아하!" 순간, 시간 압박으로 인한 긴장감, 사용자 이미지 지원
**차별화**: Img.Compare ✅(SSIM/PSNR 화질 분석 유틸리티 ≠ 게임), Deep.Diff(텍스트 diff). 게임 장르로서의 틀린 그림 찾기는 기존 목록에 없음

### 6. `Trivia.Cast` — 다양한 주제 상식 퀴즈 게임

**장르**: Casual / Educational / Quiz
**핵심 메커닉**: 과학·역사·스포츠·대중문화 등 다양한 카테고리에서 4지선다 및 O/X 퀴즈를 풀어가는 상식 퀴즈 게임. Geo.Quiz(지리 특화)와 달리 범용 지식 플랫폼 — 오프라인 완전 동작.

**게임플레이**:
- 카테고리 선택: 과학·역사·스포츠·영화·게임·IT·음식·지리·수학 등 12가지
- 모드: 빠른 퀴즈(10문제), 마라톤(50문제), 카테고리 도전(해당 카테고리 전문)
- 시간 제한 모드: 15초 안에 답하기 (시간 잔량에 따라 점수 차등)
- 스트릭 보너스: 연속 정답 시 점수 배율
- 오답 리뷰: 틀린 문제 해설 + 정답 다시 학습
- 내장 문제 2,000개 + CSV로 커스텀 문제 추가
- 일일 챌린지: 5개 문제로 스트릭 카운터

**기술**: WPF, SQLite (문제 DB + 오답 이력), Fisher-Yates 문제 셔플, CSV 가져오기
**매력**: 하루 5문제 루틴, 내가 모르는 지식 발견, 커스텀 문제로 스터디 활용
**차별화**: Geo.Quiz ✅(지리·국기 전문 ≠ 범용 상식), Crossword.Cast ✅(십자말 영단어 ≠ 4지선다 퀴즈), Cipher.Quest ✅(암호 해독 ≠ 상식). 범용 오프라인 상식 퀴즈는 목록에 없음

### 7. `Grid.Slide` — 슬라이딩 타일 퍼즐 (15-퍼즐)

**장르**: Puzzle / Classic
**핵심 메커닉**: 1~15 숫자 타일을 빈 칸으로 밀어 올바른 순서로 맞추는 고전 슬라이딩 퍼즐. 피드백이 즉각적이고 이동 수 최소화 목표로 단순하지만 깊은 퍼즐성을 제공한다.

**게임플레이**:
- 3×3(8-퍼즐), 4×4(15-퍼즐), 5×5(24-퍼즐) 크기 선택
- 이동 횟수 + 경과 시간 동시 측정
- 최적 이동 수 표시 (A* IDA* 알고리즘으로 계산)
- 이미지 모드: 숫자 대신 사진을 분할해 맞추기
- 자동 풀기 시연 (A* 최단 경로 애니메이션)
- 셔플 레벨 (최소 N수 이동 거리 보장)
- 완성 시 타일 합체 애니메이션 + 기록 저장

**기술**: WPF Canvas, A* + IDA* (슬라이딩 퍼즐 최적해), SkiaSharp 이미지 분할
**매력**: 손에서 놓기 어려운 클래식 중독성, 이미지 모드의 시각적 즐거움, 최적 경로 도전
**차별화**: Gravity.Flip ✅(플랫포머), Auto.Build ✅(테트로미노), Orbit.Craft ✅(궤도 설계) 등 기존 퍼즐과 완전히 다른 슬라이딩 메커닉. 백로그에도 15-퍼즐 없음

### 8. `Calc.Rush` — 수식 스피드 게임

**장르**: Casual / Brain Training / Speed
**핵심 메커닉**: 사칙연산 수식이 화면에 나타나면 답을 최대한 빨리 입력하는 속도 게임. 제한 시간 내 최대한 많은 문제를 풀어 점수와 콤보를 쌓는다. 단순하지만 뇌 훈련 효과가 뚜렷하다.

**게임플레이**:
- 문제 유형: 덧셈·뺄셈·곱셈·나눗셈 (난이도별 숫자 범위)
- 60초 타임어택: 정답 수 × 콤보 보너스 = 최종 점수
- 연속 정답 콤보 (2배수) + 오답 시 콤보 초기화
- 난이도: 쉬움(1자리), 보통(2자리), 어려움(2자리 혼합연산), 악마(3자리+괄호)
- 숫자 키패드 입력 최적화
- 역대 최고 점수 테이블 (난이도별)
- 오늘의 랭킹 그래프 (자신의 점수 시간대 추적)

**기술**: WPF, System.Random 수식 생성, Stopwatch 정밀 타이머
**매력**: 10초에 여러 문제, 콤보 시스템으로 집중력 유지, 뇌 트레이닝 효과, 기록 갱신 중독
**차별화**: Geo.Quiz ✅(지리 O/X ≠ 수식 계산), Crossword.Cast ✅(영단어 ≠ 수학). 순수 수학 스피드 게임은 기존 목록 어디에도 없음

### 9. `Hop.Rush` — 프로거 스타일 길 건너기 러시

**장르**: Arcade / Classic / Reflex
**핵심 메커닉**: 자동차가 쌩쌩 달리는 도로와 통나무가 떠가는 강을 타이밍 맞춰 건너 목적지에 도달하는 프로거 스타일 게임. 단순한 4방향 입력, 타이밍이 전부인 원버튼 클래식.

**게임플레이**:
- 방향키로 한 칸씩 이동 (자동스크롤 없음, 탑다운)
- 도로 구간: 좌우로 달리는 차량들 (속도·차선 수 증가)
- 강 구간: 떠내려가는 통나무·거북이 올라타기 (강에 빠지면 실패)
- 5개 목적지 모두 도달하면 다음 스테이지 (난이도 상승)
- 시간 제한 + 목숨 3개
- 스테이지 클리어 최단 시간 기록
- 날씨 이벤트: 안개(시야 제한), 비(통나무 속도 변화)

**기술**: WPF Canvas, 오브젝트 풀링 (차량/통나무), 충돌 AABB 감지
**매력**: 한 번의 타이밍 미스로 인한 통쾌한 실패, 도로→강 교차 구조의 다양성, "한 판 더" 중독
**차별화**: Neon.Run ✅(무한 사이드 러너 ≠ 탑다운 건너기), Dodge.Blitz ✅(총알 회피 ≠ 차량 타이밍). 프로거 방식 탑다운 건너기 게임은 기존 목록에 없음

### 10. `Pac.Dash` — 팩맨 스타일 미로 닷 수집

**장르**: Arcade / Classic / Chase
**핵심 메커닉**: 미로 안의 모든 점을 먹으면서 유령에게 잡히지 않는 클래식 팩맨 스타일. 파워펠릿으로 유령을 역전 사냥하는 상황 역전이 핵심 쾌감 — 단순하지만 전략적.

**게임플레이**:
- 방향키 조작, 픽셀아트 스타일 미로 + 기본 28점 그리드
- 4종 유령 AI: 추적형·예측형·무작위형·협공형 각각 다른 행동
- 파워펠릿 먹으면 유령 먹을 수 있는 무적 시간 (점수 배율)
- 스테이지 클리어 시 속도·유령 수 증가
- 스코어 하이스코어 테이블
- 레트로 8비트 효과음 + 닷 수집 효과
- 숨겨진 보너스 과일 아이템 (점수 2배)

**기술**: WPF Canvas, BFS 유령 AI (Blinky/Pinky/Inky/Clyde 행동 패턴 재현), 타일 맵 충돌
**매력**: 유령 역전 사냥의 짜릿함, 코너링 전략, 마지막 점 하나 남았을 때의 긴장감
**차별화**: Hop.Rush(타이밍 건너기 ≠ 미로 닷 수집), Maze.Dread(시야 제한 미로 공포 ≠ 유령 추격 아케이드). 팩맨 스타일 미로+유령+닷 수집은 목록·백로그 어디에도 없음
