### 1. `Serial.Cast` — COM/UART 시리얼 터미널 모니터

**카테고리**: Dev / Tools / Hardware
**핵심 기능**: 아두이노·ESP32 등 마이크로컨트롤러와의 시리얼 통신을 GUI로 모니터링하는 개발자용 터미널. COM 포트 자동 감지, 커스텀 baud rate, HEX/ASCII 동시 표시, 명령 전송 이력 관리.

**상세 기능**:
- COM 포트 자동 열거·연결, baud/parity/stop bit 전체 설정
- 수신 데이터 HEX·ASCII·UTF-8 동시 뷰, 컬러 구분 출력
- 사용자 정의 명령 단축키 팔레트 (자주 쓰는 AT 명령 등 원클릭 전송)
- 타임스탬프 토글, 자동 스크롤/고정 토글
- 수신 로그 TXT/CSV 저장, 특정 패턴 하이라이트 (정규식)
- 그래프 모드: 콤마 구분 숫자 수신 시 실시간 라인 차트 자동 렌더

**기술 스택**:
- .NET 8 / WPF
- `System.IO.Ports.SerialPort` (내장)
- `SkiaSharp` 또는 `LiveChartsCore` — 실시간 차트
- 정규식 하이라이트: `RichTextBox` + `Regex`

**수익 모델**: 무료 오픈소스 (하드웨어 개발자 커뮤니티 유틸)
**차별화**: `Api.Probe`(HTTP REST)·`Proto.Forge`(gRPC)와 달리 물리 레이어 시리얼 통신 특화. `Term.Pad`(터미널 패드)는 셸 기반이지만 Serial.Cast는 UART/TTY 프로토콜 전용.

### 5. `Time.Lapse` — 화면 타임랩스 레코더

**카테고리**: Productivity / Capture
**핵심 기능**: 정해진 간격으로 화면을 캡처하여 작업 과정 타임랩스 영상을 자동으로 생성. `Screen.Recorder`(실시간 녹화)·`GIF.Cast`(단편 GIF)와 달리 장시간 작업 요약 특화.

**상세 기능**:
- 캡처 간격 설정: 1초 ~ 60분 (슬라이더)
- 변화 감지 스마트 캡처: 픽셀 차이 임계치 이하면 중복 프레임 건너뜀
- 특정 모니터/창 선택 녹화
- 결과물: MP4(H.264)·WebM·GIF 선택 출력, 재생 속도 배수 설정
- 세션 이어쓰기: 다음 날 같은 프로젝트 녹화에 이어 붙이기
- 트레이 상주, 시작/중지 전역 단축키

**기술 스택**:
- .NET 8 / WinForms (트레이)
- `ScreenCaptureKit` 대신 DXGI Desktop Duplication API — 저CPU 캡처
- FFmpeg.AutoGen — MP4/WebM 인코딩
- `AnimatedGif` NuGet — GIF 변환

**수익 모델**: 무료
**차별화**: `Screen.Recorder`는 실시간 전체 녹화, `GIF.Cast`는 짧은 GIF 위주. `Time.Lapse`는 수 시간 장기간 → 1분 요약 영상 생성이 핵심.

### 6. `Bit.Cast` — 비트필드·레지스터 시각 에디터

**카테고리**: Dev / Tools / Embedded
**핵심 기능**: 임베디드·하드웨어 개발자를 위한 레지스터 비트 구조 정의 및 시각화 도구. 칩 데이터시트의 레지스터 맵을 GUI로 정의하고, 값 입력 시 비트 분리 표시와 C 매크로 자동 생성.

**상세 기능**:
- 레지스터 정의: 이름·비트 폭·필드명·R/W 속성 GUI 편집
- 값 입력 (Hex/Dec/Bin) → 비트 필드별 실시간 분해 시각화
- C/C++ 헤더 매크로 자동 생성 (`#define REG_FIELD_MASK`, `GET_FIELD`, `SET_FIELD`)
- 칩 프로파일 저장·로드 (JSON)
- 여러 레지스터 동시 비교 (2~4 레지스터 side-by-side)
- `Text.Forge`·`Quick.Calc`(Hex/Bin 계산기)와 연계 사용 가이드

**기술 스택**:
- .NET 8 / WPF
- `System.Text.Json` — 프로파일 저장
- 비트 분해: 순수 C# 비트 연산
- 코드 생성: `Scriban` 또는 직접 `StringBuilder` 템플릿

**수익 모델**: 무료
**차별화**: `Text.Forge`(Base64/JWT/인코딩)·`Quick.Calc`(프로그래머 계산기)와 달리 레지스터 구조 정의 + 필드 분해 + C 코드 생성이 핵심. 임베디드 엔지니어 전용 니치.

### 7. `Dither.Art` — 이미지 디더링·레트로 픽셀 필터 효과기

**카테고리**: Productivity / Creative / Media
**핵심 기능**: 사진을 Floyd-Steinberg·Bayer 매트릭스·Ordered 디더링 알고리즘으로 레트로 픽셀 아트 스타일로 변환하는 오프라인 이미지 효과 도구.

**상세 기능**:
- 알고리즘: Floyd-Steinberg·Jarvis-Judice-Ninke·Ordered(4×4~16×16)·Random·Bayer 8종
- 팔레트 제한: 2색·4색·8색·16색·커스텀 팔레트 (스포이드로 수집)
- 전/후 비교 슬라이더 (실시간 미리보기)
- 반전·밝기·대비 전처리 슬라이더
- 일괄 처리: 폴더 드래그&드롭 → 같은 설정 적용
- GIF·PNG·ICO·BMP 내보내기 (ICO는 픽셀아트 아이콘 제작 용도)

**기술 스택**:
- .NET 8 / WPF
- `ImageSharp` — 픽셀 조작
- 디더링 알고리즘: 순수 C# (외부 라이브러리 없음)
- `AnimatedGif` — GIF 내보내기

**수익 모델**: 무료
**차별화**: `Img.Forge`(리사이즈·필터 일괄 처리)·`Color.Grade`(LUT 색보정)는 사진 품질 향상 지향, `Dither.Art`는 의도적 품질 저하로 레트로 예술 표현 특화. `Pixel.Forge`(픽셀아트 에디터)는 그리기 도구, Dither.Art는 변환 도구.

### 8. `Merge.Cast` — 시각적 3-way 코드 병합 도구

**카테고리**: Dev / Tools
**핵심 기능**: Git 병합 충돌 해결을 위한 인터랙티브 3-way 비교·병합 GUI. `Deep.Diff`(2-way 비교 뷰어)와 달리 Base·Ours·Theirs 세 버전을 동시에 보며 충돌 헝크 단위로 직접 선택 병합.

**상세 기능**:
- 3분할 패널: Base(공통 조상)·Ours·Theirs + 결과 패널 4분할
- Git conflict 마커(`<<<<<<< HEAD ... >>>>>>> branch`) 자동 파싱·임포트
- 헝크 단위 Ours/Theirs/Base/수동 편집 선택 버튼
- 구문 강조: C#·JS·TS·Python·Go·Java 등 10+ 언어
- 결과 파일 저장 → Git 스테이징 안내
- 파일 드래그&드롭 또는 Git 저장소 충돌 파일 자동 감지

**기술 스택**:
- .NET 8 / WPF
- `DiffPlex` — 3-way diff 알고리즘
- `AvalonEdit` — 구문 강조 에디터 패널
- `LibGit2Sharp` — 충돌 파일 자동 감지

**수익 모델**: 무료
**차별화**: `Deep.Diff`는 2-way read-only 비교기. `Merge.Cast`는 3-way 인터랙티브 편집+저장이 핵심. `Git.Reel`(커밋 그래프)과 상호 보완.

### 10. `Ink.Trail` — 잉크 경로 설계 퍼즐

**장르**: Puzzle / Casual / Creative
**핵심 메커닉**: 잉크 발사기의 각도·속도·색상을 설정하여 목표 지점까지 잉크 경로를 설계하는 퍼즐. 흡수체·반사벽·색 혼합 구역이 경로를 복잡하게 만든다.

**게임플레이**:
- 발사기 각도(0~360°)·속도·잉크 양 슬라이더로 설정 후 "발사"
- 잉크가 중력+공기저항 포물선으로 날아가며 목표 타겟 적중
- 장애물 종류: 반사 거울(각도 반사)·흡수 스펀지(잉크 소모)·분열기(경로 2분기)·색 필터(통과 시 색 변경)
- 색 혼합 타겟: 빨강+파랑=보라, 빨강+노랑=주황 — 혼합 잉크로 특정 색 타겟만 클리어
- 레벨당 제한 발사 횟수, 최소 횟수 클리어 시 별 3개
- 샌드박스 모드: 자유 배치로 예술적 잉크 경로 만들기 + PNG 저장

**기술**: 잉크 포물선: 단순 운동학 (초기 속도·중력·공기저항), 색 혼합: CMYK 혼합 모델, `SkiaSharp` 잉크 번짐 Path 렌더, 레벨 에디터 JSON

**매력**: 예상치 못한 반사와 색 혼합이 만들어내는 "아하!" 순간. 샌드박스에서 우연히 완성된 예술적 경로.
**차별화**: `Ink.Spread`(BFS 전파 차단벽 전략)와 메커닉 완전 다름. `Echo.Shot`(음파 반사 슈터)·`Laser.Net`(거울 배치 레이저)과 달리 잉크 물리+색 혼합이 핵심.
