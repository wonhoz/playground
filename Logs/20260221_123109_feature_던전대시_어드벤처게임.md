## 작업 내용

탑다운 로그라이크 던전 크롤러 어드벤처 게임 개발.

### 구현 기능

| 기능 | 설명 |
|------|------|
| 던전 생성 | BSP 기반 랜덤 방+복도 생성, 보물상자/계단 자동 배치 |
| 플레이어 | 방향키 이동, Z 공격 (방향 히트박스), X 대시 (무적 회피), C 범위 스킬 |
| 몬스터 종류 | Slime (약), Skeleton (일반), Ghost (벽 통과), Demon (강), Dragon (보스) |
| 몬스터 AI | 시야 내 플레이어 추적, 근접 공격, 시야 밖 랜덤 배회 |
| 전투 | 히트박스 충돌, 데미지 텍스트 (떠오르며 페이드), 무적 시간 |
| 아이템 | 체력 회복 (♥), 공격력 강화 (⚔), 대시 강화 (★) |
| 던전 구조 | B1F~B5F 5층, 층별 난이도 상승, 3층부터 보스 등장 |
| 보물 상자 | 랜덤 배치, 열면 점수 + 아이템 드롭 |
| 미니맵 | 우하단 120x90 미니맵, 방/복도/계단/플레이어 위치 표시 |
| UI | HP바 (색상 변화), 보스 HP바, 층/점수/ATK/스킬 쿨다운 표시 |
| 화면 | 타이틀, 층 전환 연출, YOU DIED (게임오버), DUNGEON CLEAR (클리어) |
| 카메라 | 플레이어 중심 스크롤, 던전 전체 탐험 가능 |

### 아키텍처

```
Engine/
  GameLoop.cs            - 게임 루프
  KeyInput.cs            - 키보드 입력 (JustPressed 지원)
  DungeonGenerator.cs    - BSP 랜덤 던전 생성
Entities/
  Player.cs              - 플레이어 (이동/공격/대시/스킬)
  Monster.cs             - 몬스터 (5종류 AI)
  Pickup.cs              - 아이템 드롭
  DamageText.cs          - 데미지 숫자 이펙트
```

### 조작법

| 키 | 동작 |
|----|------|
| ↑↓←→ / WASD | 이동 |
| Z | 공격 (방향 근접) |
| X | 대시 (무적 회피) |
| C | 범위 스킬 (쿨다운 3초) |
| P | 일시정지/해제 |
| ENTER | 시작/재시작 |
| ESC | 타이틀로 |

### 던전 층별 구성

| B4F | Ghost, Demon | 고난이도 |
| B5F | Demon, Ghost + Dragon 보스 | 최종 보스 |
