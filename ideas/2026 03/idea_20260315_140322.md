### 1. `SQL.Forge` — SQLite 스키마 GUI 디자이너 + ERD 시각화

**카테고리**: Applications / Development / Inspector
**핵심 기능**: SQLite `.db` 파일을 열어 테이블·컬럼·외래키를 GUI로 편집하고, 실시간 ERD(Entity-Relationship Diagram) 다이어그램으로 시각화합니다. SQL DDL 스크립트를 직접 생성하거나 임포트할 수 있습니다.

**상세 기능**:
- `.db` 파일 로드, 테이블 목록·컬럼·인덱스·외래키 트리 표시
- 드래그앤드롭으로 테이블 간 FK 관계 연결 (ERD 다이어그램)
- 인라인 컬럼 편집: 타입, NOT NULL, DEFAULT, PK, UNIQUE
- `ALTER TABLE` / `CREATE TABLE` DDL SQL 자동 생성 및 실행
- 쿼리 탭: SQL 에디터 + 결과 DataGrid (SELECT 실행)
- ERD SVG/PNG 내보내기, DDL 스크립트 파일로 내보내기

**기술 스택**:
- .NET 10, WPF, Microsoft.Data.Sqlite 9.x
- WebView2 + 자체 SVG ERD 렌더러 (노드·연결선 포스 레이아웃)
- System.Text.Json (스키마 직렬화)

**수익 모델**: 무료 오픈소스
**차별화**: `SQL.Lens`는 쿼리 분석(실행계획)에 집중하고, `SQL.Forge`는 스키마 설계·ERD 시각화에 집중. DB 파일 뷰어가 아닌 **스키마 디자이너**

### 1. `Smoke.Trail` — 열원·바람·장애물 배치로 연기 흐름 유도 퍼즐

**장르**: 퍼즐 / 물리 시뮬레이션
**핵심 메커닉**: 격자 위에 열원, 팬(바람), 장애물을 배치하여 발생한 연기 입자가 목표 지점에 도달하도록 유도합니다. 유체 역학 기반 격자 기체 시뮬레이션으로 연기가 자연스럽게 흐릅니다.

**게임플레이**:
- 제한된 오브젝트(열원 2개, 팬 3개, 벽 5개)로 연기 경로 설계
