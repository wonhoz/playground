## 1. Str.Forge (멀티파일 Find & Replace 엔진)

**경로**: `Applications/Development/Analyzer/Str.Forge/`

### 구현 내용
- `Str.Forge.csproj` — DiffPlex 1.7.2 NuGet 참조
- `Models/ReplaceRule.cs` — 규칙 모델 (IsRegex/IgnoreCase/WholeWord, Compile/Apply)
- `Services/FileScanner.cs` — glob 패턴 파일 스캔, BOM/인코딩 자동 감지 (UTF-8/16/CP949)
- `Services/ReplaceEngine.cs` — Preview(비동기 진행률), Apply(백업 지원)
- `MainWindow.xaml` — 3열 레이아웃 (규칙 편집 / 파일 목록 / Diff 미리보기)
- `MainWindow.xaml.cs` — 전체 UI 로직 구현

### 핵심 기능
- 정규식/리터럴/전체단어/대소문자 무시 옵션
- DiffPlex 인라인 diff (추가=녹, 삭제=적)
- .bak 백업 생성 옵션
- 드래그앤드롭 폴더 오픈
- JSON 프리셋 저장/불러오기

## 2. Win.Scope (창 Z-order·투명도 실시간 조작기)

**경로**: `Applications/System/Manager/Win.Scope/`

### 구현 내용
- `Win.Scope.csproj`
- `Services/WindowEnumerator.cs` — `EnumWindows` P/Invoke, 투명도 읽기
- `MainWindow.xaml` — 2열 레이아웃 (창 목록 / 속성 패널)
- `MainWindow.xaml.cs` — SetWindowPos, SetLayeredWindowAttributes, ShowWindow

### 핵심 기능
- 실행 중 모든 창 열거 (클래스 필터링)
- 투명도 Slider (20~100%) 실시간 적용
- 항상 위(TopMost) 토글
- Z-Order: 맨 위 / 위로 / 아래로 / 맨 아래
- 창 복원/최소화/닫기, 포커스
- 3초 자동 새로고침
- 이름/프로세스 검색 필터

## 3. Orbit.Raid (관성 중력 침투 퍼즐)

**경로**: `Games/Puzzle/Orbit.Raid/`

### 구현 내용
- `Orbit.Raid.csproj`
- `Models/Body.cs` — 천체 모델 + Vector2D (완전 연산자 지원)
- `Models/Level.cs` — 5개 레벨 팩토리 (호만전이, 쌍성, 방해행성, 내행성 스윙바이, 다중 스윙바이)
- `Services/Simulator.cs` — RK4 수치 적분, SOI 충돌 감지, 궤도 예측
- `MainWindow.xaml` — 게임 캔버스 + 사이드 HUD + 결과 오버레이
- `MainWindow.xaml.cs` — 렌더링 (WPF Canvas), 입력 처리, 레벨 관리

### 핵심 기능
- RK4 4차 룬게-쿠타 물리 시뮬레이션
- 실제 물리 단위 (m, kg, G = 6.674e-11)
- 궤도 예측선 (300 스텝 미리 계산)
- 시뮬레이션 속도 조절 (1x~200x)
- 별 배경, SOI 표시 원, 드래그 패닝, 휠 줌

## 트러블슈팅

| 오류 | 원인 | 해결 |
|------|------|------|
| `BooleanToVisibilityConverter.Default` MC3011 | WPF에 해당 정적 멤버 없음 | XAML 바인딩 제거, 코드비하인드에서 `PropPanel.IsEnabled` 직접 설정 |
| `out nint → out uint` CS1503 | `GetWindowThreadProcessId` P/Invoke 시그니처 오류 | `out uint`로 수정 후 형변환 |
| `(List<Vector2D> acc)` 단일 튜플 CS1003 | C#에서 단일 요소 튜플 디컨스트럭션 불가 | 반환 타입 `List<Vector2D>`로 변경 |
