- **일시**: 2026-02-21 08:06 ~ 08:11 (KST)
- **태그**: `feature`
- **프로젝트**: `Games/Fist.Fury`

## 작업 내용

키보드로 플레이하는 횡스크롤 Beat'em Up 격투 액션 게임 개발.

### 구현 기능

| 기능 | 설명 |
|------|------|
| 게임 엔진 | `CompositionTarget.Rendering` 기반 게임 루프 (GameLoop, KeyInput) |
| 플레이어 | 방향키 이동, ↑ 점프 (중력 물리), Z 펀치 / X 킥 / C 특수기 |
| Fighter 시스템 | 상태 머신 (Idle/Walk/Jump/Punch/Kick/Special/Hit/Dead) |
| 적 AI | 자동 접근 → 거리 기반 공격 판단, 닌자 점프 |
| 적 종류 | Thug (일반), Brute (탱커), Ninja (빠름), Boss (최종 보스) |
| 전투 | 히트박스 충돌 감지, 넉백, 무적 시간, 콤보 카운터 |
| 웨이브 | 5단계 (점진적 난이도), 마지막 보스전 |
| UI | 타이틀, 플레이어 HP바, 보스 HP바, 콤보 표시, 점수, 웨이브 클리어 |
| 화면 | 게임오버(K.O.), 승리(Victory), 일시정지, 하이스코어 |
| 배경 | 도시 실루엣 + 불규칙 빌딩 + 창문 + 바닥 렌더링 |
| 다크 테마 | `#1A1A2E` 배경, 네온 액센트, DwmSetWindowAttribute 다크 타이틀바 |

### 아키텍처

```
Engine/
  GameLoop.cs        - 게임 루프 (Start/Stop/Pause/Resume)
  KeyInput.cs        - 키보드 입력 상태 추적
Entities/
  Fighter.cs         - 전투원 베이스 (물리, 공격, 히트박스, 비주얼)
  Player.cs          - 플레이어 (키 입력 바인딩)
  Enemy.cs           - 적 AI (Thug/Brute/Ninja/Boss)
  HitEffect.cs       - 타격 이펙트 (POW!)
```

### 조작법

| 키 | 동작 |
|----|------|
| ←→ | 이동 |
| ↑ / W | 점프 |
| Z | 펀치 (빠르고 약함) |
| X | 킥 (중간) |
| C | 특수기 (강하고 느림) |
| P | 일시정지/해제 |
| ENTER | 시작/재시작 |
| ESC | 타이틀로 |
