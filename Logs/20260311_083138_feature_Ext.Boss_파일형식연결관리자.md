# Ext.Boss — 파일 형식 연결 관리자 구현

**날짜**: 2026-03-11
**태그**: feature
**상태**: 진행 중

---

## 개요

`Applications/System/Ext.Boss` — Windows 파일 형식 기본 앱 연결 전체를 시각적 그리드로 관리·백업·복원하는 도구.

---

## 구현 내용

### 프로젝트 구조
- `Ext.Boss.csproj` — net10.0-windows, WPF, AssemblyName=Ext.Boss, SelfContained, PublishSingleFile
- `App.xaml` / `App.xaml.cs` — DwmSetWindowAttribute 다크 타이틀바 전역 적용, 다크 블루 테마 스타일
- `MainWindow.xaml` — 다크 블루 테마 (#12131C), 검색/필터/DataGrid/상태바 레이아웃
- `MainWindow.xaml.cs` — HKCR 레지스트리 읽기, 연결 관리, JSON 백업/복원
- `Resources/app.ico` — 멀티사이즈 (256/48/32/16px), 파일+링크 디자인

### 기능
- HKCR 레지스트리 전체 확장자 열거 + UserChoice 오버라이드 감지
- DataGrid: 확장자, ProgID, 앱 이름, 앱 경로, 소스, 상태
- 비동기 로딩 (Task.Run + Dispatcher)
- 상태: OK (실행파일 존재) / Broken (실행파일 없음) / Not Set (연결 없음)
- 검색 필터 (확장자·앱 이름), 상태 필터 콤보박스
- 앱 변경: OpenWith.exe 다이얼로그 호출
- Windows 기본 앱 설정 열기 (ms-settings:defaultapps)
- JSON 프로파일 내보내기/가져오기
- 관리자 권한 감지 및 재실행 요청

### 업데이트 파일
- `Playground.slnx` — Applications/System 폴더에 Ext.Boss 추가
- `+publish-all.cmd` — System 섹션 Env.Guard 다음에 Ext.Boss 추가
- `+publish.cmd` — Env.Guard(#22) 다음 #23에 Ext.Boss 삽입, 총 73개
