# Ink.Cast — 경량 로컬 마크다운 노트 앱 신규 개발

- **날짜**: 2026-03-03
- **태그**: feature
- **경로**: `Applications/Tools/Productivity/Ink.Cast`

---

## 개요

Obsidian 핵심 기능을 담은 완전 오프라인 마크다운 노트 앱 신규 구현.
프라이버시 중심 PKM(개인 지식 관리) 도구로, 클라우드 없이 모든 데이터를 로컬에 보관.

---

## 구현된 기능

| 기능 | 설명 |
|------|------|
| 마크다운 실시간 렌더링 | Markdig + WebView2 (GFM: 테이블·코드블록·체크리스트) |
| `[[위키링크]]` 자동완성 | 인라인 팝업 자동완성 (타이핑하며 필터링) |
| 양방향 링크 | note_links 테이블에 백링크 자동 추출·저장 |
| 태그 시스템 `#태그` | 콘텐츠에서 자동 추출, 태그별 필터 뷰 |
| 전체 텍스트 퍼지 검색 | SQLite FTS5 (prefix match, unicode61 tokenizer) |
| 그래프 뷰 | WPF Canvas + Fruchterman-Reingold Force-directed (드래그·줌·팬) |
| 일별 노트 | 날짜 제목 + 템플릿 자동 생성 |
| 다중 워크스페이스 | 논리적 워크스페이스 생성/삭제/전환 |
| 단일 HTML 내보내기 | 다크 테마 CSS 포함 독립 실행 HTML |
| 자동 저장 | 2초 디바운스 자동저장 |

---

## 기술 스택

- **프레임워크**: .NET 10 WPF
- **마크다운**: Markdig 0.40.0 (AdvancedExtensions + Emoji)
- **데이터베이스**: Microsoft.Data.Sqlite 9.0.3 + FTS5
- **미리보기**: Microsoft.Web.WebView2 1.0.3124.44
- **그래프**: 순수 WPF Canvas (외부 의존성 없음)
- **DB 경로**: `%APPDATA%\InkCast\inkcast.db`
