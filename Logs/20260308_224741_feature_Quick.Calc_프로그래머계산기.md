### 구현된 기능

#### 진법 실시간 동기화
- DEC / HEX / OCT / BIN 입력 필드
- 하나 수정 시 나머지 3개 자동 갱신
- DEC: 부호 있는 정수 (Word Size 기준)
- HEX: 0x 접두사 생략 가능
- BIN: 0b 접두사/`_` 구분자 허용

#### 비트 연산
- AND / OR / XOR / NOT / SHL / SHR 버튼
- AND/OR/XOR: 이항 연산 (= 버튼으로 적용, pending 상태 표시)
- NOT / SHL / SHR / 2s / SEXT: 즉시 단항 연산
- 2의 보수 (2s): `(~value) + 1`
- 부호 확장 (SEXT): 현재 Word Size MSB 기준 → 64비트

#### Word Size
- 8 / 16 / 32 / 64 비트 선택
- 마스킹 자동 적용, DEC는 부호 있는 표시 (sbyte/short/int/long)

#### 비트 레이아웃 (64개 토글 버튼)
- ViewModel.Bits: 64개 BitItem (bit63=MSB ~ bit0=LSB)
- 클릭 시 해당 비트 XOR 토글 → 즉시 전체 갱신

#### IEEE 754 시각화
- **Float32**: 하위 32비트를 float로 해석 (`BitConverter.UInt32BitsToSingle`)
  - Sign(1bit) / Exponent(8bit) / Mantissa(23bit) 색상 구분 영역
  - 각 비트 직접 클릭 편집
- **Float64**: 전체 64비트를 double로 해석 (`BitConverter.Int64BitsToDouble`)
  - DataTrigger로 IeeePart별 색상 구분 (Sign/Exp/Man)
  - 비트 직접 클릭 편집

#### 계산 히스토리
- 연산 실행 시 이전 값 자동 저장 (최대 50개)
- HEX / DEC / BIN 동시 표시
- 더블클릭으로 해당 값 재사용

#### 전역 단축키 Win+K
- `RegisterHotKey(MOD_WIN | VK_K)` — 앱 숨겨져 있어도 팝업
- X 버튼 클릭 시 종료 대신 숨김 (Shift+X로 실제 종료)
