## 구현 완료 목록

### 1. Schema.Mock (Applications/Development/Inspector)
- **설명**: JSON Schema / OpenAPI 3.x → Bogus Faker 가짜 데이터 생성기
- **기술**: .NET 10, WPF, Bogus 35.6.1, System.Text.Json
- **핵심 기능**:
  - JSON Schema Draft 7/2019-09/2020-12 및 OpenAPI 3.x `components/schemas` 파싱
  - `$ref` 로컬 참조 재귀 해결
  - 필드명/포맷 기반 Faker 전략 자동 감지 (AutoDetectStrategy)
  - DataGrid 전략 개별 설정 + 실시간 샘플 미리보기
  - 출력: JSON Array / NDJSON / CSV
- **버그 수정**: TextWrapping 중복, HexColor switch arm 중복

### 2. Dep.Graph (Applications/Development/Analyzer)
- **설명**: NuGet/npm/pip 의존성 버전 충돌 그래프 시각화기
- **기술**: .NET 10, WPF, WebView2, NuGet.Protocol 6.12.4
- **핵심 기능**:
  - `.csproj` / `packages.config` / `package.json` / `requirements.txt` 파싱
  - 자체 SVG 포스 시뮬레이션 그래프 (척력+인력+중심력 300 iter)
  - 버전 충돌 빨간 강조, 구버전 점선 표시
  - NuGet API 비동기 최신 버전 확인 (동시성 5개)
  - 마우스 패닝/줌, CSV/JSON 내보내기

### 3. Proc.Bench (Applications/System/Monitor)
- **설명**: CPU/메모리/스토리지 종합 벤치마크 스위트
- **기술**: .NET 10, WPF, unsafe C#, System.Management
- **핵심 기능**:
  - CPU: 정수(소수), 부동소수점(삼각함수), 멀티스레드, AES 256 암호화
  - 메모리: unsafe 포인터 기반 순차 읽기/쓰기, 랜덤 접근, 포인터 체이싱 지연
  - 스토리지: 순차 512MB, 랜덤 4K 읽기/쓰기 (FileOptions.WriteThrough)
  - 실시간 ProgressBar + 종합 점수 계산, 텍스트 내보내기

### 4. Data.Map (Applications/Data)
- **설명**: GeoJSON/TopoJSON/CSV 오프라인 인터랙티브 지도 시각화기
- **기술**: .NET 10, WPF, WebView2, Leaflet.js, CsvHelper 33.0.1
- **핵심 기능**:
  - CSS `invert+hue-rotate` 다크 지도 필터
  - GeoJSON Feature 색상/값 필드 매핑 → Choropleth 스타일링
  - CSV lat/lng 자동 감지, 색상/크기 필드 원형 마커
  - 팝업/툴팁 자동 생성, 줌/패닝

### 5. Morse.Run (Games/Casual)
- **설명**: 모스 부호 해독 + 사이드스크롤 러너 게임
- **기술**: .NET 10, WPF Canvas, DispatcherTimer 16ms 게임루프
- **핵심 기능**:
  - Space 짧게(≤300ms)=점·, 길게=선—, Enter=확정
  - A/E/I/N/T → 점프, S/H/5 → 슬라이드, J → 더블점프
  - 랜덤 장애물 생성 (지면/공중), 충돌 판정
  - 문자 출제 + 연속 정답 보너스 점수
  - 레벨업 시 장애물 속도 증가
