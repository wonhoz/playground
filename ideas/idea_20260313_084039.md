## Applications

### 1. `Tile.Cast` — 키보드 중심 타일링 윈도우 매니저

**카테고리**: Productivity / System
**핵심 기능**: Windows에서 키보드만으로 창을 자동 분할·이동·크기 조절하는 타일링 WM 트레이 앱. FancyZones보다 키보드 우선, Workspace.Switcher(배치 프리셋 저장)보다 동적·실시간 조작에 특화.

**상세 기능**:
- 전역 단축키로 활성 창을 좌/우/상/하 절반, 사분면, 전체화면에 즉시 스냅
- Master/Stack 레이아웃 모드: 주 창 비율(%) 키보드 조절
- 멀티 모니터 창 이동 단축키 (Win+Shift+Arrow처럼 모니터 간 이동)
- 레이아웃 프리셋 6개 저장·전환 (숫자 단축키)
- 제외 앱 목록 (게임·동영상 플레이어 등 타일링 제외)
- 창 포커스 이동: Vim 방식 hjkl 또는 화살표 키

**기술 스택**:
- WinForms 트레이 앱 + SetWindowPos/WinAPI
- Windows Hooks (RegisterHotKey)
- Win32 EnumWindows, GetWindowRect
- JSON 프로파일 저장

**수익 모델**: 완전 무료 오픈소스
**차별화**: Screen.Split(드래그 스냅 존 정의)·Workspace.Switcher(창 배치 저장/복원)와 달리 **실시간 키보드 구동 동적 타일링** — i3wm/Komorebi 개념의 Windows 경량 구현

### 2. `Token.Watch` — API 키·토큰·라이선스 만료 대시보드

**카테고리**: Dev / Tools / Security
**핵심 기능**: API 키, 서비스 토큰, 소프트웨어 라이선스, SSL 인증서(Cert.Watch와 통합 옵션)의 만료일을 한 화면에서 관리하는 로컬 대시보드. Cert.Watch(SSL 전용)와 달리 개발자의 모든 자격증명 만료 추적.

**상세 기능**:
- 항목 등록: 이름·만료일·URL·메모·카테고리 태그
- 만료 30일·7일·1일 Toast 알림 (Windows 알림 스케줄)
- 상태 컬러 코딩: 🟢 90일+ / 🟡 30일 이내 / 🔴 7일 이내 / ⚫ 만료
- 갱신 이력 로그 (갱신일 기록)
- JSON 단일 파일 저장 (AES-256 옵션 암호화)
- CSV 내보내기 / 가져오기
- 트레이 상주: 만료 임박 항목 툴팁 표시

**기술 스택**:
- WPF + MVVM
- Quartz.NET 스케줄러 (알림 타이밍)
- System.Security.Cryptography (AES-256-GCM 옵션 암호화)
- JSON 직렬화 (System.Text.Json)

**수익 모델**: 무료
**차별화**: Cert.Watch(SSL 만료 전용)·Key.Stash(키 저장소+env 주입 특화)와 달리 **범용 만료일 타임라인 대시보드** — 팀 내 모든 자격증명 만료 한눈에 관리

### 3. `Pad.Forge` — 게임패드 버튼 매핑 GUI

**카테고리**: Tools / Gaming / Automation
**핵심 기능**: XInput/DInput 게임패드의 버튼·축을 키보드·마우스·매크로 동작에 자유롭게 매핑하는 GUI 도구. Layout.Forge(키보드 Scancode Map)와 달리 게임패드 입력 가상화에 특화.

**상세 기능**:
- 연결된 컨트롤러 자동 감지 (XInput + DirectInput)
- 드래그&드롭 버튼→동작 매핑 (키 입력·마우스 클릭·텍스트 타이핑·스크립트 실행)
- 아날로그 스틱 축 커브 편집 (데드존·감도·반전)
- 프로파일 저장·불러오기 (앱별 자동 전환)
- ViGEmBus 가상 컨트롤러 출력 (게임 호환성 최대화)
- 버튼 테스터 UI (실시간 입력 시각화)
- 시스템 시작 시 자동 실행 트레이

**기술 스택**:
- WPF + MVVM
- SharpDX.DirectInput / XInput
- ViGEmClient (.NET 바인딩)
- Interprocess: SetForegroundWindow + WinAPI 앱 감지

**수익 모델**: 무료
**차별화**: Layout.Forge(키보드 물리 키 재배치)와 완전 다른 입력 장치 — 게임패드 축·버튼을 **가상 HID 출력으로** 변환하는 컨트롤러 리매퍼

### 4. `Desk.Pet` — 인터랙티브 데스크탑 펫

**카테고리**: Productivity / Fun / Lifestyle
**핵심 기능**: 바탕화면 위를 돌아다니는 투명 WPF 캐릭터 펫. 시스템 CPU·타이핑량·시간에 따라 감정이 변하고, 유저와 미니 인터랙션을 제공하는 앰비언트 동반자 앱.

**상세 기능**:
- WPF 투명 창(AllowsTransparency) + 항상 위(Topmost) + click-through(WS_EX_TRANSPARENT)
- 이동 AI: 랜덤 워크 + 화면 경계 반사 + 창 위에 앉기(EnumWindows)
- 감정 시스템: 게으름(타이핑 없음)·집중(타이핑 중)·흥분(CPU 고부하)·졸음(야간)
- 커스텀 스킨: PNG 스프라이트 시트 + JSON 애니메이션 정의 (기본 제공 3종)
- 클릭 인터랙션: 먹이주기·쓰다듬기·미니게임(Simon Says 3스텝)
- 타이핑 WPM 일별 통계 뷰 (서비스로 수집, 내용 저장 안 함)
- 트레이에서 보이기/숨기기 토글

**기술 스택**:
- WPF (AllowsTransparency, WS_EX_TRANSPARENT WinAPI 훅)
- DispatcherTimer 애니메이션 루프
- NAudio (간단한 효과음)
- LowLevelKeyboardHook (타이핑 카운트, 내용 무저장)

**수익 모델**: 무료 (스킨 팩 DLC 옵션)
**차별화**: 기존 목록에 없는 **앰비언트 생물 동반자** 장르 — 생산성 앱이 아닌 기분 전환·감성 UI

### 5. `Karaoke.Cast` — 로컬 오프라인 가라오케 플레이어

**카테고리**: Audio / Entertainment
**핵심 기능**: LRC 가사 동기화 + 오디오 보컬 분리로 로컬 MP3/FLAC을 즉석 가라오케 트랙으로 변환하는 플레이어. Music.Player(재생 목록)·Echo.Text(TTS)와 달리 마이크 믹서 + 가사 비주얼에 특화.

**상세 기능**:
- LRC 파일 자동 검색 + 실시간 가사 하이라이트 (카라오케 스타일 스크롤)
- Demucs 경량 모델 통합: 보컬 분리 (오프라인 ONNX 추론)
- 마이크 실시간 믹서: 볼륨·에코·리버브 슬라이더
- 키 조절: 반음 단위 ±6 피치 시프트 (NAudio + SoundTouch)
- 점수 시스템: 마이크 입력 기본 피치 매칭 → 별점 표시
- 재생 목록 + 즐겨찾기
- 전체화면 노래방 모드 (2번 모니터 지원)

**기술 스택**:
- WPF + MediaElement / NAudio
- SoundTouch.NET (피치 시프트)
- ONNX Runtime (Demucs 경량 분리 모델)
- NAudio WASAPI (마이크 캡처 + 믹싱)

**수익 모델**: 무료
**차별화**: Music.Player(재생 위주)·Pitch.Find(BPM/키 분석 도구)와 달리 **마이크 믹서 + 보컬 분리 + 가사 동기화** 삼위일체의 순수 가라오케 경험

### 6. `Flow.Beat` — 바이노럴 비트·집중 음악 로컬 생성기

**카테고리**: Audio / Wellness / Productivity
**핵심 기능**: 뇌파 동조 효과를 위한 바이노럴 비트와 Lo-Fi 배경음을 로컬에서 생성하는 앱. Ambient.Mixer(환경음 믹스)·Sound.Jar(앰비언트 UI)와 달리 주파수 수학 기반 신호 생성에 특화.

**상세 기능**:
- 바이노럴 비트 생성: 좌/우 채널 주파수 차이로 Delta(1-4Hz)·Theta(4-8Hz)·Alpha(8-13Hz)·Beta(13-30Hz) 선택
- 집중·수면·명상·창의력 4가지 프리셋 (뇌파 주파수 자동 설정)
- 배경 노이즈 레이어: White·Pink·Brown Noise + 빗소리·파도 기본 내장
- 세션 타이머: 25분 집중 / 5분 휴식 포모도로 연동
- WAV/MP3 24시간 루프 파일 내보내기
- 볼륨·주파수·노이즈 믹스 비율 실시간 조절 + 파형 시각화
- 트레이 상주 + 전역 단축키 (일시정지/재개)

**기술 스택**:
- WPF + NAudio (신호 생성 + WASAPI 출력)
- 수학적 사인파 생성 (Math.Sin, 44100Hz PCM)
- LAMEenc (MP3 내보내기)
- DispatcherTimer (포모도로 세션)

**수익 모델**: 무료
**차별화**: Ambient.Mixer(기록된 사운드 믹스)와 달리 **수학적 주파수 생성** — 오프라인에서 과학 기반 뇌파 동조 신호를 직접 합성

### 7. `Book.Log` — 독서 기록·목표 트래커

**카테고리**: Productivity / Lifestyle
**핵심 기능**: 읽은 책·읽는 중·읽고 싶은 책을 로컬에서 추적하고, 독서 통계와 목표 달성을 시각화하는 앱. Epub.Cast(전자책 리더)·Ink.Cast(마크다운 노트)와 달리 종이책 포함 독서 메타데이터 관리에 특화.

**상세 기능**:

- 읽기 세션 기록: 날짜별 페이지 범위 입력 → 일/주/월 독서량 그래프
- 연간 독서 목표 설정 + GitHub 잔디형 독서 히트맵
- 인용/메모 저장: 페이지 번호 연동 노트 (책 내 검색)
- 별점·리뷰 작성 + 장르별 통계 파이차트
- 독서 목록 3단계 뷰: 읽음 / 읽는 중 / 읽고 싶음
- JSON 단일 파일 로컬 저장 / CSV+Markdown 내보내기

**기술 스택**:
- WPF + MVVM
- SQLite (독서 세션 이력 저장)
- SkiaSharp (히트맵·차트 렌더링)
- Google Books API (선택적 메타데이터 자동완성)

**수익 모델**: 무료
**차별화**: Epub.Cast(전자책 뷰어)와 달리 종이책+전자책 모든 매체를 아우르는 **독서 메타 추적기** — Goodreads 오프라인 대체

### 9. `Mirror.Cast` — 스마트폰 화면 USB/WiFi 미러링 뷰어

**카테고리**: Tools / Mobile / Productivity
**핵심 기능**: Android 스마트폰 화면을 PC에서 USB 또는 WiFi로 실시간 미러링하는 GUI 앱. scrcpy 엔진을 WPF GUI로 래핑하여 일반 사용자도 쉽게 사용 가능.

- 전체화면 / 창 모드 / 항상 위 토글
- PC→폰 마우스·키보드 입력 전달 (scrcpy 터치 가상화)

- LibVLCSharp (H.264 저지연 디코딩)

**수익 모델**: 무료
**차별화**: Win.Cast(앱 창→가상 카메라)·Screen.Recorder(PC 화면 녹화)와 완전 다른 방향 — **스마트폰→PC 방향 미러링** (scrcpy의 GUI 친화 래퍼)

**기술**: WPF Canvas + 탄성 충돌 물리 (운동량 보존, 완전 탄성 계수 e=1), A* 스케이터 AI 경로, AABB 충돌
**매력**: 예측선 대로 안 튀는 짜릿함, 코너 반사 골의 쾌감, 가속 타일 활용 전략
**차별화**: Pool.Break(당구 회전·마찰 물리)와 달리 **마찰=0 완전 탄성 + 실시간 팀 스포츠** — 완전히 다른 스포츠 물리

### 10. `Tune.Cast` — 라디오 주파수 탐색 추리 게임

**장르**: Puzzle / Mystery / Atmospheric
**핵심 메커닉**: 노이즈로 가득 찬 라디오 주파수 다이얼을 천천히 돌려 숨겨진 신호를 찾아내는 탐정 퍼즐 게임. Signal.Cast(신호 시퀀스 기억·재현)와 달리 **주파수 공간 탐색 + 암호 메시지 해독 내러티브**가 핵심.

**게임플레이**:
- 라디오 다이얼 UI: 88.0~108.0 MHz 드래그 → 노이즈 강도 변화
- 목표 주파수 근처에서 신호 강도계 반응 (소나처럼 거리 힌트)
- 수신 성공 시: 음성 메시지(모스·목소리·음악 조각) 조각 수집
- 3스테이지: 조각들을 타임라인에 재배열 → 전체 메시지 복원
- 다중 채널 간섭: 두 주파수 동시 수신 → 위상 분리 미니퍼즐
- 배경: 1980년대 냉전 스파이 설정 (5챕터, 각 15-20분)
- 사운드 디자인: 실제 AM/FM 라디오 잡음 시뮬레이션

**기술**: NAudio 신호 생성 (사인파 믹스 + 화이트 노이즈 합성), 주파수 스펙트럼 시각화, WPF 아날로그 다이얼 UI
**매력**: 오래된 라디오를 돌리는 촉각적 긴장감, 파편 조합의 퍼즐 만족감, 냉전 분위기의 몰입감
**차별화**: Signal.Cast(청각·시각 신호 재현 메모리)·Echo.Hunt(소나 어둠 탐색)와 달리 **아날로그 주파수 다이얼 + 스파이 내러티브** — 라디오 물리 시뮬과 추리 결합
