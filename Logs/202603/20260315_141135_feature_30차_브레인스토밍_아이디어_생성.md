### 3. TOP 3 추천

#### 1위: `Perf.Flame` — .NET 프로파일러 + 플레임 그래프
- **이유**: 개발자 실용성 최고. Visual Studio 없이 성능 병목을 즉시 파악할 수 있는 도구로 일상적 사용 빈도가 높음. `DiagnosticsClient` + `d3-flame-graph` 조합이 기술적으로 흥미로움

#### 2위: `Peg.Web` — String Art 기하 패턴 퍼즐
- **이유**: 비주얼 완성도가 높고 소셜 공유 욕구를 자극. 수학적 규칙(에피사이클로이드·파라메트릭 곡선)이 게임 메커닉으로 자연스럽게 녹아듦. 자유 창작 모드가 차별점

#### 3위: `Token.Calc` — LLM 비용 계산기
- **이유**: LLM 사용이 일상화된 현 시점에 실용성 극대화. tiktoken 포트(SharpToken)로 로컬 정확한 계산. 직접 쓰면서 유용한 도구
