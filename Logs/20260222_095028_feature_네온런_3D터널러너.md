## 작업 내용

WPF Viewport3D를 활용한 3D 네온 와이어프레임 터널 러너 게임 개발.

### 구현 기능

| 기능 | 설명 |
|------|------|
| 3D 터널 | Viewport3D 기반 네온 와이어프레임 터널, 실시간 생성/재활용 |
| 3레인 시스템 | 좌/중앙/우 3개 레인, 좌우 화살표로 전환 |
| 장애물 | Wall (벽, 충돌=사망), LowBar (낮은 바, 점프로 회피) |
| 크리스탈 | 골드 큐브 수집, 회전 애니메이션 |
| 점프 | SPACE 키, 사인파 곡선 점프 |
| 속도 증가 | 거리에 비례하여 속도 점진 증가 (최대 60u/s) |
| 스코어 | 크리스탈 50점 + 장애물 통과 10점 × 거리 배율 |
| 3D 렌더링 | 앰비언트/디렉셔널/포인트 라이트, 에미시브 재질 |
| 네온 비주얼 | 시안/핑크/블루 네온 색상, 터널 벽 스트라이프 |
| 카메라 | 3인칭 추적 카메라, 플레이어 이동에 따른 부드러운 추적 |

### 아키텍처

```
Engine/
  GameLoop.cs            - CompositionTarget.Rendering 게임 루프
  Obstacle.cs            - 장애물/크리스탈 데이터 (Wall/LowBar/Crystal)
  TrackGenerator.cs      - 무한 트랙 생성, 장애물/크리스탈 스폰
MainWindow.xaml/.cs      - 3D Viewport3D 씬, 메시 생성, 게임 로직
```

### 3D 기술 스택

- **Viewport3D** + PerspectiveCamera
- **MeshGeometry3D**: 큐브/평면 프로시저럴 생성
- **MaterialGroup**: DiffuseMaterial + EmissiveMaterial (네온 발광)
- **AmbientLight / DirectionalLight / PointLight**
- **Transform3DGroup**: 이동 + 회전 조합
