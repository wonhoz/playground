# Zip.Peek 초기 구현

- **날짜**: 2026-03-05
- **태그**: feature
- **프로젝트**: Applications/Files/Zip.Peek

## 작업 내용

### 프로젝트 구조
- `Zip.Peek.csproj` — net10.0-windows, WPF, SharpCompress 0.38.0
- `GlobalUsings.cs` — System.IO, WPF 네임스페이스
- `App.xaml` — 다크 테마 (Button, TextBox, CheckBox, ScrollBar, TreeView, TreeViewItem, ListBox, TabControl, ProgressBar, GridSplitter ControlTemplate), 포인트 색 황금 #F59E0B

### MainWindow
- 상단 툴바: 로고, 파일 열기, 검색바(이름+내용), 선택추출, 전체추출
- 하단 상태바: 상태 텍스트 + 통계 + 진행바
- 드래그&드롭 지원 (아카이브 파일 드롭으로 열기)
- `DwmSetWindowAttribute` 다크 타이틀바
- OpenFolderDialog (.NET 8+ WPF 내장) 사용

### 서비스 레이어
- `ArchiveService` — SharpCompress ArchiveFactory로 ZIP/7z/RAR/TAR/GZ/BZ2/XZ 지원
  - `Open()` → 트리 노드 구조 + 통계 반환 (중간 폴더 자동 생성)
  - `ExtractNodesAsync()` / `ExtractAllAsync()` — 진행 보고 포함
  - `ReadEntry()` — 미리보기용 바이트 읽기
- `PreviewService` — 텍스트/이미지/Hex 자동 판별
  - BOM 감지 (UTF-8, UTF-16 LE/BE), 비텍스트 파일 Hex 덤프
  - 이미지: BitmapFrame.Create로 WPF BitmapSource 반환
- `SearchService` — 파일명 + 내용(텍스트) grep
  - CancellationToken 지원, 2MB 이상 텍스트 파일 스킵

### ArchiveView (메인 뷰)
- 좌측 340px: TreeView (HierarchicalDataTemplate, 파일 아이콘, 압축/원본 크기, 압축률)
- 우측: TabControl — 텍스트(모노스페이스), 이미지, Hex, 검색결과 4탭
- 트리 선택 시 자동 미리보기, 검색결과 클릭 시 해당 줄로 스크롤
- GridSplitter로 좌우 너비 조절 가능

### 아이콘
- `Resources/app.ico` — 다크 배경, 황금색 테두리 박스 + 지퍼 줄 (16/32/48/256px)
