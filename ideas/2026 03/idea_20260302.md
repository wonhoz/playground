### 1. `Disk.Lens` — 디스크 사용량 트리맵 시각화기

**카테고리**: System / Files
**컨셉**: WinDirStat·TreeSize 대체. 폴더·파일 크기를 면적 비례 트리맵(Treemap)으로 시각화, 숨어있는 거대 파일을 즉각 발견.

**핵심 기능**
- Squarified 알고리즘 트리맵 렌더링 (색상 = 파일 확장자 유형별)
- 클릭 드릴다운 → 폴더 내부 확대 / 더블클릭 상위 폴더로 복귀
- 우클릭 컨텍스트 메뉴: Explorer 열기 / 파일 삭제 / 경로 복사
- 최대 파일 TOP 20 목록 사이드패널 (크기·경로·수정일)
- 확장자별 필터·색상 커스텀 (예: .mp4는 파란색)
- 스캔 진행률 표시 + 백그라운드 비동기 처리
- 드라이브/폴더 전환 콤보박스

**수익화/비전**
- IT 관리자·파워유저 필수 도구 (즉각적 실무 가치)
- TreeSize Pro: 상용 $29 → 완전 무료·오프라인으로 차별화
- 한국 기업 PC 관리 솔루션 번들 연동 잠재력

**기술 스택**: WPF Canvas 직접 렌더링, Directory.EnumerateFiles 비동기, WPF MVVM

### 2. `Img.Forge` — 이미지 일괄 처리기

**카테고리**: Files / Media
**컨셉**: 이미지 수십~수백 장을 폴더 단위로 한번에 리사이즈·변환·필터·워터마크 적용. Photo.Video.Organizer(날짜 정리기)와 달리 이미지 처리·편집 특화.

**핵심 기능**
- 드래그&드롭 폴더/파일 입력
- 출력 형식: JPEG(품질 조절)/PNG/WebP/BMP/TIFF 변환
- 조작: 리사이즈(비율유지·픽셀지정·퍼센트), 회전·뒤집기, 크롭(중앙/사용자정의)
- 필터: 밝기·대비·채도·선명도·흑백·세피아·블러
- 워터마크: 텍스트(폰트·크기·색상·불투명도) + 이미지(위치 9방향)
- 폴더 감시 모드: 새 파일 감지 시 규칙 자동 적용
- 처리 전후 비교 미리보기 (슬라이더 split view)
- 배치 작업 큐 + 병렬 처리 진행률 바

**수익화/비전**
- 블로거·콘텐츠 크리에이터·쇼핑몰 운영자 타깃 (반복 작업 자동화)
- Pro 버전: 스크립트/CLI 자동화, 스케줄러, 클라우드 폴더 연동

**기술 스택**: WPF MVVM, ImageSharp (System.Drawing 대체), TPL 병렬처리

### 3. `Pass.Vault` — 완전 로컬 비밀번호 관리자

**카테고리**: Security / Productivity
**컨셉**: 클라우드 없이 완전 오프라인 동작. AES-256-GCM + Argon2id로 단일 마스터 패스워드가 모든 자격증명을 보호. Secret.Box(.env 파일 특화)와 달리 범용 자격증명 관리.

**핵심 기능**
- AES-256-GCM 암호화, 키 유도: Argon2id (메모리 집약적 브루트포스 방어)
- 항목 필드: 사이트·로그인ID·비밀번호·메모·URL·카테고리·태그·생성일
- 자동 클립보드 클리어 타이머 (30초 후 비밀번호 자동 삭제)
- 비밀번호 강도 측정기 (zxcvbn 알고리즘 기반)
- 비밀번호 생성기 (길이·대소문자·숫자·특수문자·발음가능 옵션)
- 중복·약한 비밀번호 보안 감사 리포트
- 암호화된 단일 파일 내보내기 (자체 포맷) / CSV 내보내기
- Windows Hello 생체인증(지문/얼굴) 잠금해제 지원

**수익화/비전**
- Bitwarden/LastPass 클라우드 불신 사용자 오프라인 대안 수요 큼
- Pro: 팀 공유 볼트 (LAN 동기화), USB 이동식 설치 버전

**기술 스택**: WPF, BouncyCastle/libsodium-net, Windows Hello API, SQLite

### 4. `Zone.Watch` — 다중 시간대 세계시계 트레이

**카테고리**: Tray / Productivity
**컨셉**: 여러 도시·팀원의 시간대를 트레이 팝업에서 즉시 확인. 미팅 최적 시간대 교차 계산 포함.

**핵심 기능**
- 트레이 팝업: 등록된 도시별 아날로그+디지털 시계 카드 (야간 표시 어둡게)
- 200+ 타임존 데이터베이스 (IANA tz)
- 팀원 이름 매핑 (예: "김팀장 — Seoul KST", "John — NYC EST")
- **미팅 타임파인더**: 등록된 모든 타임존에서 겹치는 업무시간(09~18h) 구간 시각적 강조
- 특정 시각 ↔ 각 타임존 양방향 변환기 (팝업 계산기)
- 세계지도 오버레이 뷰 (주간/야간 경계선 실시간 표시)
- 클릭 시 해당 타임존 시각을 클립보드 복사

**수익화/비전**
- 글로벌 팀 협업 · 리모트워크 필수 도구 수요 급증
- 프리랜서, 해외 클라이언트 대상 포지셔닝

**기술 스택**: WinForms Tray, TimeZoneInfo, Canvas 아날로그 시계 렌더링

- 인력 발생기·척력 발생기·소용돌이 생성기를 슬롯에 배치
- 우주선은 모든 발생기의 합력 벡터에 따라 이동
- 배치 완료 후 시뮬레이션 실행 → 궤도 예측선으로 사전 확인 가능
- 특수 오브젝트: 중력 증폭기, 반중력 게이트, 순간 정지 구역, 중력 역전 지점
- 최소 발생기 수 사용이 별 3개 조건

**물리 구현 포인트**
- 인력: `F = -G*m / r²` (거리 제곱 반비례, 방향: 발생기 쪽)
- 척력: `F = +G*m / r²` (반대 방향)
- 소용돌이: 발생기 중심 접선 방향 힘 추가 `F_tangent = ω × r`
- Runge-Kutta 4차(RK4)로 궤도 예측 계산 (시뮬레이션 전 미리보기)

**중독성 요소**
- 중력 궤도 설계의 전략적 깊이
- 우주 배경 + 궤도 시각화의 미적 감각
- 레벨 에디터 공유로 커뮤니티 확장 잠재력

---

### G10. `Orbit.Craft` — N-체 중력 궤도 시뮬레이션 퍼즐

**장르**: Puzzle / Space Simulation
**물리 핵심**: N-체 중력(뉴턴 만유인력), 궤도 역학(케플러), 탈출 속도

**게임플레이**
- 행성 질량·위치 조정 → 탐사선 발사 각도·속도 설정 → 안정 궤도 형성 목표
- 목표: 탐사선이 목표 행성 주위를 N회 공전 (궤도 이탈 없이)
- 실패 조건: 탈출 속도 초과(날아감) / 행성 충돌 / 지나치게 타원형 궤도
- 탐사선 지나온 경로 궤도선으로 시각화
- 고급 레벨: 3체 카오스 궤도 안정화, 라그랑주 포인트(L4/L5) 활용
- 교육 모드: 케플러 제1·2·3법칙 설명 패널

**물리 구현 포인트**
- `F = G*m₁*m₂/r²` N-체 합산 (쌍별 계산, 소규모 N)
- Symplectic Euler or Leapfrog Integration (에너지 보존 중요, RK4보다 장기 안정)
- 탈출 속도 실시간 계산: `v_esc = √(2GM/r)`
- 라그랑주 포인트: 세 천체의 합력이 0이 되는 점 계산

**중독성 요소**
- 실제 물리학(케플러 법칙) 학습 효과
- 우주 감성 + 수학적 아름다움 (타원 궤도 완성의 쾌감)
- "나만의 태양계 만들기" 창작 성취감

### G11. `Rogue.Tile` — 절차 생성 턴제 로그라이크

**장르**: Roguelike / Turn-based Strategy
**핵심 메커닉**: 절차 생성 던전, 영구 사망(Permadeath), 턴제 전투, 아이템 시너지

**게임플레이**
- 타일 기반 15×15 던전 (층마다 절차 생성, BSP 알고리즘)
- 방향키 이동·공격, 이동마다 적도 한 턴 행동 (턴제)
- 클래스: 검사(고체력·근접), 마법사(원거리·취약), 도적(회피·치명타)
- 아이템 시너지: 불 검 + 기름 바닥 → 광역 화염 / 얼음 갑옷 + 물 → 빙결 방어
- 죽으면 처음부터 (단 영구 해금: 플레이 누적으로 새 클래스·유물 해금)
- Dungeon.Dash(실시간 액션)와 차별화: 완전 턴제·전략·계획 중심

**중독성 요소**
- "이번 조합은 달라" 무한 시너지 탐색
- 영구 사망의 긴장감 → 성공 시 극대 성취감

### G12. `Word.Bomb` — 단어 폭발 스피드 게임

**장르**: Word / Speed Puzzle
**핵심 메커닉**: 스트레스 폭발 단어 입력, 폭탄 타이머, 콤보 체인

**게임플레이**
- 화면에 폭탄이 등장 (점점 빠른 속도로), 폭탄 안에 영어/한글 음절이 표시
- 해당 음절을 포함하는 단어를 빠르게 타이핑 → 폭탄 제거
- 연속 제거 시 콤보 보너스 배율, 폭탄이 터지면 HP 감소
- 한글 버전: 초성·받침 포함 단어 → 독특한 로컬라이징
- Glyph.Rush(슈터 + 타이핑)와 달리 순수 단어 생성 능력 중심
- 멀티: 같은 음절로 긴 단어 먼저 입력한 사람이 이기는 대결 모드

**중독성 요소**
- 단어 사전 실력이 곧 전투력 → 어휘력 향상 효과
- 시간 압박 속 단어 떠올리기 긴장감

### G13. `Ink.Spread` — 잉크 전파 차단 퍼즐

**장르**: Puzzle / Strategy
**핵심 메커닉**: 셀룰러 전파(BFS/Flood Fill), 경계선(Barrier) 그리기, 영역 분리 최적화

**게임플레이**
- N×N 격자에 잉크 방울이 BFS로 번지기 시작
- 마우스 드래그로 격자 선분 위에 차단벽 설치 (잉크 이동 불가 경계)
- 목표: 잉크가 보호 구역(금색 셀)에 도달하기 전에 완전 격리
- 벽 설치 수 제한 → 최소 벽으로 최대 격리 (위상 수학적 최적화)
- 복수 잉크 색상 레벨: 색상마다 독립 전파, 섞이면 게임 오버
- 타임 어택 모드: 잉크 속도가 점점 빨라짐 / 최소 벽 도전 모드

**중독성 요소**
- 퍼지는 잉크의 시각적 불안감 → 차단 성공의 안도감
- 수학적 최적 분리선 탐색의 두뇌 도전
- 연쇄 전파 시각화가 시각적으로 매력적
