## S급 — 즉시 착수 추천

### 1. `Api.Probe` — 미니멀 오프라인 API 테스터

**문제 정의**
Postman은 너무 무겁고, 클라우드 로그인을 강요하며, 사용 기록이 외부에 저장됨.
개발자가 원하는 건 "URL 입력 → 헤더/바디 설정 → 즉시 전송" 뿐.

**킬러 기능**
- REST / GraphQL 요청 작성 + 즉시 전송
- 컬렉션 로컬 저장 (JSON 파일, Git 추적 가능)
- 환경변수 프리셋 (dev/stage/prod URL 전환)
- 응답 JSON 트리 뷰 + 하이라이팅
- cURL 명령어 자동 변환 복사

**수익 모델**
- 무료 (로컬 전용) / Pro $4.99/월: 팀 컬렉션 공유, History 무제한, OpenAPI 임포트

**구현 난이도** ★★★☆☆ (HttpClient + WPF 에디터)

### 2. `Git.Reel` — 시각적 Git 커밋 그래프 & 히스토리 탐색기

**문제 정의**
Git 브랜치 그래프를 제대로 보려면 IDE를 켜야 하고, GitKraken·SourceTree는 무겁거나 유료.
가볍게 폴더만 선택하면 커밋 그래프·diff·blame을 즉시 보여주는 독립 앱이 없음.

**킬러 기능**
- 폴더 열기 → 브랜치 그래프 시각화 (SVG 렌더링)
- 커밋 선택 → 변경 파일 목록 + diff 뷰
- Blame 타임라인 (파일별 변경 이력)
- 커밋 검색 (메시지, 작성자, 날짜 범위)
- 로컬 전용, 인증 불필요

**수익 모델**
- 무료: 단일 저장소 / Pro $5.99/월: 다중 저장소, Pull Request 뷰, Stash 관리

**구현 난이도** ★★★☆☆ (libgit2sharp + WPF Canvas 그래프)

### 3. `Regex.Lab` — 정규식 실시간 테스터 & 빌더

**문제 정의**
regex101.com은 훌륭하지만 브라우저 의존적, 인터넷 없으면 못 씀.
개발 중 빠르게 패턴 검증하고 저장해두고 싶다.

**킬러 기능**
- 패턴 입력 → 테스트 문자열에서 실시간 하이라이팅
- 그룹 캡처 결과 컬럼별 표시
- 플래그 토글 (i, m, s, g)
- 저장된 패턴 라이브러리 (이름 + 설명)
- 자연어 → 정규식 힌트 (AI 선택 연동)
- 언어별 코드 스니펫 생성 (C#, JS, Python, Go)

**수익 모델**
- 무료 / Pro $2.99/월: 팀 패턴 공유, AI 힌트 무제한

**구현 난이도** ★★☆☆☆ (.NET Regex 엔진 + WPF RichTextBox 하이라이팅)

### 4. `Json.Craft` — JSON / YAML / TOML 포맷터 + 뷰어 + diff

**문제 정의**
VS Code를 켜기엔 오버킬, 온라인 포맷터는 민감 데이터를 붙여넣기 꺼림.
가볍고 오프라인인 JSON/YAML/TOML 올인원 도구가 절실.

**킬러 기능**
- 붙여넣기 → 즉시 자동 포맷 + 트리 뷰
- JSON ↔ YAML ↔ TOML 상호 변환
- 좌우 diff 뷰 (두 JSON 비교)
- JSON Schema 검증 + 오류 라인 하이라이팅
- 경로 복사 (jQuery 스타일 `$.a.b[0].c`)
- jq 쿼리 실행

**수익 모델**
- 무료 / Pro $3.99/월: 대용량 파일 처리, JSON5 지원, CSV 변환

**구현 난이도** ★★☆☆☆ (System.Text.Json + YamlDotNet + WPF TreeView)

### 5. `Secret.Box` — 로컬 암호화 시크릿 & .env 관리자

**문제 정의**
API 키, DB 비밀번호, .env 파일이 평문으로 여기저기 흩어짐.
클라우드 비밀번호 관리자(1Password 등)는 코드 워크플로우와 안 맞음.
Env.Guard(Windows 환경변수 GUI)와 달리, **프로젝트별 .env 파일**을 AES 암호화해서 보관.

**킬러 기능**
- 프로젝트별 .env 파일을 AES-256으로 암호화 저장
- 마스터 비밀번호 한 번으로 언락
- .env 파일 내보내기 (선택한 프로젝트만)
- 변수 이름 자동완성 (공통 패턴: DATABASE_URL, API_KEY 등)
- Git 훅 연동: commit 전 평문 .env 노출 경고
- 팀 공유: 암호화된 파일만 공유 (복호화 키는 별도 전달)

**수익 모델**
- 무료: 3개 프로젝트 / Pro $4.99/월: 무제한 + 팀 공유 + 자동 백업

**구현 난이도** ★★★☆☆ (AES-256 + SQLite + WPF)

### 6. `Uptime.Eye` — 웹사이트·API 가동 상태 모니터 트레이 앱

**문제 정의**
Cert.Watch(SSL 만료 감시)와 달리, **실시간 HTTP 가동 여부**를 트레이에서 상시 감시.
UptimeRobot 같은 외부 서비스 대신, 내 PC에서 직접 폴링.

**킬러 기능**
- URL 목록 등록 → 주기적 HTTP 상태 폴링 (30초~5분)
- 상태 변경(UP↔DOWN) 즉시 Toast 알림
- 트레이 아이콘 색상으로 전체 상태 표시 (녹색/노랑/빨강)
- 응답 시간 히스토리 그래프
- 인증 헤더, 커스텀 바디 지원 (API 헬스체크)

**수익 모델**
- 무료: 5개 URL / Pro $3.99/월: 무제한 + Slack/Teams 알림 + 통계 리포트

**구현 난이도** ★★☆☆☆ (HttpClient 폴링 + 트레이 + 간단한 차트)

### 10. `Hash.Forge` — 해시·인코딩 올인원 유틸리티

**문제 정의**
개발 중 MD5, SHA-256, Base64, JWT 디코딩, URL 인코딩을 매번 온라인 도구에서 함.
민감 데이터(토큰, 비밀번호 해시)를 외부 사이트에 붙여넣는 게 찝찝함.

**킬러 기능**
- 해시: MD5, SHA-1/256/512, HMAC, bcrypt 검증
- 인코딩: Base64, URL, HTML Entity, Hex
- JWT 디코더 (헤더/페이로드 파싱, 만료 시간 표시)
- UUID/ULID 생성기
- 비밀번호 강도 분석 + 무작위 비밀번호 생성

**수익 모델**
- 무료 (단독 유틸이라 광고 없이 바이럴) / Pro $1.99/월: 배치 처리, API 연동

**구현 난이도** ★★☆☆☆ (System.Security.Cryptography + WPF)
