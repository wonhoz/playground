### 1. `Auto.Type` — 텍스트 자동 확장 스니펫

**문제 정의**
개발자, 디자이너, 직장인이 이메일 서명, 주소, 코드 보일러플레이트, 자주 쓰는 문구를
매번 타이핑하거나 클립보드에서 찾아 붙여넣는다.
무료 대안인 Espanso는 오픈소스지만 설정 파일 기반(YAML)이라 일반인 진입장벽이 높고,
PhraseExpress는 $49.95로 비싸다. GUI 기반 저렴한 도구의 공백이 실재한다.

**킬러 기능**
- 트리거 입력: `::addr` → `서울시 강남구 테헤란로 123` 자동 치환
- 변수 지원: `{date}` (오늘 날짜), `{time}` (현재 시각), `{clip}` (현재 클립보드 내용)
- 스니펫 에디터: 이름·트리거·본문 3필드, 그룹(폴더) 분류
- 전역 단축키 팝업 검색: 단축키 → 스니펫 이름 퍼지 검색 → 즉시 삽입
- 이모지/특수문자 스니펫 지원 (`::heart` → ❤️)
- 한글 IME 완벽 지원 (트리거 인식 후 IME 상태 보존)
- 트레이 상주, 스니펫 ON/OFF 토글
- 스니펫 내보내기/가져오기 (JSON)

**수익 모델**
- 무료: 스니펫 50개 (일반 사용자 충분)
- Pro $4.99 일회성: 스니펫 무제한, 그룹 무제한, `{cursor}` 커서 위치 지정, JSON 백업/복원
- **타겟**: 개발자, 고객지원 담당자, 마케터, 의료·법률 분야 문서 작업자
- **차별화**: Espanso 대비 GUI 완전 지원 / PhraseExpress 대비 10분의 1 가격 / 완전 오프라인

**경쟁자 분석 (에이전트 검증)**
| 경쟁자 | 가격 | 약점 |
|--------|------|------|
| Espanso | 무료 | YAML 설정 파일 → GUI 없음, 진입장벽 높음 |
| PhraseExpress | $49.95 | 고가, 기업 타겟 |
| AutoHotkey | 무료 | 스크립팅 필요, 비개발자 사용 불가 |

**구현 난이도** ★★★☆☆
전역 키보드 훅(LowLevelKeyboardProc) + 입력 패턴 인식 + SendInput + 스니펫 DB(SQLite)

### 3. `Glyph.Map` — 유니코드 / 이모지 / 특수문자 오프라인 탐색기

BabelMap이 유일한 대안이지만 2003년대 UI 수준, 이모지 미지원.
개발자는 HTML 엔티티·Python 코드·CSS 유니코드까지 동시에 필요한데 복사 한 번으로 해결이 안 된다.

**킬러 기능**

- 복사 형식 선택 패널: 문자 자체 / U+HHHH / &#xHHHH; / \uHHHH / %uHHHH / CSS content값
- 즐겨찾기 저장 (자주 쓰는 문자 핀)

- 조합 문자 지원: 기저 문자 + 발음 기호 조합 미리보기

### 4. `Hot.Corner` — Windows 핫 코너 구현 (macOS 스타일)

**문제 정의**
macOS 핫 코너 기능 — 화면 4개 모서리에 마우스를 갖다 대면 Mission Control,
데스크탑 보기, 화면보호기 등이 실행되는 — 은 Windows에 기본 없다.
macOS에서 Windows로 이동한 사용자가 가장 먼저 찾는 기능 중 하나이며,
Microsoft PowerToys에도 이 기능은 없다. 현존하는 도구(WinXCorners 등)는 UI가 빈약하다.

**킬러 기능**
- 화면 4 모서리 각각에 동작 할당:
  - 화면 잠금 / 화면보호기 실행
  - 바탕화면 보기 (모든 창 최소화)
  - 작업 보기 (Win+Tab 대체)
  - 사용자 지정 앱 실행
  - 사운드 출력 장치 전환
  - 트레이 메뉴 팝업
  - 아무 동작 없음 (비활성화)
- **딜레이 설정**: 모서리 진입 후 N ms 대기 (0~1000ms, 실수 방지)
- **제외 앱 목록**: 특정 앱 전체화면 시 자동 비활성화 (게임, 프레젠테이션)
- 멀티모니터 지원 (각 모니터의 코너 개별 설정)
- 트레이 상주, ON/OFF 빠른 토글
- 코너 감지 영역 크기 조절 (1~10px)

**수익 모델**
- 무료: 4코너 기본 동작 (잠금·바탕화면·작업보기·앱 실행)
- Pro $2.99 일회성: 딜레이 설정, 앱별 자동 비활성화, 멀티모니터 독립 설정, 커스텀 스크립트 실행
- **타겟**: macOS 경험 있는 Windows 사용자, PowerToys 파워유저
- **차별화**: PowerToys에도 없는 기능. 핫 코너 전용 도구 중 가장 현대적 UI

**경쟁자 분석 (에이전트 검증)**
| 경쟁자 | 가격 | 약점 |
|--------|------|------|
| WinXCorners | 무료 | 동작 종류 4~5개뿐, 설정 UI 빈약 |
| PowerToys | 무료 | 핫 코너 기능 없음 |
| Cursor Commander | 유료 | 마우스 제스처 중심, 핫 코너 아님 |

**구현 난이도** ★★☆☆☆

### 6. `Net.Speed` — 실시간 네트워크 속도 트레이 오버레이

**문제 정의**
게임 중, 화상회의 중, 대용량 파일 다운로드 중 네트워크 속도를 실시간으로 확인하고 싶다.
Windows 작업 표시줄 → 작업 관리자 → 성능 탭 3단계가 필요하고, 트레이에 항상 보이는 도구가 없다.
기존 NetSpeedMonitor는 2012년 이후 업데이트 없어 Windows 11에서 불안정하다.

**킬러 기능**
- 트레이 아이콘에 실시간 다운로드↓ / 업로드↑ 수치 직접 렌더링 (KB/s · MB/s 자동 단위)
- 우클릭 메뉴: 최근 5분 미니 그래프 팝업
- 연결 품질 점수: 패킷 손실 · 지연(ping) 실시간 측정

### G-4. `Color.Blitz` — 스트룹 효과 기반 색상 초고속 매칭

**컨셉**
화면에 **"빨강"** 이라는 글자가 **파란색**으로 표시된다.
라운드마다 "글자 색을 선택하라" 또는 "글자 의미를 선택하라" 규칙이 바뀐다.
뇌가 자동으로 글자를 읽어버리는 인지 충돌을 이용한 반응속도 게임.

**게임플레이 루프**
```
화면 위: 현재 규칙 표시 ("글자 색을 고르세요!")
화면 중앙: 충돌하는 색상·의미의 텍스트 표시
화면 하단: 4개 컬러 버튼
→ 0.8초 안에 선택 → 맞으면 다음 라운드, 틀리면 HP 감소
→ 속도가 점점 빨라짐 (0.8초 → 0.5초 → 0.3초)
→ 규칙이 갑자기 반전 → 혼란 극대화
```

**핵심 메커닉**
- 3가지 게임 모드:
  - Classic: 글자 색 vs 의미 고정 규칙
  - Chaos: 라운드마다 규칙 랜덤 전환
  - Speed: 0.3초 목표, 반응속도 전용
- 다국어 확장: 한글로 색상 이름 표시 → 영어로 바뀜 (언어 충돌)
- 결과: 평균 반응 시간(ms), 정확도, 스트룹 점수
- 두뇌 훈련 앱으로 마케팅 가능 ("인지 유연성 향상")

**취향 연관** Hue.Flow(색상 퍼즐 최애) + Dodge.Blitz(빠른 반응)

**개발 난이도** ★☆☆☆☆ (텍스트 렌더링 + 타이머 + 점수 로직)

**에이전트 평가** 10/15 ★★★ (신박★★★★)

### G-5. `Neon.Slice` — 마우스 슬라이싱 아케이드 (Fruit Ninja PC 버전)

**컨셉**
화면에 네온 도형들이 다양한 각도로 튀어오른다. 마우스를 빠르게 그으면 도형이 잘린다.
검정 폭탄을 자르면 감점. 콤보가 쌓이면 슬로모션 피버 타임 발동.

**게임플레이 루프**
```
도형이 물리 기반으로 포물선을 그리며 솟아오름
→ 마우스 빠른 드래그 → 궤적이 도형과 교차하면 슬라이스
→ 잘린 도형이 두 조각으로 분리되며 파티클 폭발
→ 콤보 5회 → 슬로모션 1.5초 + 배율 2×
→ 놓친 도형이 3개 → 게임 오버
```

**핵심 메커닉**
- 마우스 궤적 벡터 계산 → 도형 폴리곤과 교차 판정
- 도형 분리 애니메이션 (2조각이 물리 기반으로 날아감)
- 특수 도형: 번개(모든 화면 클리어), 얼음(슬로모션 3초), 별(보너스 10점)
- 네온 사이버펑크 비주얼 (배경: 도시 야경, 도형: 발광 네온)
- 클래식 / 타임어택 / 선불제(N개 슬라이스 내 최대 콤보) 모드

**취향 연관** Dodge.Blitz/Star.Strike(아케이드) + 시각적 만족감(Word.Cloud, Char.Art)
Fruit Ninja PC 버전 공백 — Steam 버전(2016)이 저평가 → 기회

**개발 난이도** ★★★☆☆
마우스 궤적 벡터(지난 5프레임 평균) + 폴리곤 교차 판정 + 물리 기반 파티클
