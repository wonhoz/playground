# 주식 단타 시그널 엔진 개발 참고서

## 목적

실시간 KIS API 분봉 데이터를 기반으로 바닥 반등, 추세 시작, 고점 경고를
탐지하기 위한 지표 정리.

## 현재 사용 중

-   Time Window
-   Bollinger Bands
-   RSI
-   거래량
-   이동평균선(MA)

------------------------------------------------------------------------

# 1. VWAP (★★★★★)

**의미**: 거래량 가중 평균가격. 기관 평균 매수단가에 가까운 개념.

### 활용

-   BB 하단 터치
-   RSI \< 30
-   VWAP 상향 돌파
-   거래량 증가

→ 강한 반등 후보

------------------------------------------------------------------------

# 2. OBV (★★★★☆)

가격보다 거래량 흐름을 먼저 보여주는 경우가 많다.

### 활용

-   가격 횡보
-   OBV 상승

→ 매집 가능성

------------------------------------------------------------------------

# 3. MFI (★★★★☆)

가격 + 거래량 기반 RSI.

### 활용

-   MFI \< 20 : 과매도
-   MFI \> 80 : 과매수

------------------------------------------------------------------------

# 4. ATR (★★★★★)

변동성 측정.

### 활용

-   ATR 증가 → 추세 시작 가능성
-   ATR 감소 → 박스권

------------------------------------------------------------------------

# 5. ADX (★★★★★)

추세 강도.

  ADX       의미
  --------- -----------
  \<20      횡보
  25 이상   추세 시작
  40 이상   강한 추세

------------------------------------------------------------------------

# 6. MACD Histogram

MACD보다 선행성이 높은 경우가 많다.

-   음수 감소 → 상승 에너지 증가
-   양수 감소 → 상승 둔화

------------------------------------------------------------------------

# 7. Stochastic RSI

초단타에 적합.

-   20 이하 골든크로스
-   80 이상 데드크로스

------------------------------------------------------------------------

# 8. CCI

-100 돌파 시 반등 신호 참고.

------------------------------------------------------------------------

# 9. Donchian Channel

20봉 최고/최저 돌파 전략.

------------------------------------------------------------------------

# 10. 거래량 Z-Score (★★★★★)

    (CurrentVolume - MeanVolume)
    / StdDev

3 이상이면 비정상 거래량.

------------------------------------------------------------------------

# 11. RVOL (★★★★★)

    현재 동일 시간 거래량
    /
    최근 N일 동일 시간 평균

2 이상이면 강한 관심.

------------------------------------------------------------------------

# 12. EMA Ribbon

EMA 5,8,13,21,34,55

정배열 + 기울기 증가 확인.

------------------------------------------------------------------------

# 13. SuperTrend

추세 지속 판단.

------------------------------------------------------------------------

# 14. Parabolic SAR

추세 전환 보조.

------------------------------------------------------------------------

# 15. Divergence (★★★★★)

## Bullish

가격 신저가 RSI/MACD 신저가 실패

## Bearish

가격 신고가 RSI/MACD 신고가 실패

------------------------------------------------------------------------

# 16. 캔들 패턴

-   Hammer
-   Bullish Engulfing
-   Morning Star
-   Three White Soldiers
-   Marubozu

------------------------------------------------------------------------

# 17. EMA Slope

EMA 기울기 계산.

양수 + 증가 → 추세 강화

------------------------------------------------------------------------

# 18. Multi Timeframe

예시

-   1분 : 반등
-   5분 : MACD 상승
-   15분 : VWAP 위
-   30분 : EMA 정배열

동시 만족 시 신뢰도 상승.

------------------------------------------------------------------------
