- **날짜**: 2026-03-06
- **태그**: feature
- **담당**: Claude (claude-sonnet-4-6)

---

## 개요

개발자용 i18n 국제화 파일 시각적 관리 도구 `Locale.Forge`를 신규 개발함.
JSON / YAML / RESX / PO / Properties 파일을 나란히 비교하고, 누락 키 및 미사용 키를 탐지하는 WPF 앱.

---

## 구현 내용

### 프로젝트 구조
```

├── Locale.Forge.csproj         (YamlDotNet 16.3.0)
├── App.xaml / App.xaml.cs
├── MainWindow.xaml / .cs       (메인 UI + 동적 컬럼 DataGrid)
```

### 핵심 기능
1. **파일 로드**: 드래그&드롭 + 파일 열기 버튼 (다중 선택)
2. **언어 자동 감지**: 파일명에서 `en`, `ko`, `zh-CN` 등 추출
3. **동적 컬럼**: 언어 파일 추가 시 DataGrid 컬럼 자동 생성
4. **누락 키 강조**: A 언어에 있고 B 없는 키 → 행 빨간색 배경
5. **미사용 키 탐지**: 소스 코드 폴더 스캔 (`OpenFolderDialog`) → 미참조 키 황금색 배경
6. **인라인 편집**: DataGrid 직접 편집 + 하단 패널 멀티 TextBox
7. **형식 변환 내보내기**: JSON↔YAML↔RESX↔PO↔Properties 변환
8. **키 추가/삭제**: 모든 언어에 동시 반영
9. **검색 + 필터**: 키/값 검색, 누락만/미사용만 필터
10. **모두 저장**: 원본 파일 형식 그대로 저장 (UTF-8 BOM)

### 기술적 특이사항
- `System.Windows.Forms` 충돌 회피: WinForms 의존 제거, `Microsoft.Win32.OpenFolderDialog` 사용
- WPF `Spacing` 속성 미지원 → 자식 요소 `Margin`으로 대체
- 파서에 `using System.IO;` 명시 필요 (WinForms 공존 시 `File` 모호성)
- 다크 타이틀바: `DwmSetWindowAttribute(hwnd, 20, ref 1, sizeof(int))`
