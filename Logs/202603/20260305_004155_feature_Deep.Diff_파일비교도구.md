# Deep.Diff — 파일 비교 도구 개발

**날짜**: 2026-03-05
**태그**: feature
**프로젝트**: Applications/Files/Deep.Diff

---

## 개요

Beyond Compare를 참고하여 개발한 고기능 파일/폴더 비교 도구.
개발자를 위한 5가지 비교 모드를 지원하며, Beyond Compare를 넘어서는 기능도 포함.

---

## 구현된 기능

### 비교 모드 (5가지)
1. **폴더 비교** — 두 폴더의 파일/디렉터리 구조 비교
   - 동일(=) / 차이(≠) / 좌만(←) / 우만(→) 상태 표시
   - 필터: All / 차이만 / 동일만 / 좌만 / 우만
   - 파일 작업: 복사(←→), 이동(←→), 삭제
   - 하위 폴더 재귀 비교 옵션
   - 파일 필터 패턴 지원 (*.cs;*.txt)
   - 더블클릭으로 폴더 진입 / 텍스트 비교 열기
   - 컬럼 정렬 (이름/크기/날짜)

2. **텍스트 비교** — Myers diff 알고리즘 기반 라인 비교
   - 문자(Character) 수준 인라인 하이라이트
   - 다음/이전 차이 네비게이션 (F3/Shift+F3)
   - 좌↔우 복사, 저장 (Ctrl+S)
   - 공백 무시 / 대소문자 무시 옵션
   - 폰트 크기 조절, 줄 바꿈 모드
   - 동기화 스크롤

3. **이미지 비교** — 픽셀 단위 비교
   - 나란히 / 오버레이 / 픽셀 Diff 모드
   - 줌 인/아웃/맞춤/1:1
   - 차이 픽셀 수 및 비율(%) 표시

4. **HEX 비교** — 바이너리 파일 바이트 단위 비교
   - HEX + ASCII 동시 표시
   - 차이 행 하이라이트
   - 동기화 스크롤
   - 최대 1MB (65536행) 표시

5. **클립보드 비교** — 파일 없이 즉시 비교
   - 직접 입력 또는 클립보드 붙여넣기
   - 300ms 디바운스 실시간 업데이트
   - 다음/이전 차이 네비게이션

### Beyond Compare 대비 개선점
- **탭 기반 멀티 세션** — 여러 비교를 탭으로 동시에 유지
- **Myers diff** — O(ND) 최적 알고리즘 직접 구현 (외부 라이브러리 없음)
- **문자 수준 인라인 하이라이트** — 변경된 라인 내 정확히 어느 문자가 바뀌었는지 표시
- **클립보드 비교** — 파일 없이 붙여넣기만으로 즉시 비교
- **무의존성** — NuGet 패키지 없이 순수 .NET 구현
