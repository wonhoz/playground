# Net.Scan — LAN 네트워크 기기 스캐너 구현

**날짜**: 2026-03-06
**태그**: feature
**경로**: `Applications/Tools/Network/Net.Scan/`

---

## 개요

LAN 네트워크 기기 스캐너 WPF 앱 신규 개발.
외부 패키지 없이 순수 .NET API만으로 구현.

---

## 구현 내용

### 프로젝트 구조

```
Net.Scan/
├── Net.Scan.csproj          # net10.0-windows, WPF, 외부 패키지 없음
├── GlobalUsings.cs          # System.Net.*, System.Diagnostics, System.Collections.Concurrent 등
├── App.xaml / App.xaml.cs
├── MainWindow.xaml / .cs    # DwmSetWindowAttribute 다크 타이틀바
├── Models/
│   └── NetworkDevice.cs     # INotifyPropertyChanged, PingHistory Queue
├── Services/
│   ├── OuiService.cs        # 200+ OUI 벤더 Dictionary + GetDeviceIcon()
│   ├── NetworkService.cs    # 어댑터 조회, IP 범위 열거, arp -a 파싱, DNS 역조회
│   ├── ScanService.cs       # 병렬 Ping (SemaphoreSlim 50) + ARP + OUI + DNS 통합
│   ├── PortScanService.cs   # TcpClient 병렬 포트 스캔 (26개 공통 포트)
│   └── PingMonitor.cs       # System.Timers.Timer 주기적 핑 (5초, SemaphoreSlim 30)
└── Views/
    ├── ScanView.xaml        # 툴바 + GridView ListView + GridSplitter + 상세 패널
    └── ScanView.xaml.cs     # 스캔 이벤트 처리, 포트 스캔, 핑 그래프 렌더링
```

### 핵심 기술

| 항목 | 구현 방식 |
|------|-----------|
| 스캔 | Parallel `Ping.SendPingAsync()` + SemaphoreSlim(50) |
| ARP | `Process("arp", "-a")` 출력 Regex 파싱 (admin 불필요) |
| OUI | 200+ 항목 하드코딩 Dictionary (외부 DB 없음) |
| DNS | `Dns.GetHostEntryAsync()` 2초 타임아웃 |
| 포트 스캔 | `TcpClient.ConnectAsync()` + SemaphoreSlim(20) |
| 핑 모니터 | `System.Timers.Timer` 5초 간격, 최대 40샘플 Queue |
| 핑 그래프 | WPF Canvas + Polyline + Polygon (채우기 영역) |

### 테마

- 배경: `#111118` / `#0D0D16`
- 강조: `#06B6D4` (시안)
- 버튼: AccentButton (검정 텍스트)
- 다크 타이틀바: `DwmSetWindowAttribute`
