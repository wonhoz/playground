# PDF.Forge 초기 구현

- **날짜**: 2026-03-05
- **태그**: feature
- **프로젝트**: Applications/Files/PDF.Forge

## 작업 내용

### 프로젝트 구조 생성
- `PDF.Forge.csproj` — net10.0-windows, WPF + WinForms, PdfSharpCore 1.3.65, PDFtoImage 4.1.0
- `GlobalUsings.cs` — System.IO, WPF 네임스페이스
- `App.xaml` — 완전한 다크 테마 (Button, TextBox, ComboBox, CheckBox, ScrollBar, ListBox, Slider, ProgressBar 등 ControlTemplate)
- `MainWindow.xaml` — 사이드바(160px) + 콘텐츠 영역 레이아웃
- `MainWindow.xaml.cs` — DwmSetWindowAttribute 다크 타이틀바, Navigate 라우팅

### 서비스 레이어
- `Services/PdfMergeService.cs` — 다중 PDF 병합 (PdfSharpCore Import 모드)
- `Services/PdfSplitService.cs` — 범위별/페이지별 분리
- `Services/PdfPageService.cs` — 회전, 삭제, 재정렬, 페이지 수
- `Services/PdfCompressService.cs` — CompressContentStreams 재저장
- `Services/PdfWatermarkService.cs` — 텍스트/이미지 워터마크 (XGraphics)
- `Services/PdfPreviewService.cs` — PDFtoImage → SKBitmap → WriteableBitmap 변환

### 뷰 UI (5개)
- `Views/MergeView` — 드래그&드롭 파일 목록, 순서 조정, 병합 실행
- `Views/SplitView` — 페이지별/범위 지정 분리, 범위 목록 관리
- `Views/PageView` — 회전/삭제/순서 변경, ObservableCollection 기반
- `Views/CompressView` — 원본/결과 크기 표시, 압축률 계산
- `Views/WatermarkView` — 텍스트(폰트·색상·투명도·회전) / 이미지(크기 비율) 워터마크

### 아이콘
- `Resources/app.ico` — 다크 배경에 흰색 PDF 페이지 + 파란 접힌 모서리 + "PDF" 텍스트 (16/32/48/256px)

### 빌드 이슈 해결
- `LetterSpacing` WPF에 없음 → `<Run>` 태그로 대체
- `UseWindowsForms` + `UseWPF` 혼용 시 네임스페이스 충돌 → `<Using Remove>` 로 WinForms implicit usings 억제, SplitView에서 `using SWF = System.Windows.Forms` alias 사용
- `PDFtoImage.RenderOptions` vs `System.Windows.Media.RenderOptions` 충돌 → `using PdfRenderOptions = PDFtoImage.RenderOptions` alias 추가
