## 게임 소개

초기 자본 **1,000만 원**으로 시작하는 가상 주식 트레이딩.
하루(09:00~15:30)가 실제 4분으로 압축되어 흘러가며, 실시간 뉴스가 주가를 출렁이게 한다.

- **20개 가상 종목 / 10개 섹터** — 반도체·바이오·2차전지·게임·금융·화학·중공업·엔터·유통·통신
- **실시간 가격 엔진** — 랜덤워크 + 시장 무드 + 섹터 상관 + 뉴스 충격 (틱 100ms)
- **한국 시장 규칙** — 상하한가 ±30%, KRX 호가단위, VI(변동성완화장치) 발동·거래정지
- **실거래 비용** — 수수료 0.015% + 매도 거래세 0.2%, 시장가 슬리피지
- **뉴스 엔진** — 종목/섹터/시장 뉴스 + ⚡속보(즉시 갭) + 페이크 뉴스('소문에 사서 뉴스에 팔아라')
- **HTS 스타일 UI** — 관심종목 시세표 · 5분봉 캔들차트(평단가 라인) · 10단계 호가창 · 시장가/지정가 주문 · 잔고/주문/체결 탭
- **일일 사이클** — 장 마감 정산 → 오버나이트 갭 → 다음 날 개장
- **기록 저장** — 최고 총자산·수익률·누적 거래일 (`%LocalAppData%\Playground\Stock.Rush`)

## 튜토리얼 (4종)

| 시나리오 | 내용 |
|----------|------|
| ① 기초 매매 | 시장가 매수 → +2% 수익 실현 매도, 거래 비용 체감 |
| ② 뉴스 트레이딩 | 호재 속보에 빠르게 올라타고, 차익 매물에 꺾이기 전 탈출 |
| ③ 손절 연습 | 악재 급락 시 손실 -5% 이내로 끊어내는 훈련 |
| ④ 캔들·추세 읽기 | 양봉/음봉·꼬리 읽기, 망치형(Hammer) 반전 매수, 추세 추종 보유, '떨어지는 칼날' 회피 — 실제 캔들스틱 기술적 분석 기본기 |

## 조작

- **종목 선택**: 관심종목/보유종목 클릭
- **주문**: 매수/매도 + 시장가/지정가 토글, 수량 10%/25%/50%/최대 버튼
- **지정가 빠른 입력**: 호가창 가격 클릭
- **배속**: 상단 1×/2×/4× 토글 (지루한 구간은 빠르게, 긴박한 구간은 1×로)
- **알림음**: 상단 🔔 토글 (체결·속보·VI)
