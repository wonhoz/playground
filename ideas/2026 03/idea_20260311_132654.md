## Applications

### 1. Note.File — 파일·폴더 스티커 메모 첨부 도구

**카테고리**: Files / Productivity
**핵심 기능**: Windows 탐색기에서 파일이나 폴더에 직접 텍스트 메모를 붙여두는 도구. 메모가 있는 파일은 아이콘 배지로 표시.

**상세 기능**:
- 파일/폴더 우클릭 컨텍스트 메뉴 → "메모 추가/편집"
- 메모가 있는 항목: 탐색기 아이콘 우하단에 ✎ 오버레이 배지 표시 (쉘 아이콘 오버레이)
- 메모 데이터: SQLite DB에 파일 경로+해시 기준 저장 (파일 이동 시 해시로 재추적)
- 빠른 뷰: 마우스 호버 → 툴팁으로 메모 미리보기
- 전용 관리창: 메모 전체 목록, 검색, 최근 수정 정렬, 깨진 경로 감지
- 태그 지원: `#urgent`, `#review`, `#done` 태그 + 태그별 필터
- 내보내기: 파일 목록 + 메모 내용 Markdown 리포트

**기술 스택**:
- `IShellIconOverlayIdentifier` COM 인터페이스 (아이콘 배지)
- `IContextMenu` 셸 익스텐션 (우클릭 메뉴)
- Microsoft.Data.Sqlite (메모 저장)

**수익 모델**: 파일 정리가 많은 연구자·개발자·콘텐츠 제작자에게 유료 프리미엄 (다중 컬러 스티커)
**차별화**: 별도 앱 창 열 필요 없이 탐색기 자체에서 메모 확인 — 진정한 인라인 파일 메모

### 2. Dir.Cast — 폴더 구조 트리 시각화 & 내보내기

**카테고리**: Dev / Files
**핵심 기능**: 폴더를 드래그하면 내부 구조를 트리 다이어그램으로 시각화하고 여러 형식으로 내보내는 도구

**상세 기능**:
- 폴더 드래그&드롭 → 즉시 트리 렌더링
- 출력 형식:
  - ASCII Tree (터미널 스타일, `├── / └──`)
  - Markdown 리스트 (들여쓰기 체크리스트)
  - JSON 계층 구조
  - SVG 트리 다이어그램 (좌우/상하 레이아웃)
  - HTML 인터랙티브 접기/펼치기
- 필터: `.gitignore` 패턴 적용, 확장자 포함/제외, 최대 깊이 설정
- 파일 통계 오버레이: 폴더별 총 크기·파일 수·최근 수정일 표시
- 변경 감지: 이전 스냅샷과 비교 → 추가/삭제/이동 파일 색상 강조
- 클립보드 즉시 복사 (README 작성에 바로 붙여넣기)

**기술 스택**:
- 재귀 `DirectoryInfo` 탐색
- SVG 트리 레이아웃: Reingold-Tilford 알고리즘
- JSON 직렬화 → HTML 템플릿 엔진

**수익 모델**: 오픈소스 README 작성·문서화 자동화 도구 → GitHub Stars → 기부
**차별화**: Disk.Lens는 용량 시각화, Dir.Cast는 구조 문서화 + 변경 비교

- 주석 활성 상태에선 `WS_EX_TRANSPARENT` 제거 → 마우스 입력 수신

**수익 모델**: 강사·발표자·유튜버 필수 도구 → 프리미엄 (레이아웃 저장, 다중 슬라이드)
**차별화**: Cursor.Lens는 스포트라이트+줌(프레젠테이션 강조), Screen.Annotate는 직접 그리기 주석(강의/설명)

### 5. Path.Guard — PATH 환경변수 전용 시각적 관리자

**카테고리**: Dev / System
**핵심 기능**: Windows PATH 환경변수의 항목을 드래그로 순서 변경하고 중복·깨진 경로를 즉시 진단하는 전용 도구

**상세 기능**:
- 현재 PATH 목록 시각화: 시스템 PATH + 사용자 PATH 분리 색상 표시
- 드래그&드롭으로 순서 재배치 (실행 파일 검색 우선순위 직접 제어)
- 진단:
  - 빨강: 존재하지 않는 경로 (깨진 경로)
  - 노랑: 중복 항목
  - 회색: 비활성화된 항목
  - 초록: 정상
- 환경변수 확장: `%USERPROFILE%` → 실제 경로 미리보기
- 어떤 PATH에 어느 실행파일이 있나 확인: 파일명 검색 → 히트된 경로 강조
- 버전 충돌 감지: `python.exe`가 여러 경로에 있으면 경고
- 변경사항 즉시 적용 (`SendMessageTimeout HWND_BROADCAST WM_SETTINGCHANGE`)
- 스냅샷 저장/롤백

**기술 스택**:
- `Environment.GetEnvironmentVariable("PATH", EnvironmentVariableTarget.Machine/User)`
- `SendMessageTimeout` + `WM_SETTINGCHANGE`
- WPF ListBox 드래그 재정렬

**차별화**: Env.Guard는 모든 환경변수 관리기, Path.Guard는 PATH 전용 심층 진단+시각화+버전 충돌 탐지

### 7. Crash.View — Windows 덤프 파일 분석 뷰어

**카테고리**: Dev / Debugging
**핵심 기능**: Windows 미니덤프(.dmp) 파일을 드래그하면 콜스택·로드된 모듈·예외 정보를 GUI로 표시하는 WinDbg 경량 대체

**상세 기능**:
- `.dmp` / `.mdmp` 파일 드래그&드롭 → 즉시 분석
- 표시 정보:
  - 예외 타입·코드·주소 (SEH, .NET Exception)
  - 콜스택 프레임 (모듈명·함수명·오프셋)
  - 로드된 모듈 목록 (버전·경로·서명 여부)
  - 스레드 목록 (ID·상태·스택 각각)
  - 힙 정보 (메모리 점유, 비정상 블록 감지)
- PDB 심볼 로딩: 로컬 PDB 파일 + Microsoft Symbol Server 자동 다운로드
- .NET 전용 분석: ClrMD 라이브러리로 관리 스택 역추적
- 리포트 내보내기: Markdown / HTML

**기술 스택**:
- `Microsoft.Diagnostics.Runtime` (ClrMD) — .NET 힙·스택 분석
- `DbgHelp.dll` P/Invoke (StackWalk64, SymInitialize) — 네이티브 심볼 해석
- `System.Diagnostics.SymbolStore`

**수익 모델**: 개발자 필수 도구, WinDbg 없이 빠른 크래시 원인 파악
**차별화**: WinDbg는 복잡한 명령어 CLI, Crash.View는 드래그&드롭 GUI 분석

### 8. Log.Merge — 다중 소스 로그 통합 실시간 뷰어

**카테고리**: Dev / Operations
**핵심 기능**: 여러 앱·서비스의 로그 파일을 하나의 타임라인으로 통합하고 코릴레이션 ID로 요청 추적

**상세 기능**:
- 다중 로그 파일/폴더 동시 모니터링 (각각 색상 레이블)
- 타임스탬프 파싱 자동 감지 (ISO 8601, `[HH:mm:ss]`, `yyyy-MM-dd` 등 50+ 포맷)
- 통합 타임라인: 모든 소스를 시간순으로 머지 (인터리브)
- 코릴레이션 ID 추적: UUID/TraceId 클릭 → 해당 ID가 포함된 모든 로그 줄 하이라이트
- 레벨별 필터: ERROR/WARN/INFO/DEBUG 토글
- 정규식 필터: 실시간 필터링
- 시간 구간 줌: 타임라인 슬라이더로 특정 시간 구간 집중 뷰
- 로그 소스별 오류 발생 빈도 히트맵 (분 단위)

**기술 스택**:
- `FileSystemWatcher` (다중 파일 실시간 감시)
- 타임스탬프 정규식 파서 (50+ 포맷 딕셔너리)
- VirtualizingStackPanel 가상 스크롤 (수백만 줄 대응)

**차별화**: Log.Lens는 단일 파일 뷰어, Log.Merge는 **마이크로서비스·멀티 컴포넌트** 로그 통합 분석

**수익 모델**: 파워유저·IT 관리자 도구 (배치 배포용 바로가기 생성 자동화)

### 10. Mem.Lens — 프로세스 메모리 심층 분석기

**카테고리**: Dev / System
**핵심 기능**: 실행 중인 프로세스의 메모리를 Working Set·Private·Shared·Heap으로 분류하고 메모리 누수 패턴을 시각화

**상세 기능**:
- 프로세스 목록: 메모리 사용량 실시간 랭킹 (Private Bytes 기준)
- 상세 분석 창:
  - 메모리 맵: 가상 주소 공간 시각화 (커밋/예약/빈 영역)
  - 힙 분석: 힙 블록 크기 분포 히스토그램
  - 스택 vs 힙 비율
  - DLL/모듈별 메모리 점유량
- 메모리 누수 감지: 5분간 샘플링 후 지속 증가 추세 감지 → 경고
- 타임라인: 선택 프로세스 메모리 사용량 시간 그래프 (30분)
- .NET 관리 힙 상세: Gen0/Gen1/Gen2/LOH 크기 + GC 주기 표시
- 메모리 강제 해제: WorkingSet 트림 (SetProcessWorkingSetSize)
- 리포트: 현재 스냅샷 Markdown 내보내기

**기술 스택**:
- `Microsoft.Diagnostics.Runtime` (ClrMD) — .NET 힙
- `NtQueryVirtualMemory` P/Invoke — 가상 메모리 맵
- `PerformanceCounter` — 실시간 샘플링

**차별화**: Proc.Timeline은 CPU/메모리 시계열 레코더, Mem.Lens는 특정 순간 메모리 **구조 심층 해부**

### 1. Drift.Paint — 드리프트 타이어 자국으로 캔버스 채우기

**장르**: Arcade / Racing / Creative
**핵심 메커닉**: 탑다운 뷰 자동차가 드리프트할 때 남기는 타이어 자국이 컬러 페인트가 되어 캔버스 채움 % 달성

**게임플레이**:
- 탑다운 2D 자동차 물리 (미끄러짐·마찰·드리프트 각도)
- 드리프트 중 타이어에서 색상 잉크 분출 → 지면에 영구 흔적
- 목표: 제한 시간 내 캔버스 80%+ 채우기
- 색상 충돌: 다른 색 위에 겹치면 혼합 (빨강+파랑=보라)
- 여러 차량 색상 선택 + 스테이지마다 색상 혼합 목표 (최종 색 지정)
- 장애물: 빙판(드리프트 과대), 모래(저속), 지워버리는 물웅덩이
- 콤보 시스템: 드리프트 지속 시간 × 속도 = 잉크 분출량 배수
- 레벨 에디터: 캔버스 모양·장애물 배치 커스텀

**기술**: 타이어 흔적 Texture2D 실시간 픽셀 쓰기, 물리 기반 드리프트 (횡력 계산)
**매력**: Nitro.Drift(레이싱)와 달리 창의적 페인팅이 목표, 완성된 캔버스 PNG 자동 저장

### 2. Morph.Rush — 폼 변환 달리기

**장르**: Arcade / Platformer / Speed
**핵심 메커닉**: 원·삼각·사각형 세 가지 폼으로 실시간 변환하며 각 폼의 물리 특성을 이용해 장애물을 통과

**게임플레이**:
- 왼쪽에서 오른쪽으로 끊임없이 달리는 러너
- 폼 전환 (A키=원, S키=삼각, D키=사각):
  - **원**: 가속·탄성·곡면 통과 (좁은 파이프 통과 가능)
  - **삼각형**: 날카로운 끝으로 블록 관통·분리 (얼음 블록 쪼개기)
  - **사각형**: 무거워 낙하 가속·지면 쿵 충격파·플랫폼 안정 (화살 차단)
- 각 폼에만 적용되는 장애물:
  - 원형 구멍 → 원만 통과
  - 날카로운 벽 → 삼각만 관통
  - 무게 센서 → 사각만 활성화
- 연속 폼 전환으로 리듬 생성 → 콤보 배율
- 절차 생성 무한 레벨, 일일 시드 랭킹

**기술**: 충돌 체계 폼별 분기, 절차 생성 장애물 스케줄러
**매력**: 단순 조작인데 타이밍 + 패턴 인식이 필요한 리듬감, Neon.Slice 류의 빠른 판단력 요구

### 3. Flock.Guide — 새떼 조종 게이트 통과 퍼즐

**장르**: Puzzle / Strategy / Simulation
**핵심 메커닉**: Boids 알고리즘으로 움직이는 새떼를 "유인 비콘"과 "반발 비콘" 배치로 조종해 게이트를 통과시키는 전략 퍼즐

**게임플레이**:
- 20~100마리 새가 Boids 3규칙으로 자율 비행 (분리·정렬·응집)
- 플레이어가 배치하는 요소:
  - 🟡 유인 비콘: 새를 끌어당기는 중력원 (배치 코스트 있음)
  - 🔵 반발 비콘: 새를 밀어내는 장애물
  - 🟢 게이트: 특정 수 이상의 새가 통과해야 레벨 클리어
- 제한: 비콘 총 개수 제한 + 배치 불가 구역
- 고급 장애물: 강풍 구역, 포식자 AI (새가 도망치는 방향 이용)
- 퍼즐 모드: 50개 레벨 (최소 비콘으로 클리어)
- 샌드박스 모드: 자유 Boids 관찰 + 비콘 실험

**기술**: Boids 알고리즘 (벡터 분리·정렬·응집 가중치), 공간 해싱 최적화 (O(n) 근접 탐색)
**매력**: Orbit.Craft처럼 물리/알고리즘 관찰이 퍼즐로 이어지는 쾌감, 새떼 움직임 자체가 아름다움

### 4. Carbon.Chain — 낙하 원자 결합 테트리스

**장르**: Puzzle / Casual / Educational
**핵심 메커닉**: 원자 블록이 낙하하며 인접 원자와 원자가 규칙에 따라 자동 결합 → 안정 분자 완성 시 소거하는 테트리스 변형

**게임플레이**:
- 단일 원자가 아래로 낙하 (H, C, O, N, S — 원자가 1~4)
- 착지 시 인접 원자와 결합 (빈 결합 수 있으면 자동 연결)
- **안정 분자** 완성 시 즉시 소거 + 점수:
  - H₂O, CO₂, CH₄ (메탄) = 기본
  - C₂H₅OH (에탄올), C₆H₁₂O₆ (포도당) = 고점수 체인
- 불안정 라디칼은 점점 위험해짐 (타이머 후 폭발)
- 특수 원소: 불꽃(F=불활성, 결합 거부), 방사성(Ra=주변 원자 랜덤 변환)
- 분자 결합 힌트 시스템 (처음엔 안정 목표 분자 하이라이트)
- 교육 모드: 완성 시 분자 구조·이름·용도 팝업

**기술**: 그래프 BFS로 결합 연결 유효성 검사, 분자 안정성 판단 (원자가 충족 여부)
**매력**: Atom.Craft(정적 배치)와 달리 낙하 긴장감 + 화학 학습 요소 — 교육 B2B 잠재력

### 5. Weight.Cross — 균형추 물리 퍼즐

**장르**: Puzzle / Physics
**핵심 메커닉**: 저울 위에 무게 블록을 배치해 정확한 균형(0 토크)을 달성한 후 그 상태로 다리를 건너는 2단계 물리 퍼즐

**게임플레이**:
- **1단계: 저울 균형**
  - 가로 저울대(지렛대) 위에 무게 블록 드래그 배치
  - 블록 무게 × 지렛점 거리 = 토크 → 양쪽 균형 목표
  - 제한: 블록 종류와 수량 제한, 특수 블록(반사·자석·폭발)
  - 실시간 기울기 피드백
- **2단계: 다리 건너기**
  - 균형 잡힌 저울이 다리로 변함
  - 캐릭터가 건너는 동안 바람·충격파가 블록 흔들기
  - 균형이 ±5° 이상 기울면 실패
- 미니 퍼즐 모음 (60레벨): 물리 법칙 이해 심화
- 레벨 에디터: 무게 블록 세트 + 저울 모양 커스텀

**기술**: 지렛대 강체 물리 (토크 계산, 각가속도), Box2D 라이트
**매력**: 직관적 물리 원리(지렛대 법칙) + 긴장되는 "건너기" 2단계 — Orbit.Craft처럼 물리 사고가 쾌감

### 6. Strobe.Blitz — 음악 위상 생존 슈터

**장르**: Shooter / Rhythm / Arcade
**핵심 메커닉**: BGM 비트에 동기화된 화면 위상(Phase) 사이클 — 적은 "밝은 위상"에서만 실체화·공격 가능하고 "어두운 위상"에서는 투명·무적

**게임플레이**:
- BGM BPM 분석 → 2~4박자마다 "밝음 ↔ 어두움" 전환 (명도 펄스)
- 밝음 위상: 적 실체화 (피해 가능, 격파 가능), 플레이어 빠름
- 어두움 위상: 적 투명·무적, 플레이어 느림 + 충전
- 전략: 어두움에서 포지셔닝 → 밝음 순간에 연사
- 위상 예측: BGM 리듬 시각화 바(다음 전환 0.5초 전 예고 깜빡임)
- 특수 적:
  - 역위상적: 어두움에서만 실체, 밝음에 투명
  - 공명적: 특정 주파수 비트에서만 약점 노출
- 커스텀 음악 지원 (MP3/FLAC BPM 자동 분석)

**기술**: NAudio FFT BPM 탐지 + 비트 온셋 감지, 위상 타이머 동기화
**매력**: Dodge.Blitz의 빠른 회피 + 리듬게임 타이밍 = 독특한 하이브리드, 음악마다 다른 적 패턴

### 7. Laser.Blitz — 실시간 거울 배치 레이저 반사 슈터

**장르**: Shooter / Action / Puzzle
**핵심 메커닉**: 적들이 레이저를 발사하면 실시간으로 거울을 배치해 레이저를 역반사시켜 격파 — 정적 퍼즐(Laser.Net)과 달리 실시간 액션

**게임플레이**:
- 플레이어는 이동 불가, 4×4 격자 안에서 거울 배치만 가능
- 적이 순차로 레이저 발사 (각도·발사 위치 다양)
- 거울 배치: 드래그로 45°/135° 거울 놓기 (제한 수량)
- 레이저가 거울 역반사 → 발사한 적에게 피해
- 도전 요소:
  - 여러 적이 동시에 레이저 → 단일 거울로 복수 반사 필요
  - 빛 분리 프리즘: 하나를 둘로 나눔
  - 충전 레이저: 빠른 타이밍에 거울 배치해야만 반사 성공
  - 레이저 관통: 거울을 두 장 연속으로 맞춰야 경로 변경
- 적 처치 콤보: 연속 반사 격파 시 배율
- 50레벨 + 무한 서바이벌 모드

**기술**: 레이저 레이캐스팅 (반사 경로 계산), 실시간 거울 스냅 격자 배치
**매력**: Laser.Net(생각하며 배치)과 달리 빠른 반응 + 즉흥적 배치 — Dodge.Blitz류 긴장감 + 퍼즐 결합

### 8. Totem.Fall — 동물 고유 물리 토템 쌓기

**장르**: Arcade / Physics / Casual
**핵심 메커닉**: 동물마다 완전히 다른 물리 특성을 가진 토템 블록을 쌓아 높이를 유지하면서 바람·지진 이벤트를 버티는 균형 게임

**게임플레이**:
- 블록 대신 동물 토템 (낙하하며 쌓기):
  - 🐍 **뱀**: 착지 시 아래 블록을 감쌈 → 마찰 증가, 넘어지기 어려워짐
  - 🐸 **개구리**: 탄성 재질 → 쌓인 위 블록을 10% 튀어오르게 함
  - 🐘 **코끼리**: 매우 무거워 아래로 꽉 눌러줌 → 하단 안정, 상단 불안정
  - 🦅 **독수리**: 날개 펼쳐 바람 저항 내성 + 좌우로 천천히 이동하며 균형 조절
  - 🦔 **고슴도치**: 가시 때문에 위 블록과 마찰 최대화 (그러나 불규칙한 형태)
- 이벤트: 3초마다 무작위 (바람·지진·비·번개 특정 블록 타격)
- 목표: 최대한 높이 쌓기 (높이 = 점수)
- 로컬 2P 모드: 번갈아 블록 추가, 먼저 쓰러뜨리기

**기술**: Box2D 물리 엔진, 동물별 커스텀 마찰·탄성·질량 계수
**매력**: Wobble.Stack(소프트바디 젤리)과 달리 동물 캐릭터 + 전략적 배치 선택이 핵심

### 9. Sand.Castle — 모래 퇴적·침식 물리 건축 & 파도 생존

**장르**: Sandbox / Building / Survival
**핵심 메커닉**: 실제 모래 퇴적 물리(안식각, 수분에 따른 점착력)로 모래성을 쌓고, 밀려오는 파도를 버텨내는 건축 서바이벌

**게임플레이**:
- 모래 물리:
  - 안식각(Angle of Repose): 모래가 쌓이면 자연스럽게 사면 형성
  - 수분 레벨: 마른 모래(쉽게 무너짐) vs 젖은 모래(성형 가능, 강함)
  - 조수 사이클: 썰물(모래 마름) → 밀물(모래 젖음) → 파도
- 건축 도구: 손으로 모래 쌓기, 물 붓기, 삽 다지기
- 파도 물리: 사인파 + 난류 → 모래성에 침식력 가함
- 생존 목표: N번의 파도 사이클 버티기 (파도 점점 강해짐)
- 업그레이드: 나뭇가지 지지대, 조개 장식(+방어), 해자 파기
- 창작 모드: 무한 자유 건축 (파도 강도 직접 설정)

**기술**: 셀룰러 오토마타 모래 시뮬레이션 + 수분 레이어, 2D 파동 방정식 침식 계산
**차별화**: Sand.Fall은 물질 반응 관찰 샌드박스, Sand.Castle은 능동적 건축+방어 목표

### 10. Ghost.Type — 타이핑 블라인드 챌린지

**장르**: Puzzle / Skill / Casual
**핵심 메커닉**: 입력하는 글자가 화면에 표시되지 않고(또는 별표로 마스킹), 오타 발생 시에만 빨간 경고가 번쩍이는 극한 집중 타이핑 게임

**게임플레이**:
- 화면에 목표 문장 표시 (코드 스니펫, 명언, 이야기 단락)
- 타이핑 시 입력 영역에 `*****` 또는 완전 공백
- 오타 발생 시: 화면 빨간 플래시 + 삐 소리 (위치는 알려주지 않음)
- 정확도·WPM은 완료 후 공개
- 도전 모드:
  - 🔇 무음 모드: 오타 피드백도 없음 (완전 블라인드)
  - ⏱ 타임어택: 목표 WPM 이상 유지하며 오타 0개 달성
  - 🔁 반복: 오타 발생 시 문장 처음부터 재시작
  - 코드 챌린지: C#/Python/JS 코드 스니펫 정확 입력
- 연속 성공 스트릭 + 일일 챌린지 문장 (모두 같은 문장)
- 손가락별 오타 히트맵 표시 (완료 후)

**기술**: 키스트로크 타이밍 정밀 측정 (ms), 입력 마스킹 (TextBox Password char)
**매력**: 단순한데 극도로 집중력 테스트 — Echo.Drum(청각 기억)의 타이핑 버전, 매일 하고 싶어지는 자기계발 게임
