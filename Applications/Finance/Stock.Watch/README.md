# 📈 Stock.Watch

> 종목별 매수/매도 **조건을 미리 설정**해 두면, 조건이 충족되는 순간 **Slack으로 알림**을 보내는 WPF 데스크톱 앱.
> 캔들 차트 · 볼린저 밴드 · RSI · 거래량을 한눈에 보면서 조건을 자유롭게 조합한다.

**버전: v1.1.0** · .NET 10 / WPF · 데이터: 한국투자증권(KIS) Open API

---

## 핵심 기능

- **실시간 체결가(WebSocket)**: KIS 실시간 시세(`H0STCNT0`)를 구독해 장중 틱을 즉시 반영. 폴링(일봉·지표 기준선)과 병행하며, 끊기면 자동 재연결. 설정에서 끄면 폴링만으로 동작.
- **조건 기반 알림**: `RSI < 30`, `현재가 ≤ 볼린저하단`, `거래량 ≥ 거래량MA20 × 2`, `현재가 SMA20 상향돌파` 등을 드롭다운으로 조합. 매수/매도 룰셋 분리, AND/OR 결합.
- **차트 시각화**: 캔들(상승=빨강/하락=파랑) + 볼린저 밴드 + 거래량 막대 + RSI(14) 서브패널을 직접 렌더링(외부 차트 라이브러리 없음).
- **Slack 알림**: 조건 충족 시 종목명·현재가·조건·지표값을 Incoming Webhook으로 전송.
- **도배 방지**: 조건이 false→true로 **전이하는 순간 1회**만 알림 + 쿨다운(기본 600초).
- **확장성**: 종목코드 6자리 입력만으로 관심종목 추가. 기본 4종목 — SK하이닉스(000660)·SK스퀘어(402340)·보해양조(000890)·부국철강(026940).
- **장중 한정 감시**: 평일 09:00~15:30(KST)에만 폴링(옵션).

## 사전 준비 — API 키 발급 (사용자 직접)

> ⚠️ API 키는 이 PC의 `%LocalAppData%\Playground\Stock.Watch\config.json`에만 저장되며 **소스 저장소에 절대 포함되지 않습니다.**

### 1. 한국투자증권 KIS Open API
1. 한국투자증권 **실계좌 개설** (없으면 먼저 개설 — 모의투자도 계좌 필요)
2. [KIS Developers](https://apiportal.koreainvestment.com) 접속 → 신청 → **APP KEY** / **APP SECRET** 발급 (무료)
3. 처음엔 **모의투자**로 받으면 안전하게 시세 조회 테스트 가능 (설정에서 "모의투자 서버 사용" 체크)

### 2. Slack Incoming Webhook
1. Slack 앱 관리 → **Incoming Webhooks** 활성화
2. 알림 받을 채널 선택 → `https://hooks.slack.com/services/...` URL 복사

### 3. 앱에서 입력
앱 실행 → **⚙ 설정** → APP KEY / APP SECRET / Slack Webhook URL 입력 → 저장.
(설정 화면의 "테스트 메시지 전송"으로 Slack 연결을 먼저 확인할 수 있습니다.)

## 사용법

1. **⚙ 설정**에서 API 키·Slack URL·폴링 주기 입력
2. 좌측에서 종목 선택 → 우측에서 **매수/매도 조건** 편집 (조건 추가/삭제, AND·OR 선택)
3. 상단 **▶ 감시 시작** → 주기마다 시세를 조회하고 조건을 평가, 충족 시 Slack 알림 + 앱 내 알림 로그
4. **+ 추가**로 종목코드 6자리를 입력해 관심종목 확장

## 조건 구성 요소

| 구분 | 항목 |
|------|------|
| 피연산자 | 현재가, RSI, 볼린저(상단·중심·하단), 거래량, 거래량MA20, SMA(5·20·60), MACD·시그널 |
| 연산자 | `<` `≤` `>` `≥` · 상향돌파 · 하향돌파 |
| 비교 대상 | 상수값 또는 (지표 × 배수) — 예: 거래량 ≥ 거래량MA20 × 2 |

## 기술 메모

- KIS REST: `oauth2/tokenP`(토큰 24h 캐시) · `inquire-price`(현재가) · `inquire-daily-itemchartprice`(일봉 OHLCV)
- KIS 실시간: `oauth2/Approval`(approval_key) → WebSocket(`ws://ops.koreainvestment.com:21000`, 모의 31000) `H0STCNT0` 체결가 구독, PINGPONG 응답·자동 재연결
- 장중에는 당일 봉을 실시간 체결가/현재가로 갱신해 지표가 실시간 가격을 반영(틱 평가는 종목당 0.8초 스로틀)
- 외부 NuGet 의존 없음 — `HttpClient` + `ClientWebSocket` + WPF Shape 렌더링만 사용 (단일 파일 배포)
