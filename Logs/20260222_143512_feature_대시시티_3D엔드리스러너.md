- **일시**: 2026-02-22 14:30 ~ 14:35 (KST)

## 작업 내용

Subway Surfers / Temple Run 스타일의 WPF Viewport3D 기반 3D 엔드리스 러너 게임 개발.

### 구현 기능

| 기능 | 설명 |
|------|------|
| 3레인 시스템 | 좌(-1)/중앙(0)/우(1) 레인, ←→ 부드러운 전환 |
| 점프 | ↑/SPACE, 중력 물리 기반 포물선 점프 |
| 슬라이드 | ↓ 키, 0.6초간 납작해져 Beam 회피 |
| 장애물 3종 | Barrier (점프), Train (레인 회피), Beam (슬라이드) |
| 패턴 생성 | 6가지 장애물 배치 패턴 (단일/2레인/콤보/지그재그 등) |
| 코인 수집 | 3가지 코인 패턴 (직선/웨이브/상승), 회전 애니메이션 |
| 파워업 4종 | Magnet (자석), Shield (보호막), ×2 Score, Jetpack (비행) |
| 네온 시티 | 빌딩, 가로등, 네온 라인, 도로 마킹 자동 생성 |
| 속도 증가 | 거리에 비례하여 18→55 u/s 점진 증가 |
| 3D 캐릭터 | 몸통+머리 2파트, 슬라이드 시 납작해짐 |
| 카메라 | 3인칭 추적, 제트팩 시 줌아웃 |

### 장애물 패턴

| 패턴 | 설명 |
|------|------|
| 단일 트레인 | 1레인 차량 |
| 2레인 트레인 | 2개 레인 막힘, 1개 열림 |
| 바리케이드 | 낮은 장벽 (점프 회피) |
| 전체 빔 | 높은 빔 (슬라이드 회피) |
| 콤보 | 트레인 + 바리케이드 연속 |
| 지그재그 | 연속 트레인 다른 레인 |

### 파워업

| 파워업 | 효과 | 지속시간 |
|--------|------|----------|
| Magnet | 주변 코인 자동 수집 | 8초 |
| Shield | 장애물 1회 방어 | 10초 |
| ×2 Score | 점수 2배 | 10초 |
| Jetpack | 공중 비행 (무적) | 6초 |

### 아키텍처

```
Engine/
  GameLoop.cs            - 게임 루프
  WorldObject.cs         - 월드 오브젝트 (장애물/코인/파워업)
  WorldGenerator.cs      - 무한 월드 생성 (장애물/코인/파워업 스폰)
  MeshHelper.cs          - 3D 메시 유틸 (Box/Plane/Cylinder + Material)
MainWindow.xaml/.cs      - 3D 씬, 네온 시티 환경, 플레이어 물리, 충돌
```

### 조작법

| 키 | 동작 |
|----|------|
| ← → / A D | 레인 이동 |
| ↑ / SPACE / W | 점프 |
| ↓ / S | 슬라이드 |
| ENTER | 시작/재시작 |
| ESC | 종료/타이틀 |
