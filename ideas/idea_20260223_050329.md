## S급 — 즉시 착수 추천

### 1. `Meeting.Cost` — 실시간 회의 비용 계산기

**문제 정의**
회의에 참석하는 사람 수 × 연봉 × 시간 = 실제 비용인데, 대부분의 조직이 이를 인식하지 못함.
"이 회의가 얼마짜리인지"를 실시간으로 보여주면 회의 문화가 달라짐.

**킬러 기능**
- 참석자 수 + 평균 연봉(또는 시급) 입력
- 타이머 시작 → 실시간으로 누적 비용 표시 (원/달러)
- 토픽/결정사항 간단 메모
- 회의 종료 시 요약 클립보드 복사

**수익 모델**
- 무료 (바이럴) → Pro: 팀 히스토리 저장, 분석 리포트
- 기업용 라이선스 ($5~10/월)

**구현 난이도** ★★☆☆☆ (타이머 + 간단한 계산, UI 디자인이 핵심)

---

### 2. `Paste.Mask` — 화면 공유 시 민감정보 자동 마스킹

**문제 정의**
화상 회의/화면 공유 중 실수로 비밀번호, API 키, 개인정보가 노출되는 사고가 빈번.
사전에 지정한 앱/영역을 자동으로 블러/마스킹 처리해 공유 안전성 확보.

**킬러 기능**
- 화면 공유 감지 시 자동 활성화
- 규칙 기반 마스킹: 특정 앱 창, 좌표 영역, 키워드 패턴
- 오버레이 방식 (실제 화면은 정상, 공유 화면만 마스킹)
- 단축키로 즉시 ON/OFF

**수익 모델**
- 기본 무료 (3개 규칙) → Pro $4.99/월 (무제한 규칙, 팀 프리셋)
- 기업 보안팀 대상 B2B 판매

**구현 난이도** ★★★★☆ (Windows Graphics Capture API 필요, 기술적 도전)

### 3. `Time.Track` — 프리랜서용 프로젝트별 자동 업무시간 추적

**문제 정의**
프리랜서/컨설턴트가 프로젝트별 실제 작업 시간을 기록하지 않으면 청구서 근거가 없음.
수동 기록은 잊어버리기 쉬움 → 자동 추적이 핵심.

**킬러 기능**
- 활성 창 타이틀 기반 자동 분류 (규칙 설정)
- 유휴 감지 (마우스/키보드 없으면 자동 일시정지)
- 프로젝트별 누적 시간 + 시급 설정 → 예상 청구 금액
- CSV/PDF 청구서 내보내기

**수익 모델**
- 무료: 2개 프로젝트 / Pro $6.99/월: 무제한 + 청구서 자동화
- 팀 버전 ($14.99/월)

**구현 난이도** ★★★☆☆ (Windows Hook API + 타이머 + 간단한 리포트)

### 4. `Log.Lens` — 멀티 로그 파일 실시간 뷰어

**문제 정의**
개발자/운영자가 여러 서버 로그를 동시에 모니터링하려면 터미널을 여러 개 띄워야 함.
WSL/SSH 없이 Windows에서 `tail -f` + 필터링을 GUI로.

**킬러 기능**
- 복수 로그 파일 동시 탭 뷰
- 실시간 `tail -f` (파일 변경 감지)
- 키워드 하이라이트 (ERROR=빨강, WARN=노랑, INFO=초록)
- 정규식 필터링 + 검색
- 라인 번호, 타임스탬프 파싱 자동화

**수익 모델**
- 무료 (OSS) + Pro: SSH 원격 로그, 팀 공유 프리셋 ($7.99/월)
- 서버 모니터링 솔루션 업셀

**구현 난이도** ★★☆☆☆ (FileSystemWatcher + RichTextBox 렌더링)

## A급 — 완성도 높이면 히트 가능

### 5. `Env.Guard` — Windows 환경변수 GUI 관리자

**문제 정의**
Windows 기본 환경변수 편집기는 2000년대 UI, 실수로 PATH를 망가뜨리면 시스템이 작동 불능.
개발자라면 누구나 한 번쯤 겪는 문제 → 안전하고 편리한 GUI가 절실.

**킬러 기능**
- 환경변수 전체 시각화 (시스템/사용자 분리)
- 변경 전 자동 스냅샷 + 원클릭 롤백
- PATH 항목별 존재 여부 검증 (경로 유효성 표시)
- 변수 세트 프로필 저장/로드 (개발 환경 A ↔ B 전환)
- 변경사항 diff 미리보기

**수익 모델**
- 무료 기본 / Pro $3.99/월: 프로필 클라우드 백업, 팀 공유

**구현 난이도** ★★★☆☆ (레지스트리 접근 + 권한 처리)

### 6. `DNS.Flip` — DNS 프리셋 원클릭 전환 트레이 앱

**문제 정의**
VPN 사용, 지역 차단 우회, 보안 DNS 등 상황에 따라 DNS를 자주 바꾸는 사람들.
매번 네트워크 설정을 열어서 바꾸는 게 너무 불편.

**킬러 기능**
- 트레이에서 우클릭 → 프리셋 선택 (1.1.1.1, 8.8.8.8, 회사 DNS 등)
- NIC별 적용 (와이파이/이더넷 독립 관리)
- 응답 속도 Ping 표시
- 변경 로그

**수익 모델**
- 무료 (트레이 앱, 입소문) / Pro: 시간대별 자동 전환 스케줄

**구현 난이도** ★★★☆☆ (관리자 권한 필요, netsh/WMI 활용)

---

### 7. `Cert.Watch` — SSL 인증서 만료 모니터

**문제 정의**
운영 중인 사이트의 SSL 인증서가 만료되면 사용자에게 보안 경고가 뜨고 신뢰도가 폭락.
Let's Encrypt 자동 갱신 실패 사례도 많음 → 만료 전 알림이 절실.

**킬러 기능**
- 도메인 목록 등록 → 만료일 자동 조회
- 30일/14일/7일/1일 전 알림 (Toast + 이메일)
- 인증서 발급기관, 체인 상태 표시
- 대시보드: 전체 도메인 만료 현황

**수익 모델**
- 무료: 5개 도메인 / Pro $4.99/월: 무제한 + Slack/Teams 알림
- 소규모 호스팅 업체 B2B

**구현 난이도** ★★☆☆☆ (TcpClient + SslStream 인증서 조회)

### 8. `Toast.Cast` — 건강 루틴 스마트 반복 알림

**문제 정의**
장시간 모니터 앞에 앉아 있으면 눈 휴식, 스트레칭, 물 마시기 등을 잊음.
기존 알람은 너무 단순 → 맥락 인식 + 인터랙티브 알림이 필요.

**킬러 기능**
- 루틴 카테고리: 눈 휴식(20-20-20 법칙), 물 마시기, 스트레칭, 자세 교정
- 알림 시 간단한 카운트다운 오버레이 (선택)
- 유휴 감지: 이미 쉬고 있으면 스킵
- 주간 달성률 통계

**수익 모델**
- 무료 / Pro $2.99/월: 커스텀 루틴, 건강 리포트

**구현 난이도** ★★☆☆☆ (타이머 + Toast 알림 + 통계)

## B급 — 틈새 시장 타깃

### 9. `Noise.Guard` — 마이크 실시간 노이즈 필터

**문제 정의**
재택근무 중 키보드 소리, 팬 소음, 에어컨 소리가 화상 회의에 그대로 전달됨.
NVIDIA RTX Voice처럼 GPU 없이도 동작하는 경량 노이즈 필터.

**킬러 기능**
- 배경 잡음 실시간 제거 (WebRTC 노이즈 억제 알고리즘)
- 가상 마이크로 출력 (다른 앱이 필터링된 오디오를 사용)
- 필터 강도 조절 슬라이더
- 실시간 파형 before/after 비교

**수익 모델**
- 무료 기본 / Pro $4.99/월: 고급 모델, 다중 마이크

**구현 난이도** ★★★★☆ (Virtual Audio Cable + DSP 처리)

---

### 10. `Link.Vault` — 완전 오프라인 북마크 관리자

**문제 정의**
브라우저 북마크는 브라우저에 종속적이고, 클라우드 북마크 서비스는 서비스가 중단되면 날아감.
**내 PC에만** 저장되는 영구적 읽기 목록 + 오프라인 캐시.

**킬러 기능**
- URL 저장 시 페이지 전체 오프라인 스냅샷 (Chromium 엔진)
- 태그 + 전문 검색 (저장된 본문에서도 검색)
- 브라우저 확장 연동 (원클릭 저장)
- 읽음/미읽음 상태, 별점

**수익 모델**
- 무료: 100개 / Pro $3.99/월: 무제한 + 디바이스 동기화

**구현 난이도** ★★★☆☆ (WebView2 스냅샷 + SQLite 전문 검색)
