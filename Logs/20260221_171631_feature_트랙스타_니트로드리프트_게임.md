## Track.Star - 올림픽 트랙 & 필드 스포츠

### 구현 기능

| 기능 | 설명 |
|------|------|
| 100m 단거리 | ←→ 번갈아 누르기 버튼매싱, 라이벌 3명 |
| 110m 허들 | ←→ 달리기 + SPACE 점프, 허들 충돌 감속 |
| 멀리뛰기 | ←→ 도움닫기 + SPACE 도약 타이밍, 3회 시도 |
| 창던지기 | ←→ 달리기 → SPACE 파워 게이지 → SPACE 각도, 3회 시도 |
| 공통 | 카운트다운, 라이벌 AI, 메달 시스템 (🥇🥈🥉) |
| UI | 종목명, 타이머, 파워/각도 게이지 바, 대회 최종 결과 |

### 조작법
| 키 | 동작 |
|----|------|
| ←→ | 달리기 (번갈아 빠르게) |
| SPACE | 점프/파워 확정/각도 확정 |
| ENTER | 대회 시작/재시작 |

### 아키텍처
```
Engine/GameLoop.cs      - 게임 루프
Events/
  SportEvent.cs         - 종목 베이스 (카운트다운, 페이즈 관리)
  Sprint100m.cs         - 100m 단거리
  Hurdles.cs            - 110m 허들
  LongJump.cs           - 멀리뛰기
  Javelin.cs            - 창던지기
```

## Nitro.Drift - 탑다운 서킷 레이싱

### 구현 기능

| 기능 | 설명 |
|------|------|
| 서킷 코스 | 웨이포인트 기반 불규칙 타원형 서킷, 아스팔트+센터라인 렌더링 |
| 차량 물리 | 가속/브레이크/조향/마찰, 속도 비례 회전 |
| 부스트 | SPACE 부스트 (최대속도 1.5배), 연료 소모/자동 충전 |
| AI 라이벌 | 3대 (RED/GRN/ORG), 스킬레벨별 웨이포인트 추적 AI |
| 레이스 | 3바퀴 완주, 랩/포지션 실시간 추적 |
| UI | 속도계, 부스트 게이지, 랩, 포지션, 카운트다운, 결과 순위표 |

### 조작법
| 키 | 동작 |
|----|------|
| ↑ | 가속 |
| ↓ | 브레이크 |
| ←→ | 조향 |
| SPACE | 부스트 |
| P | 일시정지 |
| ENTER | 레이스 시작 |

### 아키텍처
```
Engine/
  GameLoop.cs           - 게임 루프
  Track.cs              - 서킷 트랙 (웨이포인트, 거리 계산)
Entities/
  Car.cs                - 차량 베이스 (물리, 부스트)
  AiCar.cs              - AI 라이벌 (웨이포인트 추적)
```
