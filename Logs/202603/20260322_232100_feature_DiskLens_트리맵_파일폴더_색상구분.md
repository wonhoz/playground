# Disk.Lens | 트리맵 파일/폴더 색상 구분

- **날짜**: 2026-03-22
- **태그**: feature
- **프로젝트**: Disk.Lens

## 작업 내용

트리맵에서 파일과 폴더를 색상으로 구분하는 기능 구현.

### 변경 사항

**`MainWindow.xaml.cs`**
- `DesaturateForFile(Color c)` 정적 메서드 추가
  - 원색을 채도 낮은 회색 계열로 변환 (factor = 0.30)
  - 공식: gray = 0.299R + 0.587G + 0.114B 가중 평균 후 혼합
- `RenderTreeMap()` 수정
  - 폴더: 기존 비비드 팔레트(`MapColors`) 그대로 (opacity 0.85)
  - 파일: 동일 색조에서 채도 낮춘 색상 적용 (opacity 0.75)
  - 파일 레이블 텍스트: White 대신 반투명 연회색(`#C8DCDCDC`) 적용

### 비포/애프터

| 항목 | 이전 | 이후 |
|------|------|------|
| 폴더 색상 | 비비드 팔레트 | 비비드 팔레트 (유지) |
| 파일 색상 | 폴더와 동일 | 채도 낮은 회색 계열 |
| 파일 레이블 | 흰색 | 반투명 연회색 |
