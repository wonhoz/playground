# JSON.Fmt 구현 — JSON 붙여넣기 즉시 beautify 앱

## 작업 개요

사용자 요청: JSON 복사 붙여넣기 즉시 beautify + 오류 위치 진단 + Lenient 파싱 지원

## 구현 내용

### JSON.Fmt (Applications/Tools.Utility/)
JSON 붙여넣기 즉시 beautify + 구문 강조 + 오류 진단 + Lenient 파싱

**주요 기능**:
- 클립보드 붙여넣기 / Ctrl+V 즉시 포맷 (자동 beautify)
- WPF RichTextBox + FlowDocument 구문 강조 (키=하늘색, 문자열=살몬, 숫자=연두, bool=보라, null=회색, 구두점=회색)
- `JsonException` 파싱 오류 줄·열 위치 정확 표시
- `JsonNormalizer` Lenient 전처리: `//` 주석, `/* */` 블록 주석, trailing comma, 단일 따옴표, 무따옴표 키 자동 정규화
- `System.Text.Json` 내장 AllowTrailingCommas + CommentHandling.Skip 1차 시도 → 실패 시 Lenient 전처리 후 재시도
- 키 정렬 (재귀 JsonObject 정렬)
- 축소 (Minify)
- 복사 / 지우기
- 드래그앤드롭 JSON 파일 지원
- 상태바: 줄 수 · 자 수 통계

**기술 스택**:
- `System.Text.Json` (빌트인, 경량)
- `System.Text.RegularExpressions` (Lenient 전처리)
- WPF `RichTextBox` + `FlowDocument` + `Inline Run` 구문 강조
- `DwmSetWindowAttribute` 다크 타이틀바
