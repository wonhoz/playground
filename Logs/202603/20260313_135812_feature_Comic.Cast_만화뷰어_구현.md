# Comic.Cast — CBZ/CBR/CB7/CBT 만화 뷰어 구현

**날짜**: 2026-03-13
**태그**: feature
**상태**: 완료

---

## 개요

`Comic.Cast` — WPF 기반 로컬 만화책(CBZ/CBR/CB7/CBT/폴더) 뷰어 앱 구현.
SQLite 라이브러리, ComicInfo.xml 파싱, 단면/양면 보기, WebP 일괄 변환 지원.

---

## 구현 내용

### 프로젝트 구조
- **위치**: `Applications/Photo.Picture/Comic.Cast/`
- **기술 스택**: WPF + MVVM, .NET 8.0-windows, SharpCompress, Microsoft.Data.Sqlite, SixLabors.ImageSharp

### 데이터 모델 (`Models/`)
- `ComicBook.cs` — 만화책 메타데이터, ArchiveType, LibraryFolder
- `ComicPage.cs` — 페이지 정보, ReadingSession (현재 페이지·ViewMode·Zoom)

### Core (`Core/`)
- `RelayCommand.cs` — ICommand 구현 (제네릭 포함)
- `ViewModelBase.cs` — INotifyPropertyChanged 기반 MVVM base

### 서비스 레이어 (`Services/`)
| 서비스 | 역할 |
|--------|------|
| `ArchiveService` | CBZ(ZipFile), CBR/CB7/CBT(SharpCompress) 페이지 목록 추출 및 스트림 제공 |
| `LibraryService` | SQLite DB 관리, ComicInfo.xml 파싱, 폴더 스캔, 읽기 진행 저장 |
| `ImageConvertService` | SixLabors.ImageSharp WebP 일괄 변환, 썸네일 생성 |

### ViewModels
- `ReaderViewModel` — 단면/양면(망가·웹툰) 보기, 줌, 키보드 내비게이션, 비동기 페이지 로드
- `LibraryViewModel` — 책 목록, 검색, 폴더 추가/스캔, WebP 변환 진행
- `MainViewModel` — 탭 전환(라이브러리·리더)

### UI (WPF MVVM)
- **2탭 구조**: 라이브러리(썸네일 WrapPanel) + 리더(단면/쌍면)
- **다크 테마**: `DarkTheme.xaml` (Button, TextBox, ComboBox, ScrollBar, Slider, ListBox, ProgressBar ControlTemplate)
- **다크 타이틀바**: `DwmSetWindowAttribute(handle, 20, 1)`
- **드래그앤드롭**: 커맨드라인 인수로 직접 파일 열기

### 기술적 이슈 해결
- WPF + WinForms 동시 활성화 → `UserControl` 모호성: code-behind에서 base class 명시 제거
- `SixLabors.ImageSharp.Image` vs `System.Windows.Controls.Image` 충돌 → `ISImage` alias
- `SixLabors.ImageSharp.Processing.ResizeMode` vs `System.Windows.ResizeMode` 충돌 → `ISResizeMode` alias
- `ComicCast.Models.ArchiveType` vs `SharpCompress.Common.ArchiveType` 충돌 → `ComicArchiveType` alias
- `System.Windows.Input.KeyEventArgs` vs `System.Windows.Forms.KeyEventArgs` 충돌 → code-behind에서 fully qualified name
