### G1. `Pendulum.Blitz` ⭐⭐⭐ — 진자 스윙 전투 아케이드 (최강 추천)
**장르**: Arcade / Action / Physics
**물리 엔진**: Verlet Integration 진자 (다관절 체인)

#### 핵심 개념
캐릭터가 **천장 앵커에 걸린 진자**처럼 스윙하며 적을 격파. 운동량이 클수록 타격력 증가. Neon Slice + Dodge.Blitz 사용자 취향 정조준.

#### 물리 메커닉
```
진자 운동: θ'' = -(g/L)sin(θ) - damping·θ'
체인 관절: Verlet constraint solver, 다중 앵커 순간 전환
충돌 운동량: 대미지 = m·v² / 2 (운동에너지 비례)
```

#### 게임플레이
| 요소 | 내용 |
|------|------|
| 스윙 조작 | 마우스 클릭·드래그로 앵커 전환, SPACE=놓기 |
| 콤보 | 스윙 속도 > 임계치에서 연속 타격 시 배율 누적 |
| 앵커 종류 | 고정(벽), 이동(적), 탄성(스프링), 폭발(일회성) |
| 적 | 다가오는 적 + 패턴 투사체 회피 |
| 비주얼 | 네온 + 진자 궤적 잔상 (Neon.Slice 느낌) |
| 스테이지 | 무한 모드 + 15개 테마 챌린지 |

#### 기술
- `DispatcherTimer` 120fps 물리 루프
- WPF Canvas + `StreamGeometry` 궤적 렌더링
- 체인 길이별 고유 스윙 주기 = 전략적 앵커 선택

### G2. `Bubble.Pop` ⭐⭐⭐ — 비누 거품 물리 아케이드
**장르**: Casual / Arcade / Physics
**물리 엔진**: 표면장력 최소 에너지 시뮬레이션

#### 핵심 개념
**비누 거품**의 합치기·분리·파열 물리를 핵심으로 한 중독성 아케이드. Dodge.Blitz 선호 사용자가 좋아할 만한 **만족스러운 시각적 피드백** 극대화.

#### 물리 메커닉
```
거품 크기: 체적 보존 (합치면 √(r₁³+r₂³))
표면장력: Plateau 법칙 기반 120도 접합 각도
분리 임계: 내부 압력 > 표면장력 × 2/r → 파열
```

#### 게임플레이
| 요소 | 내용 |
|------|------|
| 기본 조작 | 마우스로 작은 거품 불기(드래그), 병합(겹치기), 터뜨리기(핀 도구) |
| 목표 | 지정 크기 거품 N개 만들기 / 특정 영역 채우기 / 타겟 거품만 터뜨리기 |
| 방해 요소 | 바람(방향 랜덤), 가시(터뜨림), 자석(끌어당기기), 열원(분열 가속) |

### G3. `Crumble.Run` ⭐⭐⭐ — 보로노이 지형 붕괴 러너
**장르**: Arcade / Runner / Physics
**물리 엔진**: Voronoi 파단 + RigidBody 파편

#### 핵심 개념
달리는 캐릭터의 발 디디는 곳마다 **Voronoi 패턴으로 균열이 퍼지며 지형이 무너진다**. 무너지는 파편들이 장애물·발판·경사로 역할을 하며 매번 다른 지형 창출.

#### 물리 메커닉
```
Voronoi 셀 생성: Fortune's algorithm (실시간)
균열 전파: 셀 연결 강도 감쇠 (거리²에 반비례)
파편 물리: 강체 회전·마찰·반발, 지형 재활용
지형 재생성: 파편 퇴장 후 절차적 새 구간 생성
```

#### 게임플레이
| 요소 | 내용 |
|------|------|
| 달리기 | 자동 달리기 + SPACE 점프 + 더블 점프 |
| 파편 활용 | 큰 파편 = 임시 발판, 작은 파편 = 투사체 |
| 붕괴 속도 | 진행할수록 균열 전파 속도 증가 |
| 특수 블록 | 얼음(미끄러짐), 탄성(튕김), 폭발(연쇄), 황금(슬로우모션) |
| 시각 효과 | 균열선 발광 → 파편 분리 → 먼지 파티클 |
| 기록 | 거리·파괴 타일 수·콤보 리더보드 |

### G4. `Wobble.Stack` ⭐⭐ — 소프트 바디 젤리 타워 쌓기
**장르**: Casual / Arcade / Physics
**물리 엔진**: Shape Matching Soft Body (위치 기반 물리 PBD)

#### 핵심 개념
탄성 **젤리 블록**을 쌓아 최대 높이 도전. 블록들이 흔들리고 진동하며 서로 영향을 주는 소프트 바디 물리가 핵심. 단순하지만 예측 불가한 물리가 중독성 창출.

#### 물리 메커닉
```
형상 매칭: 목표 형태로 복원하는 탄성력 F = k(x_goal - x)
내부 압력: 영역 보존 제약 (Volume constraint)
충돌: GJK + EPA 충돌 감지, 임펄스 해결
댐핑: 속도 감쇠 c·v 로 진동 수렴
```

#### 게임플레이
| 요소 | 내용 |
|------|------|
| 블록 투하 | 마우스로 위치 조준 후 클릭 투하 |
| 균형 판정 | 3초간 모든 블록 정지 = 안정 판정, 30% 이상 탈락 = 실패 |
| 블록 종류 | 정육면체·타원·별·도넛 (형태별 탄성률 다름) |
| 방해 요소 | 진동 플랫폼, 바람 방향 전환, 갑자기 떨어지는 돌 |
| 스테이지 | 목표 높이 클리어 → 다음 스테이지 잠금 해제 |
| 비주얼 | 반투명 네온 젤리, 충돌 시 파동 효과 |

### G5. `Slice.Chain` ⭐⭐ — Verlet 체인 절단 물리 퍼즐
**장르**: Puzzle / Casual / Physics
**물리 엔진**: Verlet Integration 제약 체인

#### 핵심 개념
**체인에 달린 물체들을 적절한 위치에서 절단**해 목표를 달성하는 Cut the Rope 스타일 퍼즐. Spring.Web(거미줄)과 달리 **체인 링크 구조**와 다양한 특수 링크가 핵심.

#### 물리 메커닉
```
체인 링크: Verlet 입자 + 거리 제약 (stiffness 0~1)
절단: 링크 양 끝점 제약 제거 → 자유 낙하
특수 링크 상호작용: 자석(인력/척력), 폭발(반경 충격파), 탄성(반발)
```

#### 특수 링크 종류
| 링크 타입 | 효과 |
|-----------|------|
| 일반 | 절단 가능 |
| 철 | 절단 불가, 굴복 없음 |
| 자석 | 주변 쇠 물체 끌어당김 |
| 폭발물 | 절단 시 주변 체인 연쇄 절단 |
| 탄성 | 절단 후 반발력 방출 |
| 얼음 | 흔들면 부서짐 |

#### 게임플레이
- 마우스 드래그 → 절단 경로 그리기
- 목표: 보석을 바구니에 / 폭탄을 물에 빠뜨리기 / 별 3개 수집
- 60개 스테이지 + 하루 챌린지

### G6. `Throw.Stars` ⭐⭐ — 정밀 투척 물리 액션
**장르**: Arcade / Action / Physics
**물리 엔진**: 강체 회전 + 공기저항 + 반발계수

#### 핵심 개념
무기별로 **완전히 다른 비행 물리**를 가진 투척 액션. 표창(직선·스핀), 부메랑(곡선 귀환), 창(포물선+관통), 원반(저항·도약). 벽 반사와 적 관통 콤보가 핵심.

#### 물리 메커닉
```
표창: v = v₀ - c_drag·v·|v|, 회전 = ω₀ + 토크 (자이로 효과 없음)
부메랑: 양력 L = ½ρv²·C_L·A (비대칭 단면), 귀환 경로 계산
창: 포물선 + 관통 시 속도 감쇠·각도 변화
원반: Magnus 효과 F_m = ω×v·계수 → 도약 효과
```

#### 게임플레이
| 요소 | 내용 |
|------|------|
| 투척 | 마우스 드래그 방향·길이 = 방향·속도, 릴리스 = 발사 |
| 콤보 | 벽 1회 반사 +50%, 2회 +150%, 관통 x=x배 |
| 스테이지 | 좌측 플레이어, 우측 적·장애물 배치 |
| 특수 효과 | 느린 시간(타이밍 아슬아슬 클리어 시), 연쇄 폭발 |
| 잠금 해제 | 콤보 달성 → 새 무기 타입 언락 |

### G7. `Crush.Box` ⭐⭐ — 유압 압착 물리 샌드박스
**장르**: Sandbox / Casual / Physics Destruction
**물리 엔진**: Box2D (강체 + 연성체 혼합)

#### 핵심 개념
**"Will it crush?"** — 유리·젤리·금속·폭발물·나무 등 다양한 물체를 유압 프레스·롤러·분쇄기로 파괴. 물질별 파단 패턴과 연쇄 반응이 핵심 재미. 실제로 파괴되는 느낌의 만족감.

#### 물리 메커닉
```
강체 파단: 내부 응력 > 항복강도 → Voronoi 분열
연성체: 세포막 스프링-질량 + 압력 내부 모델
폭발: 충격파 원형 전파, 파편 속도 비례 거리⁻²
유리: 균열 전파 속도 c_crack = √(K_IC/ρ) 근사
```

#### 게임플레이
| 압착 도구 | 동작 |
|-----------|------|
| 유압 프레스 | 위에서 아래로 일정 속도 압착 |
| 롤러 | 좌우로 굴리며 납작하게 |
| 분쇄기 | 회전 톱니, 조각조각 |
| 폭발 프레스 | 폭발 충격으로 순간 압축 |

#### 모드
- **퍼즐**: 정해진 물체를 특정 모양으로 변형 (목표 실루엣 매칭)
- **샌드박스**: 자유롭게 물체 배치 + 도구 선택
- **챌린지**: 제한 시간 내 최대 파괴 점수

### G8. `Shock.Wave` ⭐⭐ — 충격파 연쇄 파괴 퍼즐
**장르**: Puzzle / Strategy / Physics
**물리 엔진**: 충격파 확산 + RigidBody 충격량

#### 핵심 개념
**제한된 폭약**을 전략적으로 배치·순서 폭발시켜 **모든 목표를 연쇄 파괴**하는 퍼즐. 충격파 벡터와 반사, 블록 종류별 반응이 핵심. 슬로우모 리플레이로 연쇄 확인.

#### 물리 메커닉
```
충격파 확산: P(r,t) = P₀·(R/r)·f(t - r/c_sound)
강체 충격량: J = ∫F·dt = m·Δv (방향 포함)
반사: 강체 벽 완전반사, 흡수재 감쇠 e^(-αd)
연쇄 폭발: 폭발물 블록에 충격량 > 임계 → 추가 폭발
```

#### 블록 종류
| 블록 | 충격파 반응 |
|------|------------|
| 나무 | 파괴 후 파편 비산 |
| 돌 | 충격파 반사 (거울) |
| 유리 | 분쇄 + 충격파 감쇠 통과 |
| 폭발물 | 충격 → 추가 폭발 연쇄 |
| 흡수재 | 충격파 완전 흡수 (장벽) |
| 금속 | 충격파 방향 90도 굴절 |

#### 게임플레이
- 폭약 N개 배치 → "DETONATE" → 연쇄 반응 관찰
- 목표: 빨간 블록 전부 파괴 + 녹색 블록 보호
- 슬로우모 리플레이 (0.1× 속도)
- 별 3개 조건: 목표 달성 + 최소 폭약 + 타임 보너스
