## 구현 완료 목록

### 1. Token.Calc (Applications/AI)
- **설명**: LLM 멀티모델 토큰 수 / 비용 실시간 계산기
- **기술**: .NET 10, WPF, SharpToken 2.0.4 (tiktoken C# 포트)
- **핵심 기능**:
  - 시스템 프롬프트 + 사용자 메시지 + 예상 응답 길이 3분할 입력
  - GPT-4o, o1, Claude, Gemini, Llama, Mistral, DeepSeek 등 20개 모델 비교
  - 인코딩별 정확한 토큰 수 계산 (o200k_base, cl100k_base) + Claude 추정값
  - DataGrid에서 입력/출력 단가($/ 1M) 직접 편집 가능
  - 최저 비용 모델 자동 강조
  - 비용순 정렬, 단가 초기화
  - CSV / JSON 내보내기

### 2. Pane.Cast (Applications/Automation)
- **설명**: Windows Terminal 멀티패인 세션 레이아웃 저장 / 복원기 (트레이 앱)
- **기술**: .NET 10, WPF + WinForms NotifyIcon, System.Text.Json
- **핵심 기능**:
  - 세션 프리셋 (탭+패인 구성) JSON 저장 (`%APPDATA%\PaneCast\sessions.json`)
  - 각 탭: 이름 편집 / 각 패인: 프로파일(ComboBox), 디렉토리, 분할방향(오른쪽/아래)
  - `wt new-tab ... ; split-pane -V/-H ...` 명령어 자동 생성 + 미리보기
  - ▶ 실행: cmd.exe /c wt ... 로 Windows Terminal 실행
  - 트레이 상주: 세션별 빠른 실행 메뉴, 더블클릭 창 열기
  - 세션 가져오기 / 내보내기 JSON
  - DarkMenuRenderer 트레이 컨텍스트 메뉴
