# Log.Tail 실시간 로그 뷰어 구현

**날짜**: 2026-02-26
**태그**: feature
**커밋 수**: 7

## 작업 내용

Windows용 `tail -f` 실시간 로그 뷰어 `Log.Tail` WPF 앱 구현.

### 생성 파일

| 파일 | 설명 |
|------|------|
| `Log.Tail.csproj` | net10.0-windows, WPF only, NuGet 없음 |
| `gen-icon.ps1` | 터미널 3선 아이콘 (그린 #22C55E, 16/32/48/256px) |
| `App.xaml` | 터미널 그린 (#22C55E) 다크 테마 |
| `GlobalUsings.cs` | System.IO 포함 (WPF ImplicitUsings에 미포함) |
| `Models/LogLevel.cs` | None/Debug/Info/Warn/Error/Fatal |
| `Models/LogLine.cs` | Number/Raw/Level/IsJson/Timestamp + Frozen LevelBrush |
| `Services/LogParserService.cs` | 레벨 감지 (7종), 타임스탬프 파싱 (4가지 형식), JSON 감지 |
| `Services/LogWatcherService.cs` | FileSystemWatcher tail-f, 로그 로테이션 감지, 초기 로드 |
| `Controls/LogTabView.xaml` | 필터바 + 가상화 ListBox + 상태바 UserControl |
| `Controls/LogTabView.xaml.cs` | 필터링 (텍스트/정규식/대소문자/제외), 자동스크롤, UTC전환, 최대줄 트림 |
| `MainWindow.xaml` | 툴바 + 탭컨트롤 + 빈 화면 안내 |
| `MainWindow.xaml.cs` | 파일 열기, 드래그앤드롭, 멀티탭 관리, DwmSetWindowAttribute |

## 기능 목록

- **FileSystemWatcher 실시간 스트리밍**: 파일 변경 시 새 줄 즉시 표시
- **멀티탭 동시 모니터링**: 여러 파일 탭으로 독립 감시
- **라인 색상 규칙**: FATAL=밝은빨강, ERROR=빨강, WARN=노랑, INFO=그린, DEBUG=파랑
- **필터 검색**: 키워드 포함/제외, 정규식, 대소문자 구분 (300ms 딜레이 최적화)
- **최대 라인 수 제한**: 10K/50K/100K/무제한, 초과 시 앞 10% 자동 트림
- **자동 스크롤 ON/OFF**: 토글 가능
- **UTC↔현지 시간 전환**: 버튼 토글
- **JSON 로그 감지**: `▸` 접두어 표시
- **로그 로테이션 감지**: 파일 크기 감소 시 처음부터 다시 읽기
- **드래그앤드롭 파일 열기**
- **중복 파일 방지**: 이미 열린 파일은 탭 활성화

---

## 해결한 이슈

### System.IO 네임스페이스 미포함
- WPF 앱의 `ImplicitUsings=enable`은 `System.IO`를 포함하지 않음
- `GlobalUsings.cs`에 `global using System.IO;` 추가로 해결
