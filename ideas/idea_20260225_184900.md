### 1. `Mute.Master` — 전체 미팅 앱 원터치 뮤트 토글

- 어느 앱이 현재 마이크를 쥐고 있는지 목록 표시

**수익 모델**
- 무료: 단일 앱 제어
- Pro $8.99 일회성: 전체 앱 동시 제어 + 시작 프로그램 등록 + 단축키 커스텀
- **타겟**: 재택근무자, 하이브리드 오피스 근로자 (전 세계 수천만 명)

### 3. `Font.Probe` — 화면 어디서나 폰트 정보 추출기

- Pro $5.99 일회성: OCR 기반 이미지 폰트 매칭 + 폰트 히스토리 무제한 + CSS/Flutter/SwiftUI 형식 복사
- **타겟**: 프론트엔드 개발자, UI/UX 디자이너

**구현 난이도** ★★★★☆ (UIAutomation + GDI + 선택적 OCR 연동)

### 8. `Cert.Watch` — SSL 인증서 만료 알림 데스크탑 모니터

**문제 정의**
SSL 인증서가 만료되면 사이트가 다운되거나 브라우저 경고가 표시됨.
Let's Encrypt 자동 갱신이 실패하거나, 직접 구매한 인증서를 깜빡하는 사고가 빈번함.
현재 저렴하고 가벼운 로컬 모니터링 도구가 거의 없음.

**킬러 기능**
- 도메인 목록 등록 → 주기적으로 SSL 핸드셰이크로 만료일 체크
- 만료 30일 전 / 7일 전 / 1일 전 Toast + 이메일 알림
- 트레이 아이콘: 녹색(정상) / 노란(경고) / 빨간(만료 임박 or 오류)
- 빠른 확인: 도메인 목록에서 현황 한눈에 (응답시간 + 남은 일수)
- 이력 로그: 언제 갱신됐는지 기록
- 선택적: 사설 인증서(사내 서버) 체크 지원

**수익 모델**
- 무료: 도메인 5개, 이메일 알림 없음
- Pro $5.99/월: 도메인 무제한, 이메일/슬랙 알림, 팀 공유 대시보드
- **타겟**: 프리랜서 웹개발자, 중소기업 IT 담당자, 스타트업

**구현 난이도** ★★☆☆☆ (TcpClient SSL 핸드셰이크 + SslStream.RemoteCertificate)

### 10. `Log.Tail` — 실시간 로그 뷰어 (Windows용 tail -f)

**문제 정의**
Linux의 `tail -f`, `less +F`는 Windows에 기본 없음. PowerShell Get-Content -Wait는 느리고 보기 불편함.
IIS, .NET 앱, 도커 로그를 실시간으로 예쁘게 보고 싶은데 NotePad로 새로 고침하는 상황.

**킬러 기능**
- 파일 열기 → 실시간 스트리밍 뷰 (FileSystemWatcher)
- **여러 파일 탭으로 동시 모니터링**
- 라인 색상 규칙: ERROR=빨강, WARN=노랑, INFO=초록 (커스텀 Regex 규칙 추가 가능)
- 필터 검색 (키워드 포함/제외, Regex)
- 최대 라인 수 제한 (메모리 보호), 자동 스크롤 ON/OFF
- 라인 타임스탬프 파싱 + UTC↔로컬 변환
- JSON 로그 자동 감지 → 구조적 뷰어로 전환

**수익 모델**
- 무료: 파일 탭 2개, 색상 규칙 3개
- Pro $7.99 일회성: 탭 무제한, 규칙 무제한, TCP 소켓 스트림 지원, 필터 저장
- **타겟**: 백엔드 개발자, DevOps, 서버 운영자

**구현 난이도** ★★☆☆☆ (FileSystemWatcher + WPF VirtualizingPanel)
