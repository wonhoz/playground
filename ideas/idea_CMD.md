  - 차별점: 경량 WPF, 전역 단축키로 클립보드 이미지 즉시 편집, 저장 없이 바로 재복사
  - 개발 난이도: 중간 (System.Drawing + 커스텀 주석 레이어)

  ③ Glyph.Map Dev / Productivity

  - 페인 포인트: 특수문자 찾으려면 구글링하거나 Character Map(구식 UI) 써야 함
  - 경쟁자: Windows Character Map(너무 불편), BabelMap(무료지만 2003년대 UI)
  - 수익 모델: $2.99 원타임 / Microsoft Store 출시 가능
  - 차별점: 모던 다크 UI, 빠른 퍼지 검색, 이모지 카테고리, HTML 엔티티·Python·CSS 코드 동시 복사
  - 개발 난이도: 중간 (유니코드 DB 파싱)
  - 취향 연관: Text.Forge, Char.Art 선호 → 문자 관련 도구 친숙

  ④ Sound.Cast Tray / Automation
  오디오 출력 장치 원클릭 전환 트레이 앱 (스피커↔헤드폰, 최대 5 장치 프리셋)

  - 페인 포인트: Windows에서 스피커↔헤드폰 전환이 설정→사운드→장치 3단계
  - 경쟁자: EarTrumpet(오픈소스, 무료) — 볼륨만 됨, 장치 전환은 별도
  - 수익 모델: 무료 + 장치 프리셋 프로파일(게임 모드/업무 모드) 유료
  - 차별점: 단축키 + 트레이 1클릭 전환, 앱별 볼륨 프로파일 저장, Stay.Awake처럼 경량 트레이
  - 개발 난이도: 중간 (CoreAudio API)
  - 취향 연관: Stay.Awake 최애 → 트레이 앱 장르 강한 선호

  - 경쟁자: WinXCorners(2016년, 기능 최소), Cornerfly(한정적)

  - 페인 포인트: 이메일, 주소, 코드 보일러플레이트를 매번 입력해야 함
  - 경쟁자: TextExpander($39.96/년), Espanso(오픈소스지만 설치 복잡)
  - 수익 모델: 기본 50개 스니펫 무료 → Pro 무제한 $4.99 원타임
  - 차별점: 오프라인 완결, 스니펫 동기화 없이 로컬 JSON 저장, {date} {clip} 변수 지원
  - Smart.Paste 차이: Smart.Paste는 붙여넣기 차단 우회, Auto.Type은 타이핑 자동화. 완전히 다름

  - 수익 가능: 타이핑 연습 게임 앱스토어 판매

  ② Mirror.Run Arcade / Unique
  화면 좌우 대칭 두 캐릭터를 단일 입력으로 동시 조작 — 장애물이 미러링됨

  왼쪽 피하면 오른쪽도 같이 피함 → 두 캐릭터 모두 살려야 클리어
  좌우 비대칭 장애물 패턴이 핵심 난이도
  레벨마다 거울 방향이 바뀜 (상하, 대각)
  - 중독 요소: "동시에 두 곳" 컨트롤의 뇌 꼬임 + 직관적 조작
  - 독창성: 이 메커닉을 쓴 Windows 게임 없음

  ③ Shadow.Run Arcade / Self-Competition
  자신의 이전 런 고스트와 경쟁하는 무한 달리기

  멀티 고스트(1위/개인최고/어제) 동시 표시
  장애물 패턴은 랜덤이지만 고스트는 완벽한 이전 경로 재현
  - 중독 요소: 자기 자신과의 경쟁 → 세상에서 제일 끈질긴 라이벌
  - 취향 연관: Dodge.Blitz 최애 → 달리기 + 피하기 장르 선호

  ④ Chain.Blast Puzzle / Colorful
