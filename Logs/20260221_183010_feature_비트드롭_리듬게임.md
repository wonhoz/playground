## 작업 내용

DJMAX/osu!mania 스타일 4키 노트 낙하 리듬 게임 개발.

### 구현 기능

| 기능 | 설명 |
|------|------|
| 4레인 노트 | D(파랑)/F(초록)/J(노랑)/K(빨강) 키 바인딩, 컬러 구분 |
| 패턴 생성 | BPM 기반 자동 생성 (120/140/160/180 BPM 선택) |
| 난이도 | Easy (1비트), Normal (반박), Hard (16분음표) |
| 판정 | Perfect (±40ms), Great (±80ms), Good (±120ms), Miss (±180ms) |
| 점수 | 판정별 기본점수 + 콤보 보너스, 가중 정확도 |
| 랭크 | S (95%+), A (90%+), B (80%+), C (70%+), D |
| 노트 종류 | 일반 노트, 롱노트, 동시타 (더블 노트) |
| UI | 레인 플래시, 판정 텍스트 팝업, 콤보 카운터, 사이드 네온 바 |
| 결과 | 랭크/점수/정확도/판정 상세/최대 콤보 |

### 아키텍처

```
Engine/
  GameLoop.cs            - 게임 루프
  Note.cs                - 노트 데이터 (레인, 타이밍, 롱노트)
  PatternGenerator.cs    - BPM/난이도 기반 자동 패턴 생성
  ScoreManager.cs        - 판정/점수/콤보/랭크 시스템
```

### 조작법

| 키 | 동작 |
|----|------|
| D F J K | 4레인 노트 입력 |
| ENTER | 시작/돌아가기 |
| ESC | 종료 |

### 판정 윈도우

| 판정 | 타이밍 차이 | 점수 |
|------|-------------|------|
| PERFECT | ±40ms | 300 + 콤보×5 |
| GREAT | ±80ms | 200 + 콤보×3 |
| GOOD | ±120ms | 100 + 콤보×1 |
| MISS | ±180ms+ | 콤보 리셋 |
